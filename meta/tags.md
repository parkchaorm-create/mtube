## 태그 사용 전략

1. **유튜브 태그 한계**: 최대 500자, 태그당 30자 이내 권장
2. **상위 10개 우선 순위**: 클로드코드, 클로드코드마케팅, AI마케팅자동화, Claude Code, Claude Code tutorial, 마케팅자동화, AI에이전트, 파자마보스, 클로드코드풀코스, 왕초보AI
3. **해시태그 vs 태그 차이**: 해시태그(설명글 하단)와 태그(태그 필드)는 별도 관리
