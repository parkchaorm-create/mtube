1. **첫 3줄이 핵심**: 유튜브 검색 결과 미리보기에 표시되는 줄이므로 가장 중요한 키워드 배치
2. **FAQ 포함**: AI 검색 엔진(ChatGPT, Perplexity)이 Q&A 형식을 자주 인용 → GEO 효과
3. **링크 업데이트**: 영상 업로드 전 실제 커뮤니티/프롬프트 링크로 교체
4. **타임스탬프**: 실제 촬영 후 편집 길이에 맞춰 조정 필요
