# 썸네일 구성 지시서

## 최종 썸네일 컨셉: "충격 + 결과"

---

## 레이아웃 구성

```
┌─────────────────────────────────────────────────┐
│  [클로드 로고]                      [풀코스 뱃지] │
│                                                  │
│  [화자 컷아웃]        왕초보 OK ←─── (노란색)   │
│                       AI 마케팅팀  ←─── (흰색)  │
│                       4시간 완성!  ←─── (오렌지) │
│                                                  │
│  [화살표: 설치→스킬→팀]                          │
└─────────────────────────────────────────────────┘
```

## 색상 팔레트

| 요소 | 색상 코드 | 설명 |
|------|-----------|------|
| 배경 | #0f0f1a | 다크 네이비 (눈에 띄는 대비) |
| 메인 텍스트 | #FFFFFF | 흰색 볼드 |
| 강조 텍스트 1 | #FFD700 | 골드 (왕초보 OK) |
| 강조 텍스트 2 | #FF6B35 | 오렌지 (4시간 완성!) |
| 서브 요소 | #A8DADC | 밝은 청록 (화살표, 아이콘) |
| 뱃지 배경 | #E63946 | 빨간색 (풀코스 뱃지) |

---

## 텍스트 구성 (3줄)

```
1번째 줄: 왕초보 OK
  - 크기: 작게 (전체 텍스트 높이의 20%)
  - 색상: #FFD700 (골드)
  - 폰트: 굵은 고딕 계열

2번째 줄: AI 마케팅팀
  - 크기: 크게 (전체 텍스트 높이의 45%)
  - 색상: #FFFFFF (흰색)
  - 폰트: 매우 굵은 고딕 (Black/Heavy weight)
  - 효과: 약한 텍스트 쉐도우

3번째 줄: 4시간 완성!
  - 크기: 크게 (전체 텍스트 높이의 35%)
  - 색상: #FF6B35 (오렌지)
  - 폰트: 굵은 고딕
  - 효과: 밑줄 또는 하이라이트 배경
```

## 아이콘 & 뱃지 요소

### 1. 클로드 로고
- 위치: 우측 상단 코너
- 크기: 전체 썸네일의 8~10%
- 처리: 흰색으로 변환 또는 원래 색상 유지 (대비 확인)

### 2. 풀코스 뱃지
- 형태: 원형 스티커 (기울어진 느낌, -15도 회전)
- 텍스트: "풀코스"
- 색상: 빨간 배경(#E63946) + 흰 텍스트
- 위치: 우측 상단 (클로드 로고 옆 또는 아래)

### 3. 진행 흐름 아이콘 (선택)
- 형태: 작은 아이콘 3개 → 화살표로 연결
- 내용: 설치(💻) → 스킬(⚙️) → 팀(🤝)
- 위치: 하단 좌측
- 색상: #A8DADC (밝은 청록)

---

## 화자 이미지 가이드

- **표정**: 놀란 표정 또는 자신 있는 포즈 (엄지 세우기)
- **시선**: 카메라를 향하거나, 텍스트 쪽을 바라보는 방향
- **의상**: 밝은 색상 추천 (배경 다크 네이비와 대비)
- **배경 제거**: 컷아웃 처리 (누끼 따기)
- **위치**: 좌측 40~60% 영역
- **크기**: 전체 썸네일 높이의 85~95%

## A/B 테스트용 썸네일 B안

### 컨셉: "Before/After 비교"

```
┌──────────────┬──────────────────────────────┐
│  Before      │                              │
│  ───────     │   클로드 코드로               │
│  수동 작업   │   AI 마케팅 팀               │
│  😩          │   만드는 법                  │
│              │                              │
│  After       │   [4시간 풀코스]             │
│  ───────     │                              │
│  자동화 완성 │                              │
│  😎          │                              │
└──────────────┴──────────────────────────────┘
```

- 색상: 좌측 회색 배경 / 우측 다크 네이비
- 텍스트: Before/After 라벨 + 이모지로 감정 전달
