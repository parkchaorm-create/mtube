# 영상 제작 규칙

이 문서는 **풀코스 영상을 실제로 만드는 방법**에 대한 규칙입니다.
스크립트 내용과는 별개이며, 제작팀(우리)이 따르는 규칙입니다.

---

## 캐릭터

- **김보스 = 2D 심플 라인 캐릭터** (실사 아님, AI 아바타 아님)
- 나노바나나로 캐릭터 시트 생성 (정면, 측면, 다양한 표정/포즈)
- 리모션으로 캐릭터 + 말풍선 + 인포그래픽 합성 애니메이션
- 스크립트에서 `[📺 화면: 웹캠 단독]` 등 화자 등장 장면은 모두 2D 캐릭터로 제작

---

## 음성

- **모든 나레이션 = 일레븐랩스(ElevenLabs) 김보스 Voice Clone**
- 5시간 전체 목소리 통일 (Edge TTS 등 다른 TTS와 혼용 금지)
- Voice Clone 세팅: 김보스 음성 샘플 녹음 → 일레븐랩스에서 학습
- 모델: Multilingual v2 권장

## B-Roll 유형별 제작 방법

### 유형 1: 인포그래픽 PPT 영상
- **대상**: `[📊 PPT 지문:]` 마커
- **도구**: 나노바나나(이미지) → 리모션(애니메이션)
- **스타일**: 블랙 & 화이트 미니멀리즘, 글자 최소화, 아이콘/도형 중심
- **초당 비용**: 무료

### 유형 2: 김보스 캐릭터 장면
- **대상**: `[📺 화면: 웹캠 단독]` 등 화자 등장
- **도구**: 나노바나나(캐릭터 이미지) → 리모션(립싱크/모션) + 일레븐랩스(음성)
- **캐릭터 포즈**: 설명, 질문, 놀람, 축하 등 상황별 포즈 시트 필요

### 유형 3: 컴퓨터 화면 녹화
- **대상**: `[📺 화면: 안티그래비티...]` 실습 장면
- **도구**: OBS 또는 QuickTime으로 직접 녹화
- **후처리**: ffmpeg로 마우스 하이라이트, 줌인 효과
- **안티그래비티로 자동화 불가** — GUI 녹화 기능 없음, 수동 녹화 필요

### 유형 4: 결과물 데모 (오프닝 전용)
- **대상**: 오프닝 0:00~0:30
- **도구**: 리모션(3D 트랜지션 + 빠른 편집) + 나노바나나(결과물 목업)
- **퀄리티**: 최고 — Spring physics, 글래스모피즘, 빠른 컷 편집

## 제작 파이프라인

```
1. 스크립트 파싱
   /parse-script → production/scenes/partXX-scenes.json

2. 인포그래픽 이미지 생성
   나노바나나 MCP → production/images/

3. 캐릭터 시트 생성 (최초 1회)
   나노바나나 MCP → production/character/

4. 나레이션 음성 생성
   일레븐랩스 Voice Clone → production/audio/

5. 리모션 영상 렌더링
   /render-masterclass, /3d-transition → production/video/clips/

6. 컴퓨터 화면 녹화
   OBS/QuickTime → production/video/screen-recordings/

7. 최종 합성
   ffmpeg → production/video/partXX-final.mp4
```

## 파일 구조

```
production/
├── character/              ← 김보스 2D 캐릭터 시트
│   ├── front.png
│   ├── side.png
│   ├── expressions/        ← 표정별 (smile, surprise, explain, celebrate)
│   └── poses/              ← 포즈별 (pointing, thumbsup, thinking)
├── images/                 ← 인포그래픽 이미지
│   ├── part00/
│   ├── part01/
│   └── ...
├── audio/                  ← 일레븐랩스 나레이션
│   ├── part00/
│   └── ...
├── video/
│   ├── clips/              ← 리모션 렌더링 클립
│   ├── screen-recordings/  ← 화면 녹화
│   └── partXX-final.mp4   ← 최종 합성 영상
├── scenes/                 ← 파싱된 씬 JSON
└── bgm/                    ← 배경음악
```

## 주의사항

- **스크립트 내용과 영상 제작 방법은 별개**. 이 문서는 제작 방법만 다룸.
- 스크립트 작성 규칙은 `SCRIPT_GUIDE.md` 참조.
- 캐릭터 디자인이 확정되면 이 문서에 레퍼런스 이미지 경로 추가.
