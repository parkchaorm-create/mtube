## [CONCEPT] 지휘자의 역할

[📺 화면: 슬라이드]
[📊 PPT 지문: 지휘자 워크플로우]

지휘자가 하는 일을 정리해보겠습니다.

```
1악장: 리서치 🔍
지휘자 → "리서치 단원, 조사해!" → 결과 받음
↓
2악장: 콘텐츠 작성 ✍️
지휘자 → 리서치 결과 + "콘텐츠 단원, 글 써!" → 결과 받음
↓
3악장: SEO 최적화 📊
지휘자 → 콘텐츠 + "SEO 단원, 최적화!" → 결과 받음
↓
4악장 (피날레): 배포 준비 📤
지휘자 → 최종본 + "배포 단원, 정리!" → 완성!
```

1악장이 끝나야 2악장이 시작되는 구조입니다. 리서치가 끝나야 콘텐츠를 쓸 수 있으니까요!

## [DEMO] 지휘자 만들기 – 그대로 복붙하세요!

[📺 화면: 안티그래비티 - .claude/commands/marketing-team.md 생성]

지휘자는 악기 보관함(commands)에 만듭니다. **복사해서 붙여넣으세요!**

```markdown
---
name: AI 마케팅 팀 오케스트레이터
description: 주제 하나로 전체 AI 마케팅 오케스트라를 지휘하여 완성된 콘텐츠 패키지를 생성합니다
---

당신은 AI 마케팅 오케스트라의 지휘자(오케스트레이터)입니다.
아래 전문 단원들을 악장별로 조율하여 완전한 콘텐츠 패키지를 생성합니다.

## 단원 명단
- 🔍 리서치 단원: @.claude/agents/research-agent.md
- ✍️ 콘텐츠 단원: @.claude/agents/content-agent.md
- 📊 SEO 단원: @.claude/agents/seo-agent.md
- 📤 배포 단원: @.claude/agents/distribution-agent.md

## 공연 전 확인 (먼저 물어보세요)
1. "오늘 다룰 콘텐츠 주제를 알려주세요"
2. "발행 타겟 채널을 알려주세요 (전체/특정 채널)"
3. "발행 예정일이 있나요?"

## 공연 순서 (STEP-BY-STEP)

### 1악장: 리서치 (리서치 단원)
"리서치 단원으로서 [주제]를 조사해줘.
최신 트렌드 3가지, 독자 질문 5가지, 경쟁 콘텐츠 분석, 차별화 포인트 1가지 제공해줘."

리서치 완료 후 → 2악장으로

### 2악장: 콘텐츠 생성 (콘텐츠 단원)
리서치 결과를 전달하며:
"콘텐츠 단원으로서 위 리서치 결과를 바탕으로 콘텐츠를 작성해줘.
@brand-voice-guide.md 브랜드 보이스 적용.
블로그 포스트 전체 + 인스타 2개 + 뉴스레터 생성해줘."

콘텐츠 완료 후 → 3악장으로

### 3악장: SEO 최적화 (SEO 단원)
"SEO 단원으로서 위 콘텐츠를 최적화해줘.
메타 태그 생성, 키워드 최적화, GEO 최적화 적용해줘."

SEO 완료 후 → 4악장으로

### 4악장 (피날레): 배포 준비 (배포 단원)
"배포 단원으로서 위 콘텐츠를 발행 형식으로 정리해줘.
발행 일정과 체크리스트도 만들어줘."

### 커튼콜: 최종 보고
모든 악장 완료 후:
- 생성된 파일 목록
- 발행 준비 현황
- 다음 액션 체크리스트

## 결과물 저장
모든 결과물을 날짜별 폴더에 저장:
content/[날짜]/
├── research-notes.md
├── blog-post.md
├── social-posts.md
├── newsletter.md
└── distribution-ready.md
```
