# Part 19: 쇼츠 영상 자동 제작 – 영상 팀까지 완성

## 메타 정보
- **타임스탬프**: 4:28:00 ~ 4:38:00
- **예상 길이**: 10분 (약 2,500~3,000자)
- **화면 구성**: 웹캠 → PPT(쇼츠 제작 워크플로우) → 안티그래비티(스킬 설정) → 쇼츠 결과물 재생 → 웹캠
- **핵심 비유**: 쇼츠 제작 팀 = 오케스트라의 녹음·편집 팀, 본공연에서 하이라이트를 뽑아 앨범으로 만드는 것
- **필요 도구**: yt-dlp, whisper, ffmpeg (자동 설치)

유튜브 쇼츠, 인스타 릴스, 틱톡. 요즘 숏폼 영상이 마케팅에서 얼마나 중요한지 다들 아시죠?

## [CONCEPT] 쇼츠 제작 워크플로우

[📊 PPT 지문: 쇼츠 자동 제작 워크플로우 — ① 유튜브 URL 입력 → ② yt-dlp로 영상 다운로드 → ③ Whisper로 자막 추출 → ④ AI가 하이라이트 구간 선정 → ⑤ ffmpeg로 쇼츠 편집(세로 9:16, 자막 포함) → ⑥ 쇼츠 3개 완성]

쇼츠 자동 제작은 이런 흐름으로 진행됩니다.

먼저 유튜브 URL을 넣으면, yt-dlp라는 도구가 영상을 다운로드합니다. 그다음 Whisper라는 음성 인식 도구가 영상의 대사를 텍스트로 변환해줍니다. AI가 그 텍스트를 분석해서 가장 재밌거나 핵심적인 구간 3개를 골라내고요. ffmpeg라는 영상 편집 도구가 해당 구간을 잘라서 세로 화면(9:16 비율)에 자막까지 입혀줍니다.

이 모든 과정이 하나의 스킬로 자동화돼 있거든요. 보스님들은 URL만 넣으면 됩니다.

## [DEMO] 쇼츠 제작 스킬 세팅

[📺 화면: 안티그래비티 — 프로젝트 폴더 구조]

쇼츠 제작 스킬을 만들겠습니다.

스킬 폴더 구조를 보면 이렇습니다.

```
.claude/skills/
├── nanobanana-visual/     ← 아까 만든 비주얼 스킬
└── shorts-video-maker/    ← 지금 만들 쇼츠 스킬
    ├── SKILL.md
    └── scripts/
        ├── download_video.py
        ├── transcribe.py
        └── cut_shorts.py
```

shorts-video-maker 폴더 안에 SKILL.md 파일과 파이썬 스크립트 3개가 들어갑니다.

여기서 한 가지 확인할 게 있거든요. 이 스킬은 Python이 필요합니다. 안티그래비티 터미널에서 "python --version"이라고 입력해보세요. 버전 번호가 나오면 OK. 안 나오면 "python.org"에서 설치하시면 됩니다. 역시 무료거든요.

그걸 확인했으면, 직접 만들 필요 없습니다. Claude Code한테 시키면 되거든요.

Claude Code 채팅창에 이렇게 입력하세요:

"유튜브 URL을 입력하면 자동으로 쇼츠 영상 3개를 만들어주는 스킬을 만들어줘. yt-dlp로 영상 다운로드하고, whisper로 자막 추출하고, ffmpeg로 하이라이트 구간을 세로 9:16 비율로 잘라서 자막까지 입혀줘. 자막 크기는 16px, 하단 배치로 해줘."

[📺 화면: Claude Code가 스킬 파일과 파이썬 스크립트를 자동 생성하는 과정]

Claude Code가 SKILL.md 파일과 파이썬 스크립트 3개를 자동으로 만들어주고 있습니다.

download_video.py는 영상을 다운로드하는 스크립트고요. transcribe.py는 음성을 텍스트로 변환하는 스크립트입니다. cut_shorts.py는 하이라이트 구간을 잘라서 쇼츠로 편집하는 스크립트죠.

[📺 화면: 필요한 라이브러리 자동 설치 과정]

필요한 라이브러리(yt-dlp, whisper, ffmpeg)도 Claude Code가 알아서 설치해줍니다. 보스님들이 직접 설치할 필요 없거든요.

## [CONCEPT] 전체 마케팅 팀 완성도 확인

[📺 화면: 웹캠 복귀]

[📊 PPT 지문: 완성된 AI 마케팅 팀 전체 구조 — 오케스트레이터(지휘자) → 텍스트 팀(블로그·뉴스레터·SNS) + 비주얼 팀(카드뉴스·썸네일) + 영상 팀(쇼츠 3개) + 검수(리뷰어) → 최종 outputs/ 폴더에 전부 저장]

보스님들, 지금 우리가 뭘 만든 건지 한번 정리해보겠습니다.

URL 하나 넣으면, 텍스트 팀이 블로그, 뉴스레터, 인스타 캡션, X 스레드, 링크드인 포스트를 만들고요. 비주얼 팀이 인스타 카드뉴스 5장과 유튜브 썸네일을 만들고요. 영상 팀이 쇼츠 3개를 만들어줍니다.

이게 전부 한 번의 명령으로, 약 10분 만에 완성됩니다.

마케팅 에이전시에 이 작업을 맡기면 얼마나 들까요? 콘텐츠 기획부터 디자인, 영상 편집까지 최소 수십만 원이 들거든요. 그걸 Claude Code 하나로 해결한 겁니다.

"아니 근데 이게 진짜 실무에서 쓸 수 있는 수준인가요?"

솔직하게 말씀드리겠습니다. 텍스트 콘텐츠는 바로 발행해도 될 수준입니다. 이미지와 쇼츠는 약간의 미세 조정이 필요할 수 있거든요. 하지만 핵심 틀을 AI가 다 만들어주니까, 거기서 살짝 다듬는 건 10~15분이면 충분합니다. 0에서 시작하는 것과 80%에서 시작하는 건 차원이 다른 거죠.

세 번째, 한 번의 명령으로 약 10분 만에 전체 마케팅 패키지가 생성됩니다. 마케팅 에이전시 수준의 결과물을 파자마 입고 집에서 만드는 겁니다.
