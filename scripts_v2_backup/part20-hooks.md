## [CONCEPT] Hooks = 알람 시계

[📺 화면: 슬라이드]
[📊 PPT 지문: Hooks = 자동 공연 알람]

Hooks는 알람 시계라고 말씀드렸죠.

"이런 일이 생기면 자동으로 이걸 해줘"라고 설정해두는 겁니다.

**공연장에서의 비유:**

| 알람 (이벤트) | 자동 반응 | 마케팅 활용 |
|--------|------|------------|
| 새 악보가 저장되면 | 자동 교정 시작 | 콘텐츠 저장 시 품질 검사 |
| 공연 준비 완료되면 | 무대 조명 자동 점등 | 발행 전 최종 검토 |
| 특정 주법 사용 후 | 음향 자동 조절 | WebFetch 후 자동 요약 |

**설정 위치**: `.claude/settings.json` 파일

## [DEMO] 설정 파일 만들기 – 따라하세요!

[📺 화면: 안티그래비티 - settings.json 파일]

`.claude/settings.json` 파일을 만들거나 열겠습니다.

가장 간단한 알람부터 시작하겠습니다. **이걸 복사해서 붙여넣으세요!**

```json
{
  "hooks": {
    "PostFileWrite": [
      {
        "matcher": "content/**/*.md",
        "command": "echo '🎵 새 악보(콘텐츠) 저장 완료! 파일: {file}'"
      }
    ]
  }
}
```

이게 뭐냐면, `content/` 폴더 안에 `.md` 파일이 생성될 때마다 알림이 울리는 겁니다.

"아니 이게 전부인가요?"

네, 이게 전부입니다! 알람 시계 맞추는 거랑 똑같거든요.

## [DEMO] 실전 알람 3개 – 진짜 쓸 수 있는 것들

[📺 화면: 안티그래비티 - settings.json 확장]

이제 진짜 실전에서 쓸 수 있는 알람 3개 설정해볼게요.

**알람 1: 콘텐츠 저장하면 자동 품질 검사** 🔍

```json
{
  "hooks": {
    "PostFileWrite": [
      {
        "matcher": "content/**/*.md",
        "command": "echo '품질 검사 시작...' && claude --print '방금 저장된 콘텐츠를 검토해줘. 브랜드 보이스 일치 여부, CTA 포함 여부, 해시태그 수를 체크해줘.' @{file}"
      }
    ]
  }
}
```

콘텐츠 저장하면 자동으로 품질 검사가 돌아갑니다! 매번 수동으로 "검토해줘" 안 해도 되죠.

**알람 2: 뉴스레터 저장하면 제목줄 최적화** 📧

```json
{
  "matcher": "content/**/newsletter*.md",
  "command": "claude --print '이 뉴스레터의 이메일 제목줄을 3가지 대안으로 개선해줘. 클릭률 최대화 초점.' @{file}"
}
```

**알람 3: 블로그 저장하면 SEO 메타 태그 자동 생성** 🏷️

```json
{
  "matcher": "content/**/blog*.md",
  "command": "claude --print '이 블로그 포스트의 SEO 메타 태그(제목, 설명)를 생성해줘.' @{file}"
}
```

## [CONCEPT] 알람 설정 주의사항

[📺 화면: 슬라이드]

알람 설정할 때 조심할 것들이 있습니다.

좋은 알람은 가벼운 작업입니다. 품질 체크, 알림, 간단한 검증 같은 것들이죠. 그리고 `echo`로 먼저 테스트한 다음에 잘 되면 실제 명령어로 교체하는 게 좋습니다.

[📊 PPT 지문: Hooks 주의사항
- 좋은 알람: 가벼운 작업 (품질 체크, 알림, 검증)
- 나쁜 알람: 무거운 작업 (전체 오케스트라 실행), 위험한 명령어 (파일 삭제)]

반대로 나쁜 알람은 너무 무거운 작업을 거는 겁니다. 전체 오케스트라 실행을 Hook에 걸면 과부하가 걸리거든요. 파일 삭제 같은 위험한 명령어도 절대 걸면 안 됩니다.

안전하게 시작하려면 처음엔 `echo '...'`으로 테스트만 하세요. 알람이 제대로 울리는지 확인한 다음에 실제 명령어로 바꾸면 됩니다!
