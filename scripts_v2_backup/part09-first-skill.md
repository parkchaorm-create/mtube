## [CONCEPT] 스킬이란? – 한 번 만들면 계속 쓰는 악기

[📺 화면: 슬라이드]

[📊 PPT 지문: 스킬 = 한 번 만들면 계속 쓰는 악기
| 구분 | 일반 ChatGPT | Claude Code 스킬 |
|------|-------------|-----------------|
| 방식 | 매번 긴 설명 입력 | 악기(스킬 파일) 한 번 만들면 끝 |
| 반복 작업 | 처음부터 다시 설명 | /social-media 명령어 하나로 실행 |
| 결과물 | 퀄리티가 매번 달라짐 | 항상 동일한 퀄리티, 동일한 형식 |]

Claude Code 스킬의 개념을 먼저 이해해볼게요.

일반적으로 ChatGPT를 쓸 때는 매번 긴 설명을 입력해야 하잖아요. 같은 작업을 반복할 때마다 처음부터 설명해야 하고, 결과물 퀄리티도 매번 달라질 수 있습니다.

근데 Claude Code 스킬은 다릅니다. 악기를 한 번 만들어두면, `/social-media` 명령어 하나로 실행되고, 항상 동일한 퀄리티와 형식으로 결과가 나옵니다.

"아니 스킬 파일은 어디에 저장해요?"

좋은 질문! 프로젝트 폴더 안에 `.claude/commands/` 폴더를 만들고 그 안에 마크다운 파일로 저장합니다.

악기 보관함이라고 생각하시면 됩니다:

```
my-ai-marketing/          ← 우리 연습실
├── CLAUDE.md              ← 총보
└── .claude/
    └── commands/          ← 🎵 악기 보관함
        ├── social-media.md   ← 오늘 만들 첫 번째 악기 (바이올린)
        ├── repurpose.md      ← 두 번째 악기 (나중에)
        └── seo-audit.md      ← 세 번째 악기 (나중에)
```

## [DEMO] 악기 보관함 만들기 – 따라하세요!

[📺 화면: 터미널 - my-ai-marketing 폴더에서]

자, 터미널에서 악기 보관함부터 만들겠습니다. 이걸 복사해서 붙여넣으세요!

```bash
mkdir -p .claude/commands
```

`-p` 옵션은 중간 폴더도 같이 만들어준다는 뜻이거든요. 보관함과 선반을 동시에 세팅!

확인해볼까요?

```bash
ls -la .claude/commands/
```

폴더가 생겼죠?

🎉 **악기 보관함 세팅 완료!**

## [DEMO] 바이올린(소셜미디어 스킬) 제작 – 핵심 파트, 그대로 복붙하세요!

[📺 화면: 안티그래비티에서 좌측 파일 탐색기 열고, .claude 폴더 → commands 폴더 클릭 → 우클릭하여 "새 파일" 선택 → 파일명에 "social-media.md" 입력하는 장면]

안티그래비티에서 `.claude/commands/social-media.md` 파일을 만들겠습니다.

자, 이 악기의 설계도를 채울게요. 이 전체를 그대로 복사해서 붙여넣으세요!

```markdown
---
name: 소셜미디어 콘텐츠 생성
description: 주제나 URL을 입력하면 인스타그램, 유튜브 쇼츠, 블로그용 콘텐츠를 동시에 생성합니다
---

당신은 CLAUDE.md에 정의된 브랜드의 전담 소셜미디어 콘텐츠 전문가입니다.
CLAUDE.md의 브랜드 보이스, 타겟 고객, 채널별 가이드라인을 반드시 따르세요.

## 먼저 확인할 것
콘텐츠 주제 또는 URL이 제공되었는지 확인하세요.
제공되지 않았다면 먼저 물어보세요: "오늘 다룰 주제나 참고할 URL이 있으신가요?"

## 생성할 콘텐츠

### 인스타그램 포스트
형식:
- 첫 줄: 시선을 끄는 후크 (30자 이내)
- 본문: 핵심 내용 + 가치 제안 (120~200자)
- 마무리: CTA (30자 이내)
- 해시태그: 한국어 8개 + 영어 5개 = 13개
- 이모지: 3~4개 자연스럽게

### 유튜브 쇼츠 스크립트
형식:
- 후크 (0~5초): 강렬한 첫 마디 (20자 이내)
- 핵심 내용 (5~50초): 3가지 포인트, 각 10초
- CTA (50~60초): 구독/링크 유도

### 블로그 제목 3개
- SEO 최적화된 제목
- 각각 다른 앵글 (호기심형, 숫자형, 방법론형)

## 출력 시 주의사항
- 채널명을 헤더(##)로 구분
- 각 콘텐츠 끝에 "📌 다음 액션:" 항목 추가
- 한국어로 작성 (브랜드 요구사항에 따라)
```

[📺 화면: 파일 저장]

Ctrl+S (맥은 Cmd+S)로 저장!

🎉 **보스님들, 방금 첫 번째 악기(바이올린)를 완성했습니다!**

## [DEMO] 악기 튜닝 – 결과물 개선하기

[📺 화면: 결과물 검토 → 수정 요청]

첫 연주가 완벽했나요? 아마 아닐 겁니다. 괜찮습니다! 악기는 튜닝하면서 완성되는 거니까요.

결과물 보고 마음에 안 드는 부분이 있으면 바로 말하세요:

```
인스타 캡션이 너무 형식적인 것 같아. 더 친근하고 구어체로 바꿔줘.
그리고 마지막 CTA에 "댓글로 알려주세요!" 같은 댓글 유도 문구 넣어줘.
```

[📺 화면: Claude Code 수정 결과]

좋아진 버전이 나왔죠?

**핵심 꿀팁:** 이렇게 개선된 내용이 마음에 들면, 그 지시사항을 악기 설계도(스킬 파일)에 반영하세요! 그러면 다음부터는 처음부터 좋은 결과가 나옵니다.

예를 들어 스킬 파일의 인스타 섹션에 이 줄을 추가:
```
- 톤: 격식체 금지. 친구에게 말하듯 편하게. "~해요" 체 사용.
- CTA: 반드시 댓글 유도 문구 포함 (예: "어떻게 생각하세요? 댓글로 알려주세요!")
```

## [DEMO] URL로 콘텐츠 생성 – 진짜 힘을 보여줄게요

[📺 화면: 터미널에서 Claude Code 프롬프트에 "/social-media" 입력 후 URL을 붙여넣는 장면]

이제 이 악기의 진짜 파워를 보여드리겠습니다. URL 넣어서 콘텐츠 자동 생성!

```
/social-media
참고 URL: https://내블로그.com/ai-marketing-post

```

[📺 화면: Claude Code가 WebFetch 툴로 URL 내용을 읽고 → 채널별 콘텐츠를 생성하는 과정]

Claude Code가 웹사이트를 읽고 그 내용을 기반으로 콘텐츠를 만들어줍니다.

블로그 포스트 하나로 인스타 캡션, 쇼츠 스크립트, 블로그 제목이 나왔습니다!

이게 매일 되는 겁니다. 블로그 글 하나 쓰면, 나머지 채널 콘텐츠는 이 악기가 알아서 만들어줍니다.
