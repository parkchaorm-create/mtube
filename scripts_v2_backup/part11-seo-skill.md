## [CONCEPT] SEO 감사란? – 사이트 건강검진

[📺 화면: 슬라이드]

[📊 PPT 지문: SEO 감사 = 내 사이트 건강검진
| 검진 영역 | 점검 항목 | 쉬운 비유 |
|----------|---------|---------|
| 기술적 SEO | 메타 제목/설명 | 이름표가 달려있나? |
| 기술적 SEO | H1 태그 | 간판이 보이나? |
| 기술적 SEO | 이미지 alt 텍스트 | 사진에 설명이 붙어있나? |
| 기술적 SEO | 깨진 링크 | 도로에 구멍 난 곳 없나? |
| 콘텐츠 SEO | 키워드 포함 여부 | 핵심 단어가 자연스럽게 들어있나? |
| 콘텐츠 SEO | 콘텐츠 길이 | 내용이 충분한가? |
| AI SEO (GEO) | AI 인용 최적화 | AI가 내 콘텐츠를 인용하게 만들기 |]

SEO 감사는 내 웹사이트의 검색 최적화 상태를 점검하는 겁니다. 마치 건강검진처럼!

기본 검진 항목부터 보면요, 기술적 SEO라고 하는 건데, 메타 제목과 설명이 있는지, 쉽게 말하면 "이름표가 달려있나?" 확인하는 거고요. H1 태그가 제대로 쓰이고 있는지, "간판이 보이나?" 확인하는 겁니다. 이미지에 alt 텍스트가 있는지, "사진에 설명이 붙어있나?" 그리고 깨진 링크가 있는지, "도로에 구멍 난 곳 없나?" 이런 것들을 체크합니다.

콘텐츠 검진도 있습니다. 키워드가 자연스럽게 포함됐는지, 콘텐츠 길이가 충분한지, 내부 링크가 연결됐는지 확인하죠.

그리고 최신 AI 검진, GEO라는 게 있는데요.

"잠깐, GEO는 또 뭔가요?"

좋은 질문! GEO는 Generative Engine Optimization입니다. 쉽게 말하면 ChatGPT나 Perplexity 같은 AI한테 내 콘텐츠가 인용되게 만드는 겁니다.

요즘 사람들이 구글 대신 AI한테 질문하잖아요? "AI 마케팅이 뭐야?"라고 물었을 때 AI가 "파자마보스 블로그에 따르면..."이라고 인용하면? 수천 명에게 무료로 노출되는 거죠!

이게 최신 트렌드고, 우리 SEO 악기에 이미 포함돼 있습니다.

## SEO 감사 실행 순서

### Step 1: 기술적 SEO 점검
URL 제공받으면 WebFetch로 페이지 읽기.
확인 항목:
- [ ] 메타 제목: 있는지, 50~60자인지, 키워드 포함인지
- [ ] 메타 설명: 있는지, 150~160자인지
- [ ] H1 태그: 딱 하나인지, 키워드 포함인지
- [ ] H2~H3: 논리적 구조인지
- [ ] 이미지 alt 텍스트: 모든 이미지에 있는지
- [ ] 내부 링크: 관련 페이지로 연결됐는지
- [ ] URL 구조: 짧고 키워드 포함인지

### Step 2: 콘텐츠 SEO 점검
- [ ] 콘텐츠 길이: 최소 1,000자 이상인지
- [ ] 키워드 밀도: 2~3% 적정인지
- [ ] 첫 단락에 키워드 포함인지
- [ ] 관련 키워드(LSI) 자연스럽게 포함인지

### Step 3: AI SEO (GEO) 점검
AI 검색 최적화 체크:
- [ ] 명확한 정의 문장 있는지 ("X는 Y입니다" 형태)
- [ ] Q&A 형식 섹션 있는지
- [ ] 리스트 또는 단계별 번호 있는지
- [ ] 전문가 경험담 또는 데이터 인용 있는지

### Step 4: 경쟁사 비교 (선택)
경쟁사 URL 제공 시:
- 키워드 커버리지 비교
- 콘텐츠 깊이 비교
- 내가 놓치고 있는 주제
