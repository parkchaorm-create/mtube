# AI 번역투 제거 프롬프트 (Humanize)

> 이 파일은 **Part 08**에서 사용합니다.
> AI가 생성한 콘텐츠의 번역투를 제거하고 사람이 쓴 것처럼 다듬을 때 사용하세요.

---

```
아래 글에서 AI 번역투를 제거하고, 실제 사람이 쓴 것처럼 자연스럽게 다듬어줘.

## 교정 규칙

### 제거할 것
- "~하는 것이 중요합니다" → "~해야 해요" 또는 "~하세요"
- "~할 수 있습니다" → "~됩니다" 또는 "~돼요"
- "다양한", "효과적인", "혁신적인" 같은 빈 수식어
- "이를 통해", "이러한", "해당" 같은 번역투 접속사
- "~에 대해", "~에 있어서" 같은 불필요한 전치사구
- "확인해 보세요", "살펴보겠습니다" 같은 관망적 표현
- 과도하게 긴 문장 (한 문장에 쉼표 3개 이상이면 쪼개기)

### 추가할 것
- 구어체 표현 ("근데", "진짜", "딱", "바로")
- 짧고 강한 문장
- 자문자답 구조 ("그래서 어떻게 하냐고요? 바로...")
- 구체적 숫자와 예시
- 독자에게 말 거는 표현

### 유지할 것
- 원래 글의 핵심 메시지와 정보
- 글의 전체 구조와 흐름
- 전문 용어 (필요한 경우)
```
