# 데이터 패킷 추출 프롬프트

> 이 파일은 **Part 08**에서 사용합니다.
> Claude Code 채팅창에 아래 내용을 그대로 복사 붙여넣기하세요.

---

```
나의 전문성, 서비스 오퍼, 의사결정 로직을 데이터 뭉치(데이터 패킷)로 변환해줘.

한 번에 하나씩 질문하면서 아래 데이터를 수집해줘:

## 1. 전문성 패킷
- 내가 가장 잘하는 것 3가지
- 업계에서의 차별화 포인트
- 고객이 나를 찾는 핵심 이유
- 내가 해결하는 고객의 가장 큰 고통(Pain Point) 3가지

## 2. 서비스 오퍼 패킷
- 현재 제공하는 서비스/상품 목록
- 각 서비스의 타겟 고객
- 가격대 (공개 가능한 범위)
- 고객 여정: 첫 접점 → 구매 → 재구매 경로

## 3. 의사결정 로직 패킷
- 콘텐츠 주제 선정 기준
- "이건 하고, 이건 안 한다"의 기준
- 고객 응대 시 핵심 원칙
- 경쟁사 대비 절대 양보 안 하는 것

## 4. 콘텐츠 전략 패킷
- 주요 콘텐츠 채널과 각 채널의 역할
- 콘텐츠 발행 빈도
- 가장 반응 좋았던 콘텐츠 유형
- 피하는 콘텐츠 유형과 이유

모든 질문이 끝나면 data-packet.md 파일로 정리해서 저장해줘.
각 패킷은 마크다운 섹션으로 구분하고,
AI가 콘텐츠 생성 시 참고할 수 있는 형식으로 작성해줘.
```

---

## 사용 팁

- 사업이 변할 때마다 업데이트하세요.
