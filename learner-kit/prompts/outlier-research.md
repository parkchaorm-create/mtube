# 아웃라이어 리서치 프롬프트

> 이 파일은 **Part 12**에서 사용합니다.
> 경쟁 콘텐츠 중 비정상적으로 높은 성과를 기록한 "아웃라이어"를 찾아 분석할 때 사용하세요.

---

```
[내 업종/니치] 분야에서 아웃라이어 콘텐츠를 리서치해줘.

## 아웃라이어란?
채널 평균 대비 비정상적으로 높은 조회수/반응을 기록한 콘텐츠.
"왜 이것만 터졌는지"를 분석하면 다음 히트 콘텐츠의 공식이 보입니다.

## 아웃라이어 스코어 공식

아웃라이어 스코어 = 해당 콘텐츠 조회수 / 채널 평균 조회수

- 스코어 3x 이상: 주목할 만한 아웃라이어
- 스코어 5x 이상: 강력한 아웃라이어
- 스코어 10x 이상: 바이럴 아웃라이어 (반드시 분석)

## 리서치 대상
- 플랫폼: [유튜브 / 인스타그램 / 블로그 / 전체]
- 니치/키워드: [내 업종 키워드를 입력하세요]
- 경쟁 채널: [분석할 채널 URL 3~5개를 입력하세요]
- 기간: 최근 [3개월 / 6개월 / 1년]

## 분석 항목

### 1. 아웃라이어 콘텐츠 발굴
각 채널에서 아웃라이어 스코어 3x 이상인 콘텐츠를 찾아주세요:
| 채널명 | 콘텐츠 제목 | 조회수 | 채널 평균 | 아웃라이어 스코어 |
|--------|-----------|--------|---------|-----------------|

### 2. 패턴 분석
발견한 아웃라이어들의 공통점:
- 제목 패턴 (숫자형? 질문형? 부정형?)
- 주제 카테고리
- 콘텐츠 길이/형식
- 발행 시점 (요일, 시간대)
- 썸네일/이미지 스타일

### 3. 후크 분석
아웃라이어 콘텐츠의 첫 문장/첫 15초 분석:
- 어떤 감정을 자극하는가? (호기심, 두려움, 놀라움)
- 어떤 약속을 하는가?
- 독자/시청자에게 어떤 이익을 제시하는가?

### 4. 나의 콘텐츠 기회
위 분석을 바탕으로:
- 내가 만들 수 있는 아웃라이어 후보 제목 10개
- 각 제목의 예상 후크 (첫 문장)
- 추천 발행 우선순위 (1~10위)

결과를 outlier-research-[날짜].md 파일로 저장해줘.
```

---

## 사용 팁

- `[대괄호]` 안의 내용을 여러분의 업종과 채널에 맞게 교체하세요.
- 경쟁 채널 URL은 구독자 수가 비슷하거나 약간 높은 채널을 선택하면 더 유용합니다.
- 매월 1회 정기적으로 실행하면 트렌드 변화를 잡을 수 있습니다.
