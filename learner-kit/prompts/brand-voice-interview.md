# 브랜드 보이스 대화형 추출 프롬프트

> 이 파일은 **Part 08**에서 사용합니다.
> Claude Code 채팅창에 아래 내용을 그대로 복사 붙여넣기하세요.

---

```
나의 브랜드 보이스를 추출해줘.

규칙:
1. 한 번에 하나씩 질문해. 여러 질문을 한꺼번에 하지 마.
2. 내 답변을 듣고 꼬리 질문으로 더 깊이 파고들어.
3. 95% 확신이 들 때까지 질문을 계속해.
4. 최소 10개 이상의 질문을 해.
5. 끝나면 brand-voice-guide.md 파일로 저장해.

추출해야 할 항목:
- 브랜드 성격 키워드 5개
- 말투/톤 (격식체/비격식체, 에너지 레벨)
- 시그니처 표현 (내가 자주 쓰는 말)
- 절대 사용 금지 표현
- 독자/고객 호칭
- 글쓰기 구조 패턴
- 이모지 사용 여부 및 방식
- 유머 사용 방식

첫 번째 질문부터 시작해줘.
```

---

## 사용 팁

- 질문에 솔직하게, 구체적으로 답변할수록 정확한 결과가 나옵니다.
- 기존에 잘 쓴 글이 있다면 함께 붙여넣기하면 더 좋습니다.
- 생성된 `brand-voice-guide.md`는 CLAUDE.md에서 참조하세요.
