# 주간 콘텐츠 플랜 프롬프트

> 이 파일은 **Part 13**에서 사용합니다.
> 매주 월요일에 실행하면 한 주간의 콘텐츠 계획이 자동으로 생성됩니다.

---

```
이번 주 콘텐츠 플랜을 만들어줘.

## 기본 정보
- 브랜드: [내 브랜드명]
- 이번 주 핵심 주제/키워드: [이번 주 다루고 싶은 주제]
- 진행 중인 캠페인: [있다면 입력, 없으면 "없음"]
- 특별한 일정: [출시, 이벤트, 시즌 등]

## 채널별 발행 일정

### 월요일
- [ ] 인스타그램: 주간 시작 포스트
- [ ] 블로그: 롱폼 콘텐츠 발행

### 화요일
- [ ] 인스타그램: 캐러셀/교육 콘텐츠

### 수요일
- [ ] 뉴스레터: 주간 뉴스레터 발송
- [ ] 인스타그램: 릴스/쇼츠

### 목요일
- [ ] 인스타그램: 사례/후기 콘텐츠
- [ ] X(트위터): 인사이트 스레드

### 금요일
- [ ] 인스타그램: 주간 마무리/CTA 포스트
- [ ] 링크드인: 주간 회고 포스트

## 각 콘텐츠별 작성 요청

각 콘텐츠마다 다음을 포함해줘:
1. 제목/후크 (3가지 옵션)
2. 핵심 메시지 (한 문장)
3. CTA
4. 해시태그 (인스타용)
5. 발행 최적 시간대

## 주간 목표
- 도달(Reach) 목표: [숫자 또는 "전주 대비 10% 증가"]
- 참여율 목표: [숫자 또는 "전주 대비 유지"]
- 전환 목표: [리드, 구매, 구독 등]

결과를 plan-week-[날짜].md 파일로 저장해줘.
```

---

## 사용 팁

- `[대괄호]` 안의 내용을 여러분 상황에 맞게 교체하세요.
- 발행 일정은 여러분의 채널과 역량에 맞게 조정하세요. 무리하지 마세요!
- 아웃라이어 리서치 결과와 함께 사용하면 더 효과적입니다.
- `@brand-voice-guide.md`를 함께 참조하면 브랜드 톤에 맞는 콘텐츠가 나옵니다.
