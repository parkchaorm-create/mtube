## 슬래시 명령어

| 명령어 | 기능 | 언제 쓰나? |
|--------|------|-----------|
| `/plan` | Plan Mode 전환 (리허설 모드) | 복잡한 작업 전, 계획 먼저 보고 싶을 때 |
| `/model` | AI 모델 변경 | 택시 등급 바꾸고 싶을 때 |
| `/clear` | 대화 초기화 (책상 비우기) | 응답이 느려지거나 이상해질 때 |
| `/compact` | 대화 요약 후 압축 | 컨텍스트가 많이 쌓였을 때 |
| `/cost` | 현재까지 사용한 비용 확인 | 택시 미터기 확인할 때 |
| `/help` | 도움말 보기 | 뭘 해야 할지 모를 때 |
| `/doctor` | Claude Code 상태 진단 | 뭔가 작동이 안 될 때 |

## 모델 비교 (택시 등급)

| 모델 | 비유 | 용도 | 비용 |
|------|------|------|------|
| Opus | 리무진 | 복잡한 설계, 전략 수립 | 비쌈 |
| Sonnet | 일반 택시 | 일상 콘텐츠 제작 (기본값) | 보통 |
| Haiku | 경차 | 간단한 반복 작업, 교정 | 저렴 |

---

## 비용 절약 팁

1. **`/clear` 자주 사용** — 대화가 길어지면 비용도 올라갑니다
2. **필요한 파일만 참조** — 불필요한 파일 넣지 마세요
3. **가벼운 작업은 Haiku** — `/model`로 변경
4. **Plan Mode 활용** — 삽질 방지 = 비용 절약
5. **스킬 파일 튜닝** — 한 번에 좋은 결과 = 재생성 비용 절약
