# 터미널 5대 명령어 치트시트

> 이 파일은 **Part 04 (터미널 기초)**에서 사용합니다.
> 인쇄해서 모니터 옆에 붙여두세요!

---

## 핵심 5대 명령어

| # | 명령어 | 하는 일 | 비유 |
|---|--------|--------|------|
| 1 | `pwd` | 현재 위치 확인 | "지금 나 어디에 있지?" |
| 2 | `ls` (맥) / `dir` (윈도우) | 파일 목록 보기 | "이 복도에 뭐가 있지?" |
| 3 | `cd [폴더명]` | 폴더 이동 | "저 복도로 이동!" |
| 4 | `mkdir [폴더명]` | 새 폴더 만들기 | "새 방 만들기!" |
| 5 | `claude` | Claude Code 실행 | "음향 장비 켜기!" |

---

## 자주 쓰는 조합

```bash
# 프로젝트 폴더로 이동
cd Documents/my-ai-marketing

# 현재 위치 확인
pwd

# 파일 목록 보기
ls          # 맥
dir         # 윈도우

# 숨김 파일까지 보기
ls -la      # 맥
dir /a      # 윈도우

# 새 폴더 만들기
mkdir context

# 중간 폴더도 같이 만들기
mkdir -p .claude/commands

# 한 단계 위로 이동
cd ..

# 홈 폴더로 이동
cd ~

# Claude Code 실행
claude
```

---

## 경로 읽는 법

```
/Users/내이름/Documents/my-ai-marketing
  │       │         │              │
  루트   사용자    문서 폴더    프로젝트 폴더
```

- `/` 또는 `\` : 폴더 구분자
- `~` : 홈 폴더 (내 사용자 폴더)
- `.` : 현재 폴더
- `..` : 상위 폴더
- `.폴더명` : 숨김 폴더 (예: `.claude`)

---

## 이것만 기억하세요

1. **길 잃었을 때** → `pwd` 입력
2. **뭐가 있는지 모를 때** → `ls` 또는 `dir` 입력
3. **이동하고 싶을 때** → `cd 폴더명` 입력
4. **폴더 만들고 싶을 때** → `mkdir 폴더명` 입력
5. **Claude Code 켜고 싶을 때** → `claude` 입력

> 잘못 입력해도 컴퓨터가 망가지지 않습니다! 겁내지 마세요!
