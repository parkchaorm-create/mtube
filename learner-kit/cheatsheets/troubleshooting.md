# 자주 나오는 에러 TOP 10 해결법

> 모든 파트에서 참고할 수 있는 트러블슈팅 가이드입니다.
> 에러가 나면 당황하지 말고 여기서 찾아보세요!

---

## 1. "claude: command not found"

**증상**: 터미널에 `claude` 입력했는데 명령어를 못 찾음

**해결법**:
```bash
# Node.js 설치 확인
node --version

# Claude Code 재설치
npm install -g @anthropic-ai/claude-code

# 터미널 재시작 후 다시 시도
claude
```

**원인**: Claude Code가 설치 안 됐거나, 터미널을 설치 후 재시작 안 한 경우

---

## 2. "API key not found" / 인증 에러

**증상**: Claude Code 실행했는데 API 키 관련 에러

**해결법**:
```bash
# Claude Code에서 로그인
claude login

# 브라우저가 열리면 Anthropic 계정으로 로그인
# Max 플랜 구독 중인지 확인
```

**원인**: 로그인이 안 됐거나, 구독이 만료된 경우

---

## 3. 스킬 명령어가 안 됨 ("/social-media 를 인식 못함")

**증상**: `/social-media` 입력했는데 스킬이 실행 안 됨

**해결법**:
```bash
# 1. 현재 위치가 프로젝트 폴더인지 확인
pwd

# 2. .claude/commands/ 폴더에 파일이 있는지 확인
ls .claude/commands/

# 3. 파일명 확인 (social-media.md 인지)
# 주의: 띄어쓰기, 대소문자 확인!

# 4. Claude Code를 프로젝트 폴더에서 실행했는지 확인
cd [프로젝트 폴더 경로]
claude
```

**원인**: 프로젝트 폴더 바깥에서 Claude Code를 실행했거나, 파일 위치/이름이 틀린 경우

---

## 4. "컨텍스트가 너무 깁니다" / 응답이 느려짐

**증상**: 대화가 길어지면서 응답이 느려지거나 이상해짐

**해결법**:
```
# Claude Code 안에서 입력
/clear
```

그리고 새로운 대화로 시작하세요.

**원인**: 지휘자 책상(컨텍스트 윈도우)이 꽉 찬 경우. `/clear`로 리셋!

---

## 5. "Permission denied" / 권한 에러

**증상**: 파일 생성이나 수정 시 권한 에러

**해결법**:
```bash
# 맥: 터미널 권한 확인
# 시스템 설정 → 개인 정보 보호 및 보안 → 전체 디스크 접근 → 터미널 허용

# 윈도우: 관리자 권한으로 PowerShell 실행
# PowerShell을 우클릭 → "관리자 권한으로 실행"

# npm 글로벌 설치 권한 문제
sudo npm install -g @anthropic-ai/claude-code  # 맥
```

**원인**: 터미널이나 폴더에 쓰기 권한이 없는 경우

---

## 6. AI가 브랜드 톤을 안 따름

**증상**: CLAUDE.md를 만들었는데 AI가 일반적인 톤으로 답변

**해결법**:
1. CLAUDE.md 파일이 프로젝트 폴더 **최상위**에 있는지 확인
2. 파일명이 정확히 `CLAUDE.md`인지 확인 (전부 대문자!)
3. Claude Code를 해당 프로젝트 폴더에서 실행했는지 확인
4. `/clear` 후 다시 시도

**원인**: CLAUDE.md 위치가 잘못됐거나, 다른 폴더에서 Claude Code를 실행한 경우

---

## 7. WebFetch가 안 됨 (URL 읽기 실패)

**증상**: URL을 넣었는데 페이지를 못 읽음

**해결법**:
- URL이 정확한지 확인 (복사 붙여넣기)
- 해당 사이트가 로그인 필요한 페이지인지 확인
- 유튜브 URL은 자막이 있는 영상만 가능
- 사이트가 봇 차단하는 경우 → 텍스트를 직접 복사 붙여넣기

**원인**: URL 오류, 로그인 필요 사이트, 봇 차단 사이트

---

## 8. 결과물 품질이 안 좋음

**증상**: AI가 만든 콘텐츠가 기대에 못 미침

**해결법**:
1. **스킬 파일 튜닝**: 구체적인 지시사항 추가
2. **예시 추가**: "좋은 예 vs 나쁜 예"를 스킬 파일에 넣기
3. **피드백 주기**: "이 부분을 이렇게 바꿔줘"로 개선 요청
4. **모델 변경**: 복잡한 작업은 Opus로 변경 (`/model`)
5. **brand-voice-guide.md** 연결 확인

**원인**: 스킬 파일의 지시가 모호하거나, 브랜드 보이스가 미연결

---

## 9. "npm ERR!" / Node.js 설치 에러

**증상**: npm install 시 에러 발생

**해결법**:
```bash
# Node.js 버전 확인 (18 이상 필요)
node --version

# 버전이 낮으면 nodejs.org에서 최신 LTS 버전 재설치

# npm 캐시 정리 후 재설치
npm cache clean --force
npm install -g @anthropic-ai/claude-code
```

**원인**: Node.js 버전이 오래됐거나, 설치가 불완전한 경우

---

## 10. 파일이 저장 안 됨 / 날아감

**증상**: Claude Code가 만든 파일이 어디 갔는지 모르겠음

**해결법**:
```bash
# 최근 생성된 파일 찾기 (맥)
find . -name "*.md" -mmin -30

# 현재 폴더 구조 보기
ls -la
ls -la content/

# Claude Code에서 물어보기
"방금 생성한 파일 경로를 알려줘"
```

**원인**: 파일 저장 경로를 확인하지 않은 경우. content/ 폴더나 프로젝트 루트를 확인하세요.
