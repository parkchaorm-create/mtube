## 데이터 제공 방법 안내 (먼저 이 안내를 해주세요)
데이터 없이 실행하면: "GA4에서 CSV 내보내기 후 @파일명.csv 로 제공해주세요"
데이터 있으면: 바로 분석 시작

## 분석 순서

### 1단계: 데이터 개요
제공된 CSV 파일의 전체 구조 파악 후:
- 분석 기간
- 포함된 지표 목록
- 이상치(비정상적으로 높거나 낮은 값) 먼저 체크

### 2단계: 핵심 지표 분석
다음 지표를 분석하고 전월 또는 전주 대비 증감:
- 총 방문자 수 (Users)
- 세션 수 (Sessions)
- 신규 방문자 vs 재방문 비율
- 이탈률 (Bounce Rate)
- 평균 세션 시간
- 전환 목표 달성률 (설정돼 있다면)

### 3단계: 트래픽 소스 분석
어디서 방문자가 오는지:
- 검색 (Organic) vs 직접 vs 소셜 vs 추천 비율
- 소셜 채널별 비교
- 가장 효과적인 트래픽 소스 특정

### 4단계: 콘텐츠 성과 분석
인기 페이지와 저성과 페이지:
- TOP 10 페이지 (방문자 기준)
- 체류 시간 긴 페이지 (좋은 콘텐츠 지표)
- 이탈률 높은 페이지 (개선 필요)

### 5단계: 실행 전략 도출
데이터를 기반으로:
- 이번 달 놓치고 있는 기회
- 다음 달 집중해야 할 채널/콘텐츠
- 즉시 중단해야 할 비효율 활동

|------|---------|---------|------|------|
| 방문자 | | | | |
| 이탈률 | | | | |

**💡 3가지 핵심 발견**
(데이터 근거 + 의미 해석 + 해야 할 행동)
1. 
2. 
3. 

**🚀 다음 달 실행 계획 (TOP 5)**

**⚠️ 지금 당장 멈춰야 할 것**
[성과 없는 활동 + 이유]

**📅 다음 달 콘텐츠 추천 주제 5가지**
(데이터에서 발견한 기회 기반)
