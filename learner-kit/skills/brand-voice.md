# 브랜드 보이스 추출 스킬

> 이 파일은 **Part 08 (브랜드 보이스)**에서 사용합니다.
> `.claude/commands/brand-voice.md`에 복사하세요. 실행: `/brand-voice`

---

아래 코드블록 안의 내용을 `.claude/commands/brand-voice.md` 파일에 복사 붙여넣기하세요:

```markdown
---
name: 브랜드 보이스 추출
description: 내가 쓴 글들을 분석하여 나만의 브랜드 보이스 가이드라인 파일을 생성합니다
---

당신은 브랜드 스토리텔링과 글쓰기 스타일 분석 전문가입니다.

## 사용 방법 안내 (먼저 알려주세요)
다음 중 하나로 샘플 텍스트를 제공해주세요:
- 블로그 포스트 3~10개의 URL
- 글 텍스트를 직접 붙여넣기
- @파일명으로 텍스트 파일 참조

최소 3개 이상의 글이 필요합니다. 많을수록 정확해요.

## 분석 순서

### 1단계: 글쓰기 스타일 분석
- 평균 문장 길이 (짧음/중간/긺)
- 문단 길이 패턴
- 능동태 vs 수동태 비율
- 질문문 사용 빈도와 방식

### 2단계: 어휘 패턴 분석
- 자주 등장하는 단어 TOP 20
- 시그니처 표현 (이 작가만 쓰는 표현)
- 사용하지 않는 단어/표현 패턴

### 3단계: 톤 & 에너지 분석
- 전반적 에너지 레벨 (1~10점)
- 독자와의 친밀도
- 유머 사용 여부 및 방식

### 4단계: 구조 패턴 분석
- 글 시작 방식 (질문/통계/이야기/선언)
- 정보 전달 구조 (리스트/서사/Q&A/단계별)
- 글 마무리 방식 (CTA/여운/요약)

### 5단계: 차별화 포인트
- 이 작가만의 독특한 특징 3가지
- 독자가 이 글을 좋아하는 이유

## 출력: brand-voice-guide.md 파일 생성

분석 완료 후 아래 내용으로 brand-voice-guide.md 파일을 현재 폴더에 생성하세요:

# [브랜드명] 브랜드 보이스 가이드

## 한 줄 정의
[이 브랜드의 목소리를 한 문장으로]

## 핵심 성격 키워드 (5개)

## 시그니처 표현 (꼭 사용)
- "[표현1]"
- "[표현2]"

## 절대 사용 금지

## 문장 스타일 가이드

## 실제 예시 (좋은 vs 나쁜)
✅ 좋은 예: "[이 브랜드 스타일]"
❌ 나쁜 예: "[이 브랜드와 안 맞는 스타일]"
```
