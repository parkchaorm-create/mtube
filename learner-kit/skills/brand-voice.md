# 브랜드 보이스 추출 스킬

> 저장 위치: `.claude/commands/brand-voice.md`
> 실행 방법: Claude Code에서 `/brand-voice` 입력

---

```markdown
---
name: 브랜드 보이스 추출
description: 내가 쓴 글들을 분석하여 나만의 글쓰기 스타일과 브랜드 보이스를 추출하고 파일로 저장합니다
---

당신은 브랜드 스토리텔링과 글쓰기 스타일 분석 전문가입니다.
제공된 샘플 텍스트를 분석하여 작성자의 고유한 브랜드 보이스를 추출합니다.

## 사용 방법
내가 쓴 글 3~10개를 아래 방법으로 제공하세요:
- 블로그 포스트 URL들을 나열
- 유튜브 영상 스크립트 파일 (@파일명)
- 인스타그램 캡션 텍스트 붙여넣기
- 뉴스레터 아카이브 URL

## 분석 항목

### 1. 문장 스타일 분석
- 평균 문장 길이 (짧음/중간/긺)
- 능동태 vs 수동태 비율
- 단락 구조 패턴
- 질문문 사용 빈도

### 2. 어휘 패턴 분석
- 자주 쓰는 단어 TOP 20
- 특징적인 표현 패턴
- 피하는 단어/표현
- 전문 용어 사용 수준

### 3. 톤 & 감정 분석
- 전반적인 에너지 수준 (높음/중간/낮음)
- 감정 색깔 (따뜻함/차가움/유머/진지함)
- 독자와의 거리감 (친근함/전문적/권위적)

### 4. 구조 패턴 분석
- 자주 쓰는 도입부 패턴
- 정보 전달 방식 (스토리텔링/리스트/Q&A)
- 결말 패턴 (CTA 유형)

### 5. 차별화 포인트
- 이 작성자만의 독특한 특징
- 독자가 기억하는 시그니처 표현
- 경쟁자와 다른 점

## 출력: 브랜드 보이스 가이드라인 파일

분석 완료 후 `brand-voice-guide.md` 파일을 생성하세요:

---

# [브랜드명] 브랜드 보이스 가이드

## 한 줄 요약
[이 브랜드의 목소리를 한 문장으로]

## 핵심 특성 3가지
1. [특성1]: [설명]
2. [특성2]: [설명]
3. [특성3]: [설명]

## 시그니처 표현 (꼭 써야 할 것들)
- "[표현1]"
- "[표현2]"
- "[표현3]"

## 절대 쓰지 않는 것들
- [금지 표현1]
- [금지 표현2]

## 문장 구조 템플릿
도입부: "[패턴]"
본문: "[구조]"
마무리: "[패턴]"

## 실제 예시
좋은 예: "[샘플 문장]"
나쁜 예: "[샘플 문장]"

## AI 프롬프트용 요약 (복붙하세요)
"""
이 글을 [브랜드명] 스타일로 작성해주세요.
- 톤: [톤 설명]
- 문장: [문장 스타일]
- 꼭 포함: [시그니처 표현]
- 피할 것: [금지 표현]
"""
```

---

## 실행 예시

**입력**: 
```
/brand-voice 
아래 내 블로그 글들을 분석해줘:
@blog-post-1.txt
@blog-post-2.txt
https://내블로그.com/post3
```

**예상 출력**:
```
# 파자마보스 브랜드 보이스 가이드

## 한 줄 요약
"40대 문과 선배가 이미 다 삽질해봤으니까, 딱 따라만 오세요"

## 핵심 특성 3가지
1. 공감형 도입: 항상 독자의 고충에서 시작
2. 단계별 명확성: "첫 번째... 두 번째..."로 구조화
3. 자신감 있는 결말: "확실히 해드리겠습니다"로 마무리

## 시그니처 표현
- "보스님들"
- "삽질"
- "복붙하세요"
- "이게 전부예요"
```
