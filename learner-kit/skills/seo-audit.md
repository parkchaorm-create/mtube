먼저 물어보세요:
1. "감사할 웹사이트 URL을 알려주세요"
2. "타겟 키워드 3~5개를 알려주세요"
3. "경쟁사 URL이 있다면 알려주세요 (선택)"

## SEO 감사 실행 순서

### Step 1: 기술적 SEO 점검
URL 제공받으면 WebFetch로 페이지 읽기.
확인 항목:
- [ ] 메타 제목: 있는지, 50~60자인지, 키워드 포함인지
- [ ] 메타 설명: 있는지, 150~160자인지
- [ ] H1 태그: 딱 하나인지, 키워드 포함인지
- [ ] H2~H3: 논리적 구조인지
- [ ] 이미지 alt 텍스트: 모든 이미지에 있는지
- [ ] 내부 링크: 관련 페이지로 연결됐는지
- [ ] URL 구조: 짧고 키워드 포함인지

### Step 2: 콘텐츠 SEO 점검
- [ ] 콘텐츠 길이: 최소 1,000자 이상인지
- [ ] 키워드 밀도: 2~3% 적정인지
- [ ] 첫 단락에 키워드 포함인지
- [ ] 관련 키워드(LSI) 자연스럽게 포함인지

### Step 3: AI SEO (GEO) 점검
AI 검색 최적화 체크:
- [ ] 명확한 정의 문장 있는지 ("X는 Y입니다" 형태)
- [ ] Q&A 형식 섹션 있는지
- [ ] 리스트 또는 단계별 번호 있는지
- [ ] 전문가 경험담 또는 데이터 인용 있는지

### Step 4: 경쟁사 비교 (선택)
경쟁사 URL 제공 시:
- 키워드 커버리지 비교
- 콘텐츠 깊이 비교
- 내가 놓치고 있는 주제
