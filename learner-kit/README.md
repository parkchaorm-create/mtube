# 파자마보스 풀코스 – 왕초보 학습 키트

영상을 보면서 함께 사용하는 자료 모음입니다.

## 폴더 구조

```
learner-kit/
├── prompts/        ← 영상에서 사용하는 프롬프트 전체 (복붙용)
├── skills/         ← .claude/commands/ 에 넣을 스킬 파일 완성본
├── slides/         ← 강의 PPT 대본 및 슬라이드 텍스트
├── cheatsheets/    ← 명령어 치트시트, 요금제 비교표 등 한장 요약
└── exercises/      ← 파트별 실습 체크리스트
```

## 사용 방법
1. `skills/` 폴더의 파일들을 여러분의 `.claude/commands/` 폴더에 복사하세요.
2. `prompts/` 폴더는 영상 순서대로 번호가 매겨져 있습니다.
3. `cheatsheets/` 는 영상 보면서 옆에 펼쳐두세요.
4. `exercises/` 로 각 파트 완료 여부를 체크하세요.
