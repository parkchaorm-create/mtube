# 파자마보스 AI 마케팅 오케스트라 – 학습 키트

> 구독+좋아요+댓글 남기신 분들에게 드리는 학습 키트입니다

이 학습 키트는 영상을 따라하면서 바로 복사 붙여넣기할 수 있도록 구성했습니다.

---

## 폴더 구조

```
learner-kit/
├── prompts/          ← Claude Code에 복사 붙여넣기할 프롬프트 모음
├── skills/           ← .claude/commands/에 복사할 스킬 파일
├── templates/        ← 프로젝트 세팅 시 필요한 템플릿
└── cheatsheets/      ← 빠른 참조용 치트시트
```

---

## 사용법

### 1. 프롬프트 (prompts/)
Claude Code 채팅창에 파일 내용을 **그대로 복사 붙여넣기**하세요.

### 2. 스킬 (skills/)
프로젝트 폴더의 `.claude/commands/` 안에 파일을 복사하세요.
```bash
# 예시: 소셜미디어 스킬 복사
cp learner-kit/skills/social-media.md .claude/commands/social-media.md
```
스킬 파일명이 곧 명령어입니다. `social-media.md` → `/social-media`로 실행!

### 3. 템플릿 (templates/)
프로젝트 폴더에 맞게 내용을 수정해서 사용하세요. `[대괄호]` 안의 내용을 여러분 브랜드에 맞게 교체하면 됩니다.

### 4. 치트시트 (cheatsheets/)
작업 중 막힐 때 빠르게 참고하세요. 인쇄해서 모니터 옆에 붙여두면 편합니다.
