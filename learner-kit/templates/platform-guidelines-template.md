# 플랫폼 가이드라인 템플릿

> 이 파일은 **Part 08**에서 사용합니다.

---

아래 코드블록 안의 내용을 `context/platform-guidelines.md` 파일에 복사 붙여넣기하세요.
`[대괄호]` 안의 내용을 여러분 상황에 맞게 교체하세요:

```markdown
# 채널별 플랫폼 가이드라인

## 인스타그램
- **계정**: @[아이디]
- **발행 빈도**: 주 [N]회
- **최적 발행 시간**: [예: 오전 8시, 오후 12시, 오후 7시]
- **콘텐츠 유형 비율**:
  - 교육/정보: [40]%
  - 참여/소통: [30]%
  - 프로모션: [20]%
  - 비하인드/일상: [10]%
- **캡션 길이**: [150~300]자
- **해시태그 전략**:
  - 브랜드 해시태그: #[내브랜드] #[시그니처태그]
  - 니치 해시태그 (5개): #[키워드1] #[키워드2] ...
  - 인기 해시태그 (3개): #[대중키워드1] ...
  - 영어 해시태그 (5개): #[English1] ...
- **이모지 사용**: [3~5개, 자연스럽게]
- **CTA 패턴**: [예: "댓글로 알려주세요!", "저장해두세요!"]
- **비주얼 스타일**: [예: 밝은 톤, 미니멀, 텍스트 오버레이]

## 유튜브
- **채널명**: [채널명]
- **발행 빈도**: 주 [N]회
- **영상 길이**: 롱폼 [10~20]분 / 쇼츠 [30~60]초
- **썸네일 스타일**: [예: 얼굴 클로즈업 + 큰 텍스트 + 강렬한 배경색]
- **제목 패턴**: [예: 숫자형 "3가지 방법", 질문형 "왜 안 되는 걸까?"]
- **설명란 구조**:
  - 첫 줄: [핵심 요약]
  - 타임스탬프
  - 관련 링크
  - 소셜미디어 링크
- **태그**: [주요 태그 10~15개]

## 블로그
- **플랫폼**: [예: 워드프레스, 티스토리, 네이버 블로그]
- **발행 빈도**: 주 [N]회
- **글 길이**: 최소 [1,500]자
- **SEO 체크리스트**:
  - [ ] 타겟 키워드가 제목에 포함
  - [ ] 메타 설명 150~160자
  - [ ] H2 태그 5~7개 사용
  - [ ] 이미지 alt 텍스트 포함
  - [ ] 내부 링크 2~3개 포함
- **글 구조**: [예: 후크 → 문제 제기 → 해결책 → CTA]

## 뉴스레터
- **플랫폼**: [예: Beehiiv, Mailchimp, Stibee]
- **발행 빈도**: 주 [N]회
- **발행 요일/시간**: [예: 화요일 오전 9시]
- **이메일 제목 패턴**: [예: 숫자형, 질문형, 긴급형]
- **본문 길이**: [600~800]자
- **CTA**: [1개, 명확하게]
- **구독자 혜택**: [예: 템플릿, 체크리스트, 무료 가이드]

## X (트위터)
- **계정**: @[아이디]
- **발행 빈도**: 주 [N]회
- **트윗 스타일**: [예: 인사이트 요약, 스레드, 의견]
- **스레드 길이**: [5~7개 트윗]
- **해시태그**: [최대 2개]

## 링크드인
- **프로필**: [URL]
- **발행 빈도**: 주 [N]회
- **포스트 스타일**: [예: 개인 경험 + 인사이트, 업계 트렌드 분석]
- **본문 길이**: [400~500]자
- **해시태그**: [3~5개]
```
