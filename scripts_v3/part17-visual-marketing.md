# Part 17: 비주얼 마케팅 팀 – 카드뉴스와 썸네일까지 자동 생성

## 메타 정보
- **예상 길이**: 12분 (약 3,000~3,600자)
- **화면 구성**: 웹캠 → PPT(비주얼 팀 구조) → 브라우저(API 키 발급) → 안티그래비티(MCP 설정) → 안티그래비티(스킬 생성) → 결과물 확인 → 웹캠
- **핵심 비유**: 비주얼 마케팅 팀 = 오케스트라의 무대 미술팀, 나노바나 MCP = 무대 위 조명 장비, API 키 = 주문 전화번호
- **필요 도구**: Google AI Studio API Key, 나노바나(Nanobanana) MCP

아닙니다. 우리 오케스트라에 무대 미술팀을 추가하면 됩니다. AI가 카드뉴스도 만들고, 썸네일도 만들고, 이것까지 전부 자동으로 생성해줍니다.

## [CONCEPT] 비주얼 마케팅 팀이란?

[📺 화면: 슬라이드]
[📊 PPT 지문: 비주얼 마케팅 팀 구조 — 나노바나 비주얼 스킬: 인스타 카드뉴스 생성 + 유튜브 썸네일 생성 / 핵심: 제미나이 3.0 나노바나 프로 이미지 모델 활용]

우리 오케스트라에 비유하면, 텍스트 팀이 곡을 연주하는 연주자들이라면, 비주얼 팀은 무대 세트와 조명을 담당하는 미술팀입니다.

아무리 연주가 좋아도 무대가 허전하면 관객이 빠지잖습니까. 마찬가지로 아무리 글이 좋아도 비주얼이 없으면 소셜미디어에서 스크롤에 묻히거든요.

이 비주얼 팀은 스킬, 그러니까 악기로 만듭니다. "에이전트가 아니라 왜 스킬이냐?"라고 궁금하실 수 있는데요. 이미지 생성은 판단이 필요한 게 아니라 정해진 템플릿에 따라 반복 실행하는 작업이기 때문입니다. 에이전트는 판단이 필요한 전문 단원이고, 스킬은 반복 실행하는 악기라고 했었죠. 이미지 생성은 악기에 해당합니다.

## [DEMO] API 키 발급 – 이미지 생성 엔진 연결

[📺 화면: 웹캠 단독]

비주얼 팀이 이미지를 만들려면 이미지 생성 엔진이 필요합니다. 나노바나(Nanobanana)라는 도구를 쓸 건데요, 이건 제미나이 3.0 프로 모델로 이미지를 만들어주는 도구입니다.

먼저 API 키를 발급받아야 합니다. API 키, 쉽게 말하면 주문 전화번호입니다. 이 번호가 있어야 이미지 생성 서비스에 전화를 걸 수 있는 거죠.

중요한 거 하나. Google AI Studio API 키는 무료로 발급받을 수 있습니다. 추가 비용 없거든요.

[📺 화면: 브라우저 → Google AI Studio 접속]

브라우저를 열고 "aistudio.google.com"에 접속하세요. 구글 계정으로 로그인하시면 됩니다.

[📺 화면: Google AI Studio 메인 화면 — 왼쪽 사이드바]

로그인하셨으면 왼쪽 사이드바에서 "Get API key" 버튼을 클릭하세요.

[📺 화면: API Key 관리 페이지 — "Create API key" 버튼]

"Create API key" 버튼을 클릭하고요. 키 이름은 아무거나 입력하시면 됩니다. 저는 "nanobanana"라고 적겠습니다. 프로젝트를 선택하고 "Create" 버튼을 누르세요.

[📺 화면: 생성된 API Key 표시 — 복사 버튼]

API 키가 생성됐습니다. 여기서 중요한 게 있거든요. 이 API 키는 절대 외부에 유출하시면 안 됩니다. 유심칩처럼 소중하게 보관해야 하는 정보입니다. 복사 버튼을 눌러서 키를 복사해주세요.

## [DEMO] 나노바나 MCP 연결

[📺 화면: 안티그래비티 화면]

이제 복사한 API 키를 우리 프로젝트에 연결하겠습니다.

MCP가 뭐냐고요? 쉽게 말하면 외부 도구와 Claude Code를 연결하는 케이블입니다. 오케스트라에서 음향 장비를 외부 스피커와 연결하는 케이블과 같은 거죠.

[📺 화면: 안티그래비티 — 프로젝트 루트에 .mcp.json 파일]

프로젝트 루트 폴더에 ".mcp.json" 파일을 만들겠습니다. 이 파일이 Claude Code에게 "이 프로젝트에서는 이런 외부 도구를 쓸 수 있어"라고 알려주는 역할을 합니다.

근데 보스님들, 이 JSON 파일을 직접 작성하면 따옴표나 중괄호를 빼먹기 쉽거든요. 그래서 Claude Code한테 시키는 걸 추천합니다. 채팅창에 "나노바나 MCP 설정 파일 만들어줘. API 키는 [아까 복사한 키]야."라고 입력하시면 알아서 .mcp.json 파일을 만들어줍니다. 직접 만들고 싶은 분들을 위해 코드도 보여드릴게요.

```json
{
  "mcpServers": {
    "nanobanana": {
      "command": "npx",
      "args": ["-y", "@anthropic-ai/nanobanana-mcp"],
      "env": {
        "GEMINI_API_KEY": "여기에_복사한_API_키_붙여넣기"
      }
    }
  }
}
```

이 코드를 그대로 복사해서 붙여넣으시고요, "여기에_복사한_API_키_붙여넣기" 부분만 아까 복사한 API 키로 교체하시면 됩니다.

저장하셨으면 안티그래비티를 한번 재시작해주세요. 그래야 MCP 연결이 활성화됩니다.

[📺 화면: 안티그래비티 터미널 — /mcp 명령어 입력]

확인해볼게요. Claude Code 채팅창에 "/mcp"라고 입력하시면 연결된 MCP 서버 목록이 나옵니다. 여기에 "nanobanana"가 보이면 성공입니다.

## [DEMO] 비주얼 스킬 생성 – 카드뉴스 & 썸네일

[📺 화면: 안티그래비티 — 스킬 폴더 구조]

이제 비주얼 스킬을 만들겠습니다. 직접 만들 필요 없거든요. Claude Code한테 시키면 됩니다.

Claude Code 채팅창에 이렇게 입력하세요:

"나노바나 MCP를 활용해서 인스타 카드뉴스 5장과 유튜브 썸네일을 생성하는 비주얼 마케팅 스킬을 만들어줘. 카드뉴스는 인포그래픽 스타일로, 썸네일은 1280x720 사이즈로 만들어줘."

[📺 화면: Claude Code가 스킬 파일을 자동 생성하는 과정]

Claude Code가 스킬 파일을 자동으로 만들어주고 있습니다. SKILL.md 파일에는 카드뉴스와 썸네일을 생성하는 전체 워크플로우가 들어가 있거든요. 레퍼런스 폴더에는 카드뉴스 가이드와 썸네일 가이드도 자동으로 생성됐습니다.

## [DEMO] 테스트 – 이미지 자동 생성

[📺 화면: 안티그래비티 터미널]

스킬이 만들어졌으니 테스트해보겠습니다.

Claude Code 채팅창에 이렇게 입력하세요:

"아까 만든 블로그 포스트 내용을 기반으로 인스타 카드뉴스 5장과 유튜브 썸네일 1장을 만들어줘."

[📺 화면: 이미지 생성 진행 중 → 결과물 폴더에 파일 생성]

나노바나가 제미나이 3.0 프로 모델로 이미지를 생성하고 있습니다.

[📺 화면: 생성된 카드뉴스 이미지 5장 순서대로 열어서 확인]

보세요. 카드뉴스 5장이 생성됐습니다. 내용을 전부 이해하고 인포그래픽 스타일로 만들어줬거든요. 폰트도 깔끔하고, 색감도 브랜드 톤에 맞게 들어가 있습니다.

[📺 화면: 유튜브 썸네일 이미지 확인]

유튜브 썸네일도 생성됐습니다. 제목이 들어가 있고, 시각적으로도 클릭하고 싶게 만들어져 있죠.

물론 100% 완벽하지는 않을 수 있습니다. 로고가 약간 깨지거나, 레이아웃을 미세 조정해야 할 수도 있거든요. 하지만 기본 틀을 AI가 만들어주니까 거기서 살짝 수정하는 건 10분이면 충분합니다. 디자이너 없이 이 정도 퀄리티가 나온다는 게 핵심이죠.
