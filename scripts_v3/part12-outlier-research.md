- **특별 요소**: 아웃라이어 스코어 공식, 소규모 크리에이터 우선 탐색, 프리스코어링 게이트

## [HOOK]

[📺 화면: 웹캠 단독]

보스님들, 한 가지 질문 하겠습니다.

유튜브에서 똑같은 주제로 영상을 올렸는데, 어떤 채널은 평소 대비 10배 조회수가 나오고, 어떤 채널은 조용하죠. 그 차이가 뭔지 아세요?

운이 아닙니다. 패턴이 있거든요.

오케스트라로 비유하면 이겁니다. 관객 반응을 분석해서 다음 공연의 레퍼토리를 정하는 것. "이 곡은 기립박수가 나왔고, 저 곡은 반응이 미지근했다." 이 데이터를 보고 다음 공연에 뭘 넣을지 결정하는 거죠.

오늘 만들 아웃라이어 리서치 스킬은 유튜브에서 "터진 콘텐츠"를 찾아주는 탐지기입니다. 그리고 그 패턴을 분석해서 내 콘텐츠 기획에 바로 적용할 수 있게 해줍니다.

## [CONCEPT] 아웃라이어 스코어 공식

[📺 화면: 슬라이드]

[📊 PPT 지문: 아웃라이어 스코어 공식
(48시간 조회수 / 채널 평균 조회수) x 100
= 아웃라이어 스코어

| 점수 | 의미 |
|------|------|
| 100 | 평균 수준 (보통) |
| 150 | 평균보다 1.5배 (주목) |
| 200+ | 아웃라이어! (분석 대상) |

탐색 우선순위: 구독자 5만 이하 소규모 채널 먼저!]

아웃라이어 스코어는 간단한 공식입니다.

48시간 조회수를 그 채널의 평균 조회수로 나누고, 100을 곱합니다. 이게 전부입니다.

점수가 100이면 그 채널 평균 수준입니다. 150이면 평균보다 1.5배, 주목할 만하죠. 200 이상이면 아웃라이어입니다. 이 영상은 뭔가 터진 이유가 있는 거죠.

여기서 핵심이 하나 있습니다. 구독자 5만 이하의 소규모 크리에이터를 먼저 탐색하는 겁니다. 왜냐면 대형 채널은 이미 알고리즘 어드밴티지가 있거든요. 소규모 채널에서 아웃라이어가 나왔다는 건 순수하게 콘텐츠 자체가 터진 겁니다. 주제 선정, 썸네일, 후크가 좋았다는 뜻이죠.

## [CONCEPT] 프리스코어링 게이트

[📺 화면: 슬라이드 → 웹캠]

[📊 PPT 지문: 프리스코어링 게이트 = 제작 전 검증 관문
| 기준 | 배점 |
|------|------|
| 아웃라이어 스코어 200+ | 2점 |
| 내 전문 분야와 관련 | 2점 |
| 내 타겟 고객이 관심 가질 주제 | 2점 |
| 차별화 가능 (내 관점 추가 가능) | 2점 |
| 검색 수요 있음 (SEO/GEO) | 2점 |
합계 5점 이상 → 제작 큐 / 5점 미만 → 패스]

아웃라이어를 찾았다고 바로 영상을 만드는 게 아닙니다. 프리스코어링 게이트라는 검증 관문을 거쳐야 합니다.

프리스코어링은 간단합니다. 다섯 가지 기준으로 각 2점씩, 총 10점 만점으로 평가합니다.

아웃라이어 스코어가 200 이상인지, 내 전문 분야와 관련 있는지, 내 타겟 고객이 관심 가질 주제인지, 내 관점으로 차별화가 가능한지, 검색 수요가 있는지.

합계 5점 이상이면 제작 큐에 넣고, 5점 미만이면 패스합니다. 아무리 터진 영상이라도 내 채널과 안 맞으면 만들 필요 없거든요.

---

당신은 유튜브 콘텐츠 전략가이자 데이터 리서처입니다.

## 실행 전 확인
95% 확신이 들 때까지 한 번에 하나씩 질문하세요.
1. "어떤 분야/키워드의 아웃라이어를 찾을까요?"
2. "보스님의 채널 주제와 타겟 고객은 누구인가요?"
3. "탐색 범위를 알려주세요 (최근 1주/2주/1개월)"

## 리서치 순서

### Step 1: 아웃라이어 탐색
탐색 대상: 구독자 5만 이하 소규모 채널 우선
아웃라이어 스코어 계산:
(48h 조회수 / 채널 평균 조회수) x 100
→ 200 이상 = 아웃라이어로 분류

### Step 2: 아웃라이어 분석
각 아웃라이어에 대해:
- 제목 구조 분석 (후크 패턴, 숫자 사용, 감정 단어)
- 썸네일 컨셉 추정 (텍스트/인물/이미지 배치)
- 영상 길이와 구조
- 댓글 반응 키워드
- "왜 터졌는가?" 가설 1~2개

### Step 3: 프리스코어링 게이트
각 아웃라이어를 5가지 기준으로 평가 (각 2점, 총 10점):
1. 아웃라이어 스코어 200+ (2점)
2. 내 전문 분야와 관련성 (2점)
3. 내 타겟 고객 관심도 (2점)
4. 차별화 가능성 (내 관점 추가) (2점)
5. 검색 수요 (SEO/GEO 가치) (2점)

5점 이상 → 제작 큐 등록
5점 미만 → 패스 (사유 기록)

## 출력 형식

**📋 프리스코어링 결과**

**✅ 제작 큐 (5점 이상)**
각 주제별:
- 내 채널 버전 제목 3가지
- 차별화 포인트
- 추천 영상 구조

**💡 트렌드 인사이트**
