# Part 19: 리모션 영상 자동화 – 데이터만 넣으면 영상이 나온다

## 메타 정보
- **예상 길이**: 10분 (약 2,500~3,000자)
- **화면 구성**: 웹캠 → PPT(리모션 개념) → 안티그래비티(프로젝트 세팅) → 브라우저(프리뷰) → 렌더링 결과물 재생 → 웹캠
- **핵심 비유**: 악보(JSON)만 주면 오케스트라(리모션)가 알아서 연주해서 녹음(영상)까지 해주는 것
- **필요 도구**: Node.js (이미 설치됨), Remotion

---

## [HOOK] 오프닝 후크

[📺 화면: 웹캠 단독]

보스님들, 쇼츠까지 자동으로 만들었죠? 근데 쇼츠는 기존 영상에서 하이라이트를 '잘라내는' 거였습니다.

이번에는 완전히 새로운 영상을 '만드는' 겁니다.

데이터만 넣으면 브랜드 전용 인포그래픽 영상이 자동으로 렌더링됩니다. 코드 한 줄 안 치고요.

비유로 설명하면, 악보(JSON 데이터)만 주면 오케스트라(리모션)가 알아서 연주해서 녹음(영상 파일)까지 해주는 겁니다. 작곡가가 악보만 쓰면 나머지는 알아서 되는 거죠.

"JSON이 뭔가요?"

쉽게 말하면 데이터를 정리해놓은 양식입니다. 이름, 전화번호, 주소를 칸에 맞춰 적는 서류 양식처럼, 제목, 내용, 색상 같은 정보를 칸에 맞춰 적어놓은 겁니다. 어렵지 않습니다.

리모션(Remotion)이 뭐냐고요? 코드로 영상을 만드는 도구입니다. 근데 보스님들이 코드를 짤 필요는 없거든요. Claude Code가 다 해줍니다. 보스님들은 데이터만 준비하면 됩니다.

---

## [CONCEPT] 리모션 영상 자동화 워크플로우

[📺 화면: 슬라이드]
[📊 PPT 지문: 리모션 영상 자동화 워크플로우 — 1. JSON 데이터 준비 (제목, 통계, 핵심 메시지) → 2. 리모션 템플릿이 데이터를 영상으로 변환 → 3. 자동 렌더링 (MP4 출력) → 4. 브랜드 전용 인포그래픽 영상 완성]

전체 흐름은 이렇습니다.

먼저 JSON 데이터를 준비합니다. "이번 달 매출이 30% 올랐다", "고객 만족도 4.8점", "핵심 메시지: AI로 시간을 절약하세요" 이런 정보를 정리하는 거죠.

그다음 리모션 템플릿이 이 데이터를 받아서 영상으로 변환합니다. 차트가 올라가는 애니메이션, 숫자가 카운트업 되는 효과, 텍스트가 등장하는 트랜지션 이런 게 전부 자동으로 들어갑니다.

마지막으로 렌더링. 버튼 하나 누르면 MP4 영상 파일이 뚝 떨어집니다.

[📊 PPT 지문: 리모션 vs 쇼츠 비교
- 쇼츠: 기존 영상에서 하이라이트 잘라내기 (편집)
- 리모션: 데이터로 새 영상 만들기 (생성)
- 둘 다 URL/데이터만 넣으면 자동화]

쇼츠가 기존 영상을 편집하는 거라면, 리모션은 새 영상을 생성하는 겁니다. 둘 다 자동화돼 있다는 건 같지만, 쓰임새가 다르죠.

## [DEMO] 리모션 프로젝트 세팅 – 따라하세요!

[📺 화면: 안티그래비티 터미널]

리모션 프로젝트를 세팅하겠습니다.

Claude Code 채팅창에 이렇게 입력하세요:

"리모션(Remotion)으로 인포그래픽 영상을 자동 생성하는 프로젝트를 세팅해줘. JSON 데이터를 입력받아서 브랜드 색상과 로고가 들어간 통계 영상을 만들 수 있게 해줘."

[📺 화면: Claude Code가 리모션 프로젝트를 자동 세팅하는 과정]

Claude Code가 리모션 프로젝트를 만들고 있습니다. 필요한 패키지도 자동으로 설치하고 있거든요.

[📺 화면: 생성된 폴더 구조 확인]

폴더 구조를 보면 이렇습니다:

```
skills/remotion-video/
├── src/
│   ├── Root.tsx           ← 메인 구성 파일
│   ├── Composition.tsx    ← 영상 레이아웃
│   └── components/        ← 차트, 텍스트, 카운터 등
├── data/
│   └── sample-data.json   ← 데이터 템플릿
├── remotion.config.ts     ← 설정 파일
└── package.json
```

보스님들이 직접 건드릴 파일은 `data/sample-data.json` 하나뿐입니다. 나머지는 전부 Claude Code가 만들어줬거든요.

## [DEMO] JSON 데이터 준비 – 악보 작성

[📺 화면: 안티그래비티 — data/sample-data.json 파일 열기]

JSON 데이터를 보겠습니다. 마우스를 `data/sample-data.json` 파일로 가져가세요. 더블클릭해서 열어줍니다.

```json
{
  "title": "이번 달 마케팅 성과",
  "brand": {
    "name": "파자마보스",
    "primaryColor": "#FF6B35",
    "logo": "./assets/logo.png"
  },
  "stats": [
    { "label": "콘텐츠 생산량", "value": "320%", "direction": "up" },
    { "label": "제작 시간", "value": "85%", "direction": "down" },
    { "label": "소셜 도달률", "value": "2.4배", "direction": "up" }
  ],
  "message": "AI 마케팅 팀이 만든 성과입니다",
  "cta": "지금 시작하세요"
}
```

보이시죠? 제목, 브랜드 색상, 통계 숫자, 핵심 메시지, CTA 이렇게 칸에 맞춰 적으면 됩니다. 이게 악보입니다. 이 악보를 리모션 오케스트라에게 주면 알아서 연주하고 녹음까지 해주는 거죠.

이 데이터를 보스님 브랜드에 맞게 수정하시면 됩니다. 숫자만 바꾸면 되니까 어렵지 않죠?

## [DEMO] 프리뷰 확인 + 렌더링

[📺 화면: 안티그래비티 터미널]

영상이 어떻게 나올지 미리 보겠습니다. 터미널에 이렇게 입력하세요:

```bash
npx remotion preview
```

[📺 화면: 브라우저에 리모션 프리뷰 화면이 열림]

브라우저에 프리뷰가 열렸습니다. 왼쪽에 타임라인이 보이고, 오른쪽에 영상 미리보기가 보이죠.

재생 버튼을 눌러보세요. 보세요! 제목이 등장하고, 통계 숫자가 카운트업 되면서 올라가고, 차트가 애니메이션으로 그려지고, 마지막에 CTA 문구가 나옵니다.

이 모든 게 아까 JSON에 적은 데이터에서 자동으로 생성된 겁니다. 코드 한 줄 안 쳤습니다.

마음에 들면 렌더링하겠습니다. 터미널에 이렇게 입력하세요:

```bash
npx remotion render
```

[📺 화면: 렌더링 진행 바 — 100%까지 진행]

렌더링 중입니다... 30%... 60%... 100%!

[📺 화면: 출력된 MP4 영상 파일 재생]

MP4 파일이 생성됐습니다. 재생해볼게요.

브랜드 색상이 들어가 있고, 통계 숫자가 애니메이션으로 표시되고, 전문가가 만든 것 같은 인포그래픽 영상이 완성됐습니다.

이걸 인스타 릴스로 올리거나, 유튜브 쇼츠로 올리거나, 발표 자료에 삽입하거나 자유롭게 활용하시면 됩니다.

## [CONCEPT] 무인 렌더링 – 데이터만 바꾸면 영상이 나온다

[📺 화면: 웹캠 복귀]

여기서 진짜 핵심을 말씀드리겠습니다.

이 리모션 템플릿은 한 번만 세팅하면 됩니다. 그 다음부터는 JSON 데이터만 바꾸면 새 영상이 나오거든요.

매월 성과 보고서를 영상으로 만들고 싶으시면? JSON에서 숫자만 수정하고 렌더링하면 끝입니다. 새 제품을 출시했을 때? JSON에서 제품명과 특장점만 바꾸면 됩니다.

이게 무인 렌더링입니다. 사람이 디자인 프로그램을 열 필요 없이, 데이터만 업데이트하면 영상이 자동으로 만들어지는 시스템이죠.

## [RECAP]

[📊 PPT 지문: 리모션 자동화 핵심 정리 — 1. JSON(악보) → 리모션(오케스트라) → MP4(녹음 파일) 2. 코드 작성 불필요, 데이터만 준비하면 됨 3. 템플릿 1회 세팅 → JSON 데이터만 바꾸면 무한 영상 생성]

핵심 정리하겠습니다.

첫 번째, JSON 악보만 주면 리모션 오케스트라가 알아서 연주해서 MP4 녹음 파일까지 만들어줍니다.
두 번째, 코드를 직접 작성할 필요 없습니다. Claude Code가 템플릿을 만들어주고, 보스님들은 데이터만 준비하면 됩니다.
세 번째, 템플릿은 한 번만 세팅하면 됩니다. 그 다음부터는 JSON 데이터만 바꾸면 새 영상이 무한으로 생성됩니다.
