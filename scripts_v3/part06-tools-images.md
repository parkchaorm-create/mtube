- **핵심 비유**: 이미지/PDF = 악보에 첨부하는 참고 자료, 툴 = 악기의 특수 주법(피치카토, 뮤트 등)

## [HOOK]

[📺 화면: 웹캠 단독]

보스님들, 여기서 많이 놀라시는 분들이 있거든요.

"Claude Code에 이미지랑 PDF도 넣을 수 있다고요?"

네, 됩니다.

경쟁사 웹사이트 스크린샷 주면서 "이거 분석해서 내 사이트 개선점 찾아줘" 할 수 있고요. 제품 카탈로그 PDF 주면서 "이거 읽고 인스타 콘텐츠 10개 만들어줘" 할 수 있습니다.

오케스트라로 치면 악보에 참고 이미지나 녹음 파일을 첨부하는 것과 같습니다. "이 느낌으로 연주해줘"라고 예시를 주는 거죠.

이게 가능한 이유가 Claude Code의 "툴" 기능 때문인데요, 툴은 악기의 특수 주법이라고 생각하시면 됩니다. 바이올린으로도 피치카토라고 해서 줄을 뜯는 주법, 아르코라고 해서 활로 켜는 주법, 여러 가지 기법을 쓸 수 있잖아요? 그것과 같습니다.

## [DEMO] 이미지·PDF 넣는 방법

[📺 화면: 터미널 — Claude Code 실행 중]

이미지 넣는 방법 세 가지입니다. 따라해보세요.

첫 번째, 드래그앤드롭입니다. 가장 쉬운 방법이죠. 터미널이나 안티그래비티 채팅창에 이미지 파일을 그냥 끌어다 놓으면 됩니다.

[📺 화면: 이미지 파일을 마우스로 드래그하여 채팅창에 드롭하는 장면]

이미지가 첨부되면 질문을 입력합니다.

```
"이 경쟁사 사이트 스크린샷 분석해서 내가 개선해야 할 점 3가지 알려줘"
```

두 번째, @ 참조입니다. 이미지 파일이 프로젝트 폴더 안에 있다면 골뱅이로 부르면 됩니다.

```
"@competitor-screenshot.png 이 이미지 분석해줘"
```

세 번째, URL입니다. 인터넷에 있는 이미지를 바로 넣을 수 있습니다.

PDF도 같은 방법으로 넣습니다. 실제로 해볼게요.

```
"@product-catalog.pdf
이 카탈로그의 주요 제품 5개를 뽑아서,
각 제품별로 인스타그램 캡션 150자 이내 + 해시태그 5개를 만들어줘."
```

[📺 화면: Claude Code가 PDF를 분석하고 결과를 출력하는 장면]

PDF 내용을 읽고 각 제품별로 콘텐츠를 만들어줍니다. 수동으로 하면 몇 시간이지만 몇 분이면 끝납니다.

## [CONCEPT] 툴(Tool) = 악기의 특수 주법

[📺 화면: 슬라이드]

[📊 PPT 지문: 툴 = 악기의 특수 주법
| 주법(툴) | 하는 일 | 마케팅 활용 |
|--------|---------|---------|
| Read | 파일 읽기 | 블로그 글 읽기 |
| Write | 파일 만들기 | 콘텐츠 파일 생성 |
| Edit | 파일 수정 | 기존 콘텐츠 일부 수정 |
| Bash | 터미널 명령 | 패키지 설치, 스크립트 실행 |
| WebFetch | 웹페이지 읽기 | 경쟁사 사이트 분석 |
| Glob | 파일 검색 | 특정 파일 찾기 |
| Grep | 텍스트 검색 | 특정 단어가 있는 파일 찾기 |]

Claude Code는 단순히 텍스트를 읽고 쓰는 것 외에 실제로 행동할 수 있거든요. 이 행동 하나하나를 "툴"이라고 합니다.

이 주법들은 우리가 외울 필요 없습니다. Claude Code가 필요할 때 알아서 적절한 주법을 선택하거든요. 우리는 "무슨 곡 연주해줘"라고만 하면 됩니다.

다만 한 가지만 기억하세요. Claude Code가 "이 주법 써도 될까요?" 하고 허가를 묻는 경우가 있습니다.

```
Claude wants to run: npm install
[Allow] [Allow All] [Deny]
```

[📊 PPT 지문: Allow = 이번 한 번만 OK / Allow All = 이 세션에서 계속 OK / Deny = 거절]

처음에는 하나씩 Allow 하면서 어떤 주법을 쓰는지 구경하시고요. 익숙해지면 Allow All 해도 됩니다. 단, 파일 삭제하는 명령은 무조건 직접 확인하세요.

---

## [RECAP]

이미지·PDF + 툴 핵심 정리 세 줄입니다.

첫 번째, 이미지와 PDF는 드래그앤드롭이 가장 쉽습니다. 프로젝트 안 파일은 @파일명으로 부르면 됩니다. 두 번째, 툴은 악기의 특수 주법입니다. Claude Code가 알아서 적절한 주법을 선택하니까 외울 필요 없습니다. 세 번째, 툴 허가 요청은 Allow와 Deny로 관리합니다. 익숙해지면 Allow All 써도 됩니다.
