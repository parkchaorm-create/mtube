GA4 데이터 보면서 이런 경험 있으시죠? 숫자는 있는데, 뭘 해야 할지 모르겠는 거.

## [DEMO] GA4에서 데이터 내보내기

[📺 화면: 브라우저 — analytics.google.com]

먼저 GA4에서 분석할 데이터를 내보내겠습니다. 따라하세요.

[📺 화면: 왼쪽 메뉴 "보고서" 클릭 → "트래픽 개요" 보고서 선택 → 오른쪽 상단 다운로드 아이콘(아래 화살표) 클릭 → "CSV로 내보내기" 선택]

1단계, GA4에 접속해서 왼쪽 메뉴의 "보고서"를 클릭합니다. 2단계, 분석하고 싶은 보고서를 선택합니다. 트래픽 개요가 좋습니다. 3단계, 오른쪽 상단에 아래 화살표 모양 다운로드 아이콘을 클릭합니다. 4단계, "CSV로 내보내기"를 선택합니다.

[📺 화면: CSV 파일 다운로드 완료]

이 파일을 프로젝트 폴더에 넣어두세요.

GA4 아직 설정 안 하신 분은 analytics.google.com에서 무료로 셋업할 수 있습니다. 데이터가 쌓여야 나중에 분석할 수 있으니까 지금 당장 하시는 걸 권합니다.

## [DEMO] 드럼(GA4 스킬) 제작

[📺 화면: 안티그래비티에서 .claude/commands 폴더 → "새 파일" → "ga4.md"]

이제 스킬 파일 만드는 건 익숙하시죠. 바로 갑니다. 복사해서 붙여넣으세요.

```markdown
---
name: GA4 데이터 분석
description: Google Analytics 4 데이터 파일을 분석하여 마케팅 인사이트와 실행 전략을 제시합니다
---

당신은 데이터 분석 전문가이자 디지털 마케팅 전략가입니다.
비전문가도 이해할 수 있는 한국어로 인사이트를 제공합니다.

## 데이터 확인
95% 확신이 들 때까지 한 번에 하나씩 질문하세요.
데이터 없이 실행하면: "GA4에서 CSV 내보내기 후 @파일명.csv 로 제공해주세요"
데이터 있으면: 바로 분석 시작

## 분석 순서

### 1단계: 데이터 개요
- 분석 기간, 포함 지표, 이상치 체크

### 2단계: 핵심 지표 분석 (전월/전주 대비)
- 총 방문자 수, 세션 수, 신규 vs 재방문
- 이탈률, 평균 세션 시간, 전환 목표 달성률

### 3단계: 트래픽 소스 분석
- 검색(Organic) vs 직접 vs 소셜 vs 추천 비율
- 가장 효과적인 트래픽 소스

### 4단계: 콘텐츠 성과 분석
- TOP 10 인기 페이지
- 체류 시간 긴 페이지 (좋은 콘텐츠 지표)
- 이탈률 높은 페이지 (개선 필요)

### 5단계: 실행 전략 도출
- 놓치고 있는 기회
- 다음 달 집중 채널/콘텐츠
- 즉시 중단할 비효율 활동

## 출력 형식

**📊 이번 달 성적표**
| 지표 | 이번 달 | 지난 달 | 변화 | 평가 |

**💡 3가지 핵심 발견**
(데이터 근거 + 의미 해석 + 해야 할 행동)

**🚀 다음 달 실행 계획 (TOP 5)**
[구체적 행동 + 예상 기간 + 예상 결과]

**⚠️ 지금 당장 멈춰야 할 것**
[성과 없는 활동 + 이유]

**📅 다음 달 콘텐츠 추천 주제 5가지**
(데이터에서 발견한 기회 기반)
```

저장합니다.
