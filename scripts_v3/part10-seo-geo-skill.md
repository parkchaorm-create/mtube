# Part 10: SEO + GEO 감사 – 내 사이트 건강검진 트럼펫

## [HOOK]

[📺 화면: 웹캠 단독]

보스님들, SEO 어렵죠? 저도 그랬습니다.

키워드 분석, 메타 태그, 백링크, Core Web Vitals... 용어만 들어도 머리가 아프거든요.

근데 알고 보면 SEO는 간단합니다. 내 콘텐츠가 검색될 때 사람들 눈에 잘 보이게 만드는 거거든요.

오케스트라로 치면 트럼펫입니다. 소리가 멀리까지 퍼지잖아요. 아무리 좋은 연주를 해도 관객 귀에 안 들리면 소용없습니다. SEO는 우리 콘텐츠 소리를 구글 검색 첫 페이지까지 퍼뜨려주는 악기입니다.

그런데 요즘은 SEO만으로는 부족합니다. GEO라는 게 있거든요. Generative Engine Optimization, 쉽게 말하면 ChatGPT나 Perplexity 같은 AI한테 내 콘텐츠가 인용되게 만드는 겁니다. AI한테 질문했을 때 "파자마보스 블로그에 따르면..."이라고 인용하면? 무료로 노출되는 거죠.

오늘 만들 스킬에는 SEO와 GEO가 둘 다 들어 있습니다.

## [CONCEPT] SEO + GEO 감사 = 사이트 건강검진

[📺 화면: 슬라이드]

[📊 PPT 지문: SEO + GEO 감사 = 내 사이트 건강검진
| 검진 영역 | 점검 항목 | 쉬운 비유 |
|----------|---------|---------|
| 기술 SEO | 메타 제목/설명 | 이름표가 달려있나? |
| 기술 SEO | H1 태그 | 간판이 보이나? |
| 기술 SEO | 이미지 alt 텍스트 | 사진에 설명이 붙어있나? |
| 콘텐츠 SEO | 키워드 밀도 | 핵심 단어가 자연스럽게 들어있나? |
| AI SEO (GEO) | 정의 문장 | "X는 Y입니다" 형태가 있나? |
| AI SEO (GEO) | Q&A 형식 | AI가 인용하기 좋은 구조인가? |]

기본 검진 항목부터 보겠습니다. 기술적 SEO는 메타 제목이 있는지, 쉽게 말하면 이름표가 달려있는지 확인하는 겁니다. H1 태그가 제대로 쓰이고 있는지는 간판이 보이는지 확인하는 거고요.

콘텐츠 SEO는 키워드가 자연스럽게 포함됐는지, 콘텐츠 길이가 충분한지 체크합니다.

그리고 GEO 검진입니다. 이게 최신 트렌드인데요. AI가 내 콘텐츠를 인용하려면 특정 구조가 필요합니다. "X는 Y입니다" 같은 명확한 정의 문장, Q&A 형식, 리스트나 단계별 번호. 이런 게 있어야 AI가 "아, 이건 인용할 만한 콘텐츠다"라고 판단하거든요.

## [DEMO] 트럼펫(SEO+GEO 스킬) 제작

```markdown

name: SEO + GEO 감사
description: 웹사이트 URL을 입력하면 SEO·GEO 문제점을 찾아 우선순위별 개선 방법을 제시합니다
---

당신은 SEO·GEO 전문가이자 디지털 마케팅 전략가입니다.
제공된 웹사이트를 철저히 분석하여 구체적인 개선 액션을 제시합니다.

## 입력 정보 확인
95% 확신이 들 때까지 한 번에 하나씩 질문하세요.
1. "감사할 웹사이트 URL을 알려주세요"
2. "타겟 키워드 3~5개를 알려주세요"
3. "경쟁사 URL이 있다면 알려주세요 (선택)"

## SEO·GEO 감사 실행 순서

### Step 1: 기술적 SEO 점검
URL 제공받으면 WebFetch로 페이지 읽기.
- [ ] 메타 제목: 50~60자, 키워드 포함
- [ ] 메타 설명: 150~160자
- [ ] H1 태그: 딱 하나, 키워드 포함
- [ ] H2~H3: 논리적 구조
- [ ] 이미지 alt 텍스트
- [ ] 내부 링크 연결
- [ ] URL 구조: 짧고 키워드 포함

### Step 2: 콘텐츠 SEO 점검
- [ ] 콘텐츠 길이: 최소 1,000자
- [ ] 키워드 밀도: 2~3%
- [ ] 첫 단락에 키워드 포함
- [ ] LSI 관련 키워드 자연스럽게 포함

### Step 3: AI SEO (GEO) 점검
- [ ] 명확한 정의 문장 ("X는 Y입니다" 형태)
- [ ] Q&A 형식 섹션
- [ ] 리스트 또는 단계별 번호
- [ ] 전문가 경험담 또는 데이터 인용
- [ ] 구조화된 데이터(Schema) 권장사항

### Step 4: 경쟁사 비교 (선택)
경쟁사 URL 제공 시:
- 키워드 커버리지 비교
- 콘텐츠 깊이 비교
- 내가 놓치고 있는 주제
```

## [DEMO] 건강검진 실행

[📺 화면: 터미널 — Claude Code]

내 사이트 건강검진을 바로 해보겠습니다.

```

```

사이트가 없으시면 블로그 URL을 넣어도 됩니다.

[📺 화면: Claude Code가 WebFetch로 사이트를 스캔하고 결과를 출력하는 장면]

결과가 나왔습니다. 빨간색, 노란색, 초록색으로 우선순위가 구분됩니다.

"메타 설명이 없는 페이지 8개", "GEO 정의 문장 부족"...

이걸 수동으로 찾으려면 몇 시간이지만 지금 몇 분이면 나옵니다.

빨간색부터 고치면 됩니다. 메타 설명이 누락됐다면 이렇게 바로 요청하세요.

```
"메타 설명이 없는 페이지들에 대해 최적화된 메타 설명을 각 150~160자로 만들어줘.
타겟 키워드 포함하고, 클릭하고 싶게."
```

SEO 대행사에 월 수백만 원 내는 일을 이 악기 하나가 해줍니다.
