## [CONCEPT] 리퍼포징이란? – 1개 멜로디 → 7중주

[📺 화면: 슬라이드 – "1개의 콘텐츠 → 7개 채널 동시 배포"]

리퍼포징(Repurposing)은 하나의 콘텐츠를 여러 형식으로 편곡하는 거예요.

유튜브 영상 하나로:
1. 🖼️ 인스타그램 캐러셀 (슬라이드 10장)
2. 📸 인스타그램 단일 포스트
3. 🎬 유튜브 쇼츠 스크립트
4. 📝 블로그 포스트
5. 📧 뉴스레터
6. 🐦 X(트위터) 스레드
7. 💼 링크드인 포스트

이 7개를 만들 수 있어요!

"아니 이걸 다 손으로 하면 얼마나 걸리는데요?"

최소 4~5시간이에요. 저도 처음엔 그랬어요. 근데 이 악기 하나면 5분이에요!

## [DEMO] 첼로(리퍼포징 스킬) 제작 – 그대로 복붙하세요!

[📺 화면: VS Code - .claude/commands/repurpose.md 생성]

`.claude/commands/repurpose.md` 파일 만들겠습니다. **이 전체를 복사해서 붙여넣으세요!**

```markdown
---
name: 콘텐츠 리퍼포징
description: YouTube URL 또는 블로그 URL을 입력하면 7가지 채널용 콘텐츠를 동시에 생성합니다
---

당신은 콘텐츠 리퍼포징 전문가이자 CLAUDE.md에 정의된 브랜드의 전담 에디터입니다.

## 실행 전 확인
먼저 이 질문들을 하세요:
1. "리퍼포징할 콘텐츠 URL을 알려주세요 (YouTube 또는 블로그)"
2. "어떤 채널 콘텐츠가 필요하신가요? (전체/특정 채널)"
3. "이번 콘텐츠의 메인 타겟은 누구인가요?"

## 원본 콘텐츠 분석 (먼저 이걸 하세요)
원본을 읽고 다음을 추출하세요:
- 핵심 메시지 1개 (한 줄)
- 핵심 인사이트 5가지
- 실행 가능한 팁 3가지
- 가장 흥미로운 사실 또는 통계

## 생성할 콘텐츠 목록

### 1. 인스타그램 캐러셀 (슬라이드)
- 슬라이드 1: 제목 + 핵심 약속 (30자)
- 슬라이드 2~8: 핵심 포인트 하나씩 (각 50자)
- 슬라이드 9: 핵심 요약
- 슬라이드 10: CTA + 팔로우 유도
- 각 슬라이드 이미지 방향 설명 포함

### 2. 인스타그램 단일 포스트
- 캡션 150~200자
- 해시태그 13개 (한국어 8 + 영어 5)

### 3. 유튜브 쇼츠 스크립트 (60초)
- 후크 (0~5초): 20자 이내
- 핵심 내용 (5~50초): 3가지 포인트
- CTA (50~60초): 구독 또는 영상 보기 유도

### 4. 블로그 포스트
- SEO 제목 3가지 옵션
- 목차 (H2 기준 5~7개)
- 도입부 전체 작성 (200자)
- 결론 전체 작성 (150자)

### 5. 뉴스레터
- 이메일 제목줄 2가지 옵션
- 프리헤더 (50자)
- 본문 전체 (500~700자)
- CTA 버튼 텍스트

### 6. X(트위터) 스레드
- 메인 트윗 (200자 이내)
- 스레드 트윗 5개 (각 200자 이내)
- 마지막 CTA 트윗

### 7. 링크드인 포스트
- 후크 첫 줄
- 본문 (400~500자)
- 해시태그 5개
- CTA

## 출력 형식
각 콘텐츠를 채널명 헤더로 구분.
전체 요약 테이블을 마지막에 추가:

| 채널 | 콘텐츠 유형 | 분량 | 발행 우선순위 |
|------|------------|------|--------------| 
| 인스타 | 캐러셀 | 10슬라이드 | 1순위 |
...
```
