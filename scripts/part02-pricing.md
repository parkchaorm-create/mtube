- **핵심 비유**: 요금제 = 오케스트라 대관료, 토큰 = 택시 미터기

보스님들, 여기서 솔직하게 얘기할게요.

Claude Code 시작하려면 돈이 필요합니다.

얼마냐고요? 한 달에 약 3만 원 정도입니다.

"에이, 그냥 무료로 쓸 수 없나요?"라고 생각하실 수 있는데요. 여기서 딱 한 가지만 말씀드릴게요.

이 3만 원이 오케스트라 공연장 대관료거든요. 공연장 없이는 오케스트라가 연주할 무대가 없잖아요? 3만 원으로 무대를 빌리면, 그 위에서 AI 악단이 매주 수십 시간어치 일을 대신해줍니다.

실제로 저는 이 구독 시작하고 나서 직원 한 명을 안 써도 되게 됐거든요. 그 차이가 어마어마합니다.

자, 요금제 정확히 어떻게 생겼는지 같이 보겠습니다.

## [DEMO] 요금제 비교 분석

[📺 화면: 브라우저 → claude.ai/pricing 접속]

지금 브라우저를 열어볼게요. 크롬이든 엣지든 상관없습니다. 주소창에 "claude.ai/pricing"을 입력하고 엔터를 눌러주세요.

[📊 PPT 지문: 요금제 비교 화면 캡처 — Free / Pro / Max 세 컬럼]

화면에 세 가지 플랜이 보이실 겁니다. 왼쪽부터 Free, Pro, Max 순서로 나열돼 있습니다.

첫 번째, 왼쪽에 있는 Free 플랜입니다. 무료라서 매력적으로 보이죠? 근데 한 가지 치명적인 단점이 있거든요. Claude Code를 사용할 수가 없습니다. 그냥 일반 채팅만 되는 겁니다. 오케스트라로 치면 공연장 견학만 할 수 있고, 무대에 올라갈 수는 없는 거죠.

두 번째, 가운데에 있는 Pro 플랜. 월 20달러, 한화로 약 2만 8천원에서 3만원 사이입니다. 이게 바로 오늘 보스님들이 시작하실 플랜이거든요. Claude Code 전체 기능을 쓸 수 있고, 하루에 상당한 양을 작업할 수 있습니다.

세 번째, 오른쪽에 있는 Max 플랜. 월 25달러부터 100달러까지 여러 단계가 있습니다. 매일 몇 시간씩 Claude Code를 돌리는 헤비유저나, 여러 명이 팀으로 쓸 때 선택하는 거죠. 지금 당장은 필요 없습니다.

[📊 PPT 지문: 요금제 비교표 — Free(견학) / Pro($20, 소극장 대관, 왕초보 추천) / Max($100~, 대극장 대관, 헤비유저)]

결론만 드리면, Pro 플랜으로 시작하세요. 왕초보 단계에서는 Pro로 충분합니다.

## [CONCEPT] 토큰이란? – 택시 미터기로 이해하기

[📺 화면: 웹캠 복귀]

[📊 PPT 지문: 토큰 = 택시 미터기 비유 그래픽 — 짧은 대화 = 단거리 요금, 긴 대화 = 장거리 요금]

"그럼 Pro 쓰다가 사용량 다 쓰면 어떻게 돼요?"

좋은 질문이죠. 여기서 '토큰'이라는 개념이 등장합니다.

토큰이 뭐냐면요, 택시 미터기거든요. 택시 타면 이동 거리에 따라 미터기가 올라가잖아요? AI도 똑같습니다. 제가 말을 많이 하고, AI가 답을 많이 할수록 미터기, 즉 토큰이 올라갑니다.

Pro 플랜은 하루에 탈 수 있는 택시 거리가 정해져 있거든요. 한도를 초과하면 그날은 속도가 느려지거나 일시 중지됩니다.

그래서 처음에는 작업을 몰아서 하지 말고, 하루 1~2시간씩 꾸준히 쓰는 게 좋습니다.

제 경험상 Pro 플랜으로 왕초보 분들은 충분히 커버됩니다. 매일 악기 하나 연습하고, 콘텐츠 1~2개 만드는 정도면 Pro로 충분하거든요.

## [CONCEPT] Claude Code vs API 직접 사용 차이

[📊 PPT 지문: Claude Code(Pro 구독) vs API 직접 사용 비교 — 초보자: Pro 구독 추천 / 개발자: API도 가능]

"API로 쓰면 더 저렴하지 않나요?" 이런 질문 꼭 나오더라고요.

API가 뭐냐면요, 쉽게 말해서 음식점 주문 전화번호입니다. 배달 앱에서 버튼 누르면 음식 주문이 들어가잖아요? 그게 API 호출이죠. AI한테 직접 전화 걸어서 주문하는 방식인 거죠.

API는 사용량 기반으로 과금하는 방식이거든요. 많이 쓰면 많이 내고, 적게 쓰면 적게 내죠.

근데 왕초보한테 API 직접 사용은 비추입니다. 왜냐면 설정이 복잡하거든요. API 키 발급받고, 연결하고, 비용이 얼마나 나올지 예측도 어렵고, 실수로 비용 폭탄 맞는 경우도 있습니다.

Pro 구독으로 Claude Code 쓰는 게 훨씬 편하고 안전합니다. 월 정액제 택시라고 생각하시면 됩니다. 나중에 쓰다 보면 API도 쓸 일이 생기는데, 그건 그때 배우면 되고요. 오늘은 Pro 구독으로 시작하겠습니다.

## [DEMO] 구독 시작하는 법

[📺 화면: 브라우저 → claude.ai/pricing]

자, 지금 바로 구독 시작하는 방법 보여드릴게요. 화면 같이 보시죠.

[📺 화면: 브라우저에서 claude.ai/pricing 열려 있는 상태]

지금 claude.ai/pricing 페이지에 계신 거 맞죠? 가운데 Pro 플랜 칸 아래를 보시면 파란색 버튼이 하나 있어요. "Get started with Pro"라고 써있는 버튼입니다. 그걸 클릭해주세요.

[📺 화면: 로그인 또는 회원가입 화면]

클릭하면 로그인 화면이 나와요. 계정이 없으신 분은 "Sign up"을 눌러서 가입하시면 되고, 이미 계정 있으신 분은 이메일 입력하고 로그인하시면 됩니다. 구글 계정으로 로그인도 되니까 편하신 방법으로 하세요.

[📺 화면: 결제 정보 입력 화면]

로그인하면 결제 정보 입력하는 화면이 나와요. 신용카드 번호, 유효기간, CVC 번호를 입력하시면 됩니다. 한국 카드 전부 됩니다. 저도 국내 카드 쓰고 있어요.

카드 정보 입력하고 아래에 있는 "Subscribe" 버튼 클릭하면 끝입니다.

[📺 화면: 구독 완료 확인 화면 또는 claude.ai 메인 화면]

이제 오케스트라 공연장 대관이 완료됐습니다! Pro 배지가 붙은 거 보이시죠? 이제 Claude Code를 쓸 수 있는 자격이 생긴 겁니다.

## [CONCEPT] 절약 꿀팁

[📺 화면: 웹캠 단독]

[📊 PPT 지문: 토큰 절약 꿀팁 3가지 — /clear 리셋 / 작업별 세션 분리 / Haiku 모델 활용]

택시 미터기, 즉 토큰 절약 꿀팁을 하나만 미리 알려드릴게요. 이건 뒤에 더 자세히 다루지만, 미리 알고 계시면 좋거든요.

Claude Code는 매 대화마다 이전 내용을 기억하는 데 토큰을 씁니다. 대화가 길어질수록 택시비가 계속 올라가거든요.

해결법 세 가지만 기억하세요. 첫 번째는 `/clear` 명령어로 대화를 주기적으로 초기화하는 겁니다. 미터기를 리셋하는 거라고 생각하시면 되고요. 두 번째는 작업별로 새 세션을 시작하는 거죠. 새 택시 타는 겁니다. 세 번째는 가벼운 작업은 Haiku 모델을 쓰는 건데요, 경차 택시라서 기본료가 싸거든요.

이 얘기는 뒤에서 자세히 다루니까, 일단 이 정도만 알고 계세요.
