## [CONCEPT] 토큰이란? – 택시 미터기로 이해하기

[📺 화면: 웹캠 복귀]

"그럼 Pro 쓰다가 사용량 다 쓰면 어떻게 돼요?"

좋은 질문이에요. 여기서 '토큰'이라는 개념이 등장합니다.

"토큰? 그게 뭔데요?"

쉽게 말하면 택시 미터기예요. 택시 타면 이동 거리에 따라 미터기가 올라가잖아요? AI도 마찬가지예요. 여러분이 말을 많이 하고, AI가 답을 많이 할수록 미터기(토큰)가 올라갑니다.

Pro 플랜은 하루에 탈 수 있는 택시 거리가 정해져 있어요. 한도를 초과하면 그날은 속도가 느려지거나 일시 중지됩니다.

그래서 처음에는 작업을 몰아서 하지 말고, 하루 1~2시간씩 꾸준히 쓰는 게 좋아요.

제 경험상 Pro 플랜으로 왕초보분들은 충분히 커버 됩니다. 매일 악기 하나 연습하고, 콘텐츠 1~2개 만드는 정도면 Pro로 충분해요.

## [CONCEPT] Claude Code vs API 직접 사용 차이

"API로 쓰면 더 저렴하지 않나요?"

네, 이런 질문 꼭 나오더라고요.

API가 뭐냐고요? 쉽게 말하면 음식점 '주문 전화번호'예요. 배달 앱에서 버튼 누르면 음식 주문이 들어가잖아요? 그게 API 호출이에요. AI한테 직접 전화 걸어서 주문하는 방식인 거죠.

API는 사용량 기반으로 과금하는 방식이에요. 많이 쓰면 많이 내고, 적게 쓰면 적게 내죠.

왕초보에게 API 직접 사용은 비추입니다. 왜냐면:
- 설정이 복잡해요 (전화번호 발급받고 연결하는 과정이 까다로움)
- 비용 예측이 어려워요 (택시 미터기가 얼마나 올라갈지 모르니까)
- 실수로 비용 폭탄 맞을 수 있어요

Pro 구독으로 Claude Code 쓰는 게 훨씬 편하고 안전합니다. 월 정액제 택시라고 생각하시면 돼요.

나중에 쓰다 보면 API도 쓸 일이 생기는데, 그건 그때 배우면 됩니다. 오늘은 Pro 구독으로 시작해요.

## [DEMO] 구독 시작하는 법

[📺 화면: 브라우저 → claude.ai 로그인 → 업그레이드]

지금 claude.ai에서 Pro로 업그레이드하는 방법 보여드릴게요.

1단계, claude.ai 접속 후 로그인.

2단계, 왼쪽 하단 "Upgrade to Pro" 클릭.

3단계, 결제 정보 입력. 신용카드나 페이팔 됩니다.

4단계, 완료! 이제 오케스트라 공연장 대관 끝났어요.

참고로, 한국 카드로 결제 잘 됩니다. 저도 국내 카드 쓰고 있어요.

## [CONCEPT] 절약 꿀팁

[📺 화면: 웹캠 단독]

택시 미터기(토큰) 절약 꿀팁 하나만 드릴게요. 이건 뒤에 더 자세히 다루지만, 미리 알고 계시면 좋아요.

Claude Code는 매 대화마다 이전 내용을 기억하는 데 토큰을 씁니다. 대화가 길어지면 택시비가 계속 올라가요.

해결법:
- `/clear` 명령어로 대화를 주기적으로 초기화 (미터기 리셋!)
- 작업별로 새 세션 시작 (새 택시 타기)
- 가벼운 작업은 Haiku 모델 사용 (경차 택시 – 기본료가 싸요)

이 얘기는 1시간 30분에서 자세히 다루니까, 일단은 이 정도만 알고 계세요.
