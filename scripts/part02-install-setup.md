## [CONCEPT] 설치할 세 가지 장비

[📊 PPT 지문: 공연장 장비 3종 세트 — 안티그래비티(무대/공연 공간) / Node.js(전기 배선/동력) / Claude Code(음향 장비/핵심)]

우리가 설치할 장비는 딱 세 가지입니다.

첫 번째는 안티그래비티(Antigravity)입니다. 오케스트라가 연주할 무대죠. Claude Code가 가장 편하게 일할 수 있는 전용 작업 도구인데요, 완전 무료거든요.

두 번째는 Node.js입니다. 공연장에 전기를 들어오게 하는 배선 작업이죠. 음향 장비가 작동하려면 전기가 있어야 하잖아요? 한 번 깔아두면 다시는 신경 쓸 일 없습니다.

세 번째가 진짜 핵심이거든요. Claude Code입니다. AI 오케스트라가 실제로 연주하게 해주는 음향 장비죠.

자, 이 순서대로 하나씩 같이 설치해볼게요!

## [DEMO] 장비 1: 안티그래비티 설치 (무대 세팅)

[📺 화면: 브라우저 열기 → 안티그래비티 공식 사이트 접속]

첫 번째 장비, 무대를 세팅하겠습니다.

브라우저를 열어주세요. 엣지나 크롬 아무거나 상관없거든요. 주소창에 안티그래비티 공식 사이트 주소를 입력하고 엔터를 눌러주세요.

[📺 화면: 안티그래비티 공식 홈페이지 — 다운로드 버튼 강조]

페이지 가운데에 다운로드 버튼이 보이실 겁니다.

윈도우 사용자분들은 "Download for Windows"라고 써있는 버튼을 클릭해주세요. 맥 사용자분들은 Mac 버전 다운로드 링크를 클릭해서 받으시면 됩니다.

[📺 화면: 다운로드 진행 → 파일 탐색기에서 설치 파일 확인]

다운로드 완료되면 설치 파일을 더블클릭해서 실행해주세요. 설치 창이 뜨면 "동의합니다" 클릭하고요, 그다음은 계속 "다음"만 눌러주시면 됩니다. 마지막에 "설치" 버튼 클릭하면 끝이거든요.

맥 사용자분들은 .dmg 파일을 열고 안티그래비티 앱을 응용 프로그램 폴더로 끌어다 놓으시면 됩니다.

[📺 화면: 안티그래비티 첫 실행 화면]

설치 완료되면 안티그래비티를 실행해보세요. 이렇게 화면이 나오면 성공입니다. 무대 세팅 완료!

## [DEMO] 장비 2: Node.js 설치 (전기 배선)

[📺 화면: 브라우저 → nodejs.org 접속]

두 번째 장비, 전기 배선을 설치하겠습니다.

브라우저 주소창에 "nodejs.org"를 입력하고 엔터를 눌러주세요.

[📺 화면: Node.js 공식 홈페이지 — LTS 버튼 강조]

페이지 가운데에 두 개의 버튼이 보일 겁니다. 왼쪽에 "LTS"라고 써있는 초록 버튼이 있고요, 오른쪽에 "Current"가 있거든요. 반드시 왼쪽의 LTS 버튼을 클릭해주세요. LTS가 안정적인 버전이거든요. 오른쪽 Current는 건드리지 마세요.

[📺 화면: 다운로드 → 설치 파일 실행]

다운로드된 파일을 더블클릭해서 실행해주세요. 설치 창이 뜨면 "Next" 버튼을 계속 눌러주시면 됩니다. 중간에 체크박스 같은 거 건드리실 필요 없거든요. 그냥 "Next", "Next", "Install" 순서로 클릭하시면 되는 겁니다.

맥 사용자분들은 .pkg 파일 더블클릭하고 "계속"을 누르다 보면 설치 완료됩니다.

[📺 화면: 설치 완료 → 파워셸/터미널 실행]

설치 완료됐으면 제대로 됐는지 확인해볼게요.

윈도우 사용자분들은 키보드에서 윈도우 키를 누르고 "PowerShell"을 검색해서 실행해주세요. 파란 배경에 커맨드 라인이 뜨는 창이거든요. 맥 사용자분들은 Spotlight에서 "터미널"을 검색하시면 됩니다.

[📺 화면: PowerShell/터미널 창]

창이 뜨면 여기에 이렇게 입력하고 엔터를 눌러주세요.

```bash
node --version
```

[📺 화면: v20.xx.x 버전 숫자 출력]

v20이나 v22 같은 버전 숫자가 나오면 전기 배선 성공입니다! 숫자가 뜨지 않고 에러가 나오면요, 창을 완전히 닫고 다시 열어보세요. 설치 후에는 창을 새로 열어야 반영이 되거든요.

## [DEMO] 장비 3: Claude Code 설치 (음향 장비 – 핵심!)

[📺 화면: PowerShell/터미널 창]

자, 이제 진짜 핵심입니다! 음향 장비를 설치할게요.

아까 열었던 창으로 돌아오세요. 여기에 아래 명령어를 그대로 복사해서 붙여넣고 엔터를 눌러주세요. 직접 치려고 하지 마시고요, 꼭 복사 붙여넣기 하세요.

```bash
npm install -g @anthropic-ai/claude-code
```

[📺 화면: 설치 진행 화면 — 텍스트가 쭉 내려가는 모습]

이렇게 글자들이 쭉쭉 내려가면서 설치가 진행되고 있는 겁니다. 30초에서 1분 정도 걸리니까 기다려주세요. 멈춘 것처럼 보여도 괜찮거든요. 그냥 기다리시면 됩니다.

[📺 화면: 설치 완료 — 커맨드 라인으로 돌아온 화면]

설치 완료됐으면 확인해볼게요.

```bash
claude --version
```

[📺 화면: 버전 번호 출력]

버전 번호가 나오면 음향 장비 설치 완료! 보스님들, 여기까지 오셨으면 공연장 세팅의 90%가 끝난 거거든요.

## [TROUBLESHOOT] 자주 나오는 에러 해결법

[📊 PPT 지문: 자주 나오는 에러 3가지 — npm 명령어 없음 / claude 명령어 없음 / permission denied]

설치하다 에러 나는 분들 계실 겁니다. 가장 자주 나오는 에러 세 가지만 알려드릴게요.

첫 번째, "'npm'은(는) 내부 또는 외부 명령이 아닙니다"라고 나오는 경우입니다. Node.js 설치가 제대로 안 됐거나, 창을 관리자 권한으로 열지 않은 경우거든요. PowerShell을 닫고요, 다시 검색해서 아이콘 위에서 마우스 오른쪽 버튼을 클릭하세요. "관리자 권한으로 실행"을 선택하고 다시 명령어를 입력해보시면 됩니다.

두 번째, "claude를 찾을 수 없습니다"라고 나오는 경우입니다. 창을 새로 열지 않아서 그런 거거든요. 완전히 닫고 다시 열어서 입력해보세요.

세 번째, 맥에서 "permission denied" 에러가 나오면요, 명령어 앞에 "sudo"를 붙이고 맥 비밀번호를 입력하시면 됩니다.

그래도 해결이 안 되면 댓글에 에러 메시지 올려주세요. 제가 직접 확인해드리겠습니다.

## [DEMO] 안티그래비티에서 Claude Code 실행 (무대와 음향 연결)

[📺 화면: 안티그래비티 실행 화면]

자, 이제 무대와 음향 장비를 연결하겠습니다.

안티그래비티를 열어주세요. 화면 맨 위 메뉴바에서 "Terminal"을 클릭하면 드롭다운 메뉴가 나오는데요, "New Terminal"을 클릭해주세요.

[📺 화면: 안티그래비티 하단에 터미널 창 열림]

안티그래비티 화면 아래쪽에 검은 창이 하나 열립니다. 이게 안티그래비티 안에 내장된 터미널이거든요. 여기에 claude를 입력하고 엔터를 눌러주세요.

```bash
claude
```

[📺 화면: 브라우저 자동 열림 — 로그인 화면]

브라우저가 자동으로 열리면서 로그인하라는 화면이 뜹니다. 아까 구독한 Pro 계정으로 로그인하시면 되거든요. 구글 계정 연동도 되니까 편하신 방법으로 하세요.

[📺 화면: 로그인 완료 → 터미널로 돌아옴]

로그인하면 브라우저에 "You're now connected"라고 뜨고요, 터미널에 이런 문구가 보이면 성공이죠.

```
✓ Claude Code ready
Type your request below...
```

보스님들, 공연장 세팅이 완성됐습니다!
