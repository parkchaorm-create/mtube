근데 제가 원하던 게 아닌 거예요. 수정하려고 하니까 이미 다 얽혀있고, 처음부터 다시 해야 하는 상황이 됐어요.

## [CONCEPT] Plan Mode vs Edit Mode

[📺 화면: 슬라이드 – "리허설 vs 바로 본공연"]

Claude Code에는 기본적으로 두 가지 모드가 있어요.

**Edit Mode (기본) – 바로 본공연:**
- Claude가 바로 파일을 만들고 수정합니다
- 빠르지만 원하지 않는 결과가 나올 수 있음
- 간단한 수정 작업에 적합

**Plan Mode – 리허설 먼저:**
- Claude가 먼저 "이렇게 연주할게요, 맞죠?" 계획을 보여줍니다
- 악보 확인 후 승인하면 본공연 시작
- 복잡한 작업, 새로운 악기(스킬) 만들 때 필수

쉽게 말하면, Plan Mode는 "리허설 먼저 보여주고, 괜찮으면 본공연 시작하는 모드"예요.

"아니 그러면 매번 Plan Mode 써야 해요?"

좋은 질문! 아니요. 간단한 수정은 바로 본공연(Edit Mode)해도 돼요. 새 악기 만들거나 복잡한 일 할 때만 리허설(Plan Mode) 하면 됩니다.

## [DEMO] Plan Mode 켜는 방법 – 따라해보세요

[📺 화면: 터미널 - Claude Code 실행 중]

Plan Mode 켜는 방법은 세 가지예요. 제일 쉬운 것부터 알려드릴게요.

**방법 1: 그냥 말로 하기 (가장 쉬움, 추천!)**

```
소셜미디어 자동화 스킬을 만들고 싶어.
근데 바로 만들지 말고, 계획을 먼저 보여줘. 내가 OK하면 그때 실행해.
```

이렇게 "계획 먼저 보여줘"라고 말하면 됩니다. 이게 제일 쉬워요!

**방법 2: 슬래시 명령어**

```
/plan
```

이렇게 입력하면 Plan Mode가 켜집니다.

**방법 3: VS Code 하단 버튼**

VS Code에 Claude Code 확장 설치했다면 하단에 "Plan/Edit" 토글 버튼 있어요. 클릭 한 번이면 전환!

## [DEMO] 리허설 있을 때 vs 없을 때 – 직접 비교

[📺 화면: 터미널 - Plan Mode 실습]

직접 비교해볼게요. 따라하시면서 차이를 느껴보세요.

**리허설 없이 바로 본공연 (Edit Mode):**

```
소셜미디어 콘텐츠 자동화 스킬 만들어줘
```

→ Claude가 바로 파일들 만들기 시작. 어디로 가는지 모르겠음. 중간에 방향 바꾸기 어려움.

**리허설 먼저 (Plan Mode) – 이렇게 입력해보세요:**

```
소셜미디어 콘텐츠 자동화 스킬을 만들 거야.
인스타그램과 유튜브 쇼츠 콘텐츠를 생성하는 스킬이야.
계획을 먼저 보여줘. 내가 OK하면 실행해.
```

→ Claude 응답:

```
계획을 먼저 설명드릴게요:

1. .claude/commands/ 폴더 생성
2. social-media.md 파일 생성
   - 인스타그램 캡션 생성 로직
   - 유튜브 쇼츠 스크립트 생성 로직
   - 해시태그 자동 추가
3. CLAUDE.md에 /social-media 스킬 등록

이렇게 진행할까요? 수정할 부분 있으시면 알려주세요.
```

보이시죠? 리허설 악보가 먼저 나옵니다!

여기서 보스님들이 "2번에서 링크드인도 추가해줘"라고 할 수 있어요. 

```
좋아, 근데 링크드인 포스트도 추가해줘.
```

그러면 Claude가 계획 수정 후 다시 보여줍니다. OK하면 그제서야 실행!

→ 처음부터 원하는 결과물이 나옵니다.

🎉 **리허설 한 번으로 삽질 80% 제거!**

## [CONCEPT] 언제 리허설하고, 언제 바로 공연하나?

[📺 화면: 슬라이드]

규칙 딱 하나만 기억하세요.

**리허설(Plan Mode) 필수인 상황:**
✅ 새로운 악기(스킬)를 처음 만들 때
✅ 여러 파일에 영향을 줄 때
✅ 복잡한 오케스트라(에이전트) 설계할 때
✅ "왜 이렇게 됐지?" 싶을 때 (의도한 게 아닌 경우)

**바로 본공연(Edit Mode) OK인 상황:**
❌ 단순 문장 수정할 때
❌ 기존 파일 한두 줄만 바꿀 때
❌ 간단한 질문에 답 받을 때

**한 줄 규칙: "처음 만드는 거면 → 리허설!"**

## [DEMO] 실수 복구 – 되돌리기 기능

[📺 화면: 터미널]

보너스! 리허설 없이 공연했다가 망했을 때 대처법도 있어요.

Claude Code에는 "되돌리기" 기능이 있습니다.

```
/rewind
```

또는 `Esc` 키 두 번 누르기

이렇게 하면 Claude Code가 최근 변경사항을 취소해줍니다.

공연 중에 틀린 음 나왔을 때 "다시!"하고 처음으로 돌아가는 것과 같아요.

겁먹지 마세요. 실수해도 되돌릴 수 있어요.
