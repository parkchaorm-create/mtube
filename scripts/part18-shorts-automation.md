# Part 18: 쇼츠 영상 자동 제작 – 영상 팀까지 완성

## 메타 정보
- **예상 길이**: 10분 (약 2,500~3,000자)
- **화면 구성**: 웹캠 → PPT(쇼츠 제작 워크플로우) → 안티그래비티(스킬 설정) → 쇼츠 결과물 재생 → 웹캠
- **핵심 비유**: 쇼츠 제작 팀 = 오케스트라의 녹음 편집 팀, 본공연에서 하이라이트를 뽑아 앨범으로 만드는 것
- **필요 도구**: yt-dlp, whisper, ffmpeg (자동 설치)

[📺 화면: 웹캠 단독]

보스님들, 텍스트 콘텐츠 됐고, 이미지 콘텐츠도 됐습니다.

이제 마지막 한 가지가 남았죠.
영상!

유튜브 쇼츠, 인스타 릴스, 틱톡.
요즘 숏폼 영상이 마케팅에서 얼마나 중요한지 다들 아시잖습니까.

"근데 영상 편집은 진짜 어렵잖습니까. 프리미어 프로도 못 쓰는데..."

걱정 마세요!
우리 오케스트라에 녹음 편집 팀을 추가하면 됩니다.
기존 유튜브 영상에서 하이라이트를 자동으로 추출하고, 자막까지 입혀서 쇼츠 영상 3개를 뚝딱 만들어주죠.

오케스트라 공연을 녹음해서 베스트 트랙만 뽑아 앨범으로 내는 것과 같은 겁니다.
직접 해보겠습니다!

## [CONCEPT] 쇼츠 제작 워크플로우

[📺 화면: 슬라이드]
[📊 PPT 지문: 쇼츠 자동 제작 워크플로우 — 1. 유튜브 URL 입력 → 2. yt-dlp로 영상 다운로드 → 3. Whisper로 자막 추출 → 4. AI가 하이라이트 구간 선정 → 5. ffmpeg로 쇼츠 편집(세로 9:16, 자막 포함) → 6. 쇼츠 3개 완성]

쇼츠 자동 제작은 이런 흐름으로 진행됩니다.

먼저 유튜브 URL을 넣으면요, yt-dlp라는 도구가 영상을 다운로드합니다.
yt-dlp, 이름이 좀 어렵죠?
유튜브 영상을 컴퓨터에 저장해주는 택배 기사라고 생각하시면 됩니다.

그다음에 Whisper라는 음성 인식 도구가 영상의 대사를 텍스트로 변환해줍니다.
Whisper는 통역사입니다.
사람 말을 글자로 바꿔주는 통역사죠.

그러면 AI가 그 텍스트를 분석해서 가장 재밌거나 핵심적인 구간 3개를 골라냅니다.
ffmpeg라는 영상 편집 도구가 해당 구간을 잘라서 세로 화면에 자막까지 입혀주죠.
9:16 비율입니다.

ffmpeg는 가위와 풀이라고 생각하시면 됩니다.
영상을 자르고 붙이는 도구죠.

이 모든 과정이 하나의 스킬로 자동화돼 있습니다.
보스님들은 URL만 넣으면 됩니다.
이게 전부입니다!

## [DEMO] 쇼츠 제작 스킬 세팅

[📺 화면: 안티그래비티 — 프로젝트 폴더 구조]

쇼츠 제작 스킬을 만들겠습니다.

여기서 한 가지 확인할 게 있습니다.
이 스킬은 Python이 필요하거든요.

안티그래비티 터미널에서 "python --version"이라고 입력해보세요.
마우스를 터미널 영역으로 가져가시고, 클릭한 다음에 타이핑하시면 됩니다.
버전 번호가 나오면 OK.
안 나오면 "python.org"에서 설치하시면 되는데요, 역시 무료입니다.

그걸 확인했으면요, 직접 만들 필요 없습니다.
Claude Code한테 시키면 되죠.

Claude Code 채팅창에 이렇게 입력하세요:

"유튜브 URL을 입력하면 자동으로 쇼츠 영상 3개를 만들어주는 스킬을 만들어줘. yt-dlp로 영상 다운로드하고, whisper로 자막 추출하고, ffmpeg로 하이라이트 구간을 세로 9:16 비율로 잘라서 자막까지 입혀줘. 자막 크기는 16px, 하단 배치로 해줘."

[📺 화면: Claude Code가 스킬 파일과 파이썬 스크립트를 자동 생성하는 과정]

Claude Code가 SKILL.md 파일이랑 파이썬 스크립트를 자동으로 만들어주고 있습니다.

download_video.py는 영상을 다운로드하는 스크립트고요.
transcribe.py는 음성을 텍스트로 변환하는 스크립트입니다.
cut_shorts.py는 하이라이트 구간을 잘라서 쇼츠로 편집하는 스크립트죠.

[📺 화면: 필요한 라이브러리 자동 설치 과정]

필요한 라이브러리도 Claude Code가 알아서 설치해줍니다.
yt-dlp, whisper, ffmpeg 전부요.
보스님들이 직접 설치할 필요 없습니다.
