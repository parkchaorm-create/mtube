## [CONCEPT] 두 가지 경로

[📺 화면: 슬라이드]
[📊 PPT 지문: SNS 배포 두 가지 경로
🟢 쉬운 경로: Blotato/Buffer → 9개 채널 동시 게시 (추천!)
🟡 직접 경로: YouTube API + Meta API → 완전 자동 (개발자 향)]

배포 방법은 두 가지가 있거든요.

**쉬운 경로**가 있고, **직접 경로**가 있습니다.

쉬운 경로는 Blotato나 Buffer 같은 SNS 관리 도구를 쓰는 겁니다. Blotato, 이름이 좀 특이하죠. 감자를 블로그에 올린다는 뜻은 아니고요, SNS 통합 발행 도구입니다. API를 제공하기 때문에 Claude Code에서 직접 호출할 수 있거든요.

직접 경로는 YouTube API, Meta API를 직접 연결하는 건데요. 완전 자동이지만 설정이 좀 복잡하죠.

오늘은 쉬운 경로로 가겠습니다. 90%의 보스님한테는 이 방법이 최적이거든요.

## [DEMO] Blotato 연동 설정

[📺 화면: 안티그래비티 - social-publish.md 파일]

먼저 Blotato 계정을 만들고 API 키를 발급받으세요. 무료 플랜도 있습니다.

API 키를 받았으면 `.env` 파일에 저장하고요.

```
BLOTATO_API_KEY=여기에_API_키_붙여넣기
```

그다음, 발행 스킬 파일을 만들겠습니다. `.claude/commands/social-publish.md` 파일입니다.

```markdown
# SNS 동시 발행 스킬

## 목적
완성된 콘텐츠를 여러 SNS 채널에 동시 발행합니다.

## 발행 채널
Instagram, Facebook, Twitter/X, LinkedIn, Threads, 
YouTube Community, Pinterest, TikTok, Blog(WordPress)

## 발행 옵션 (3가지)
1. **즉시 발행** — 지금 바로 게시
2. **예약 발행** — 특정 날짜/시간 지정
3. **슬롯 발행** — 다음 빈 슬롯에 자동 배치 (추천)

## Blotato API 참조
- 엔드포인트: https://api.blotato.com/v1/posts
- 인증: Bearer token (.env의 BLOTATO_API_KEY)
- 이미지 포함 시: multipart/form-data

## 발행 후 처리
- 발행된 URL을 content/publish-log.md에 기록
- 발행 날짜, 채널, URL 포함

## 주의사항
- 발행 전 반드시 사용자 승인 받기 (초안 보여주고 확인)
- 테스트 시에는 초안(draft) 모드로 먼저 발행
```

[📺 화면: 마우스로 "슬롯 발행" 부분 가리킴]

세 번째 옵션, "슬롯 발행"이 핵심이거든요.

## [CONCEPT] 슬롯 기반 스케줄링

[📺 화면: 슬라이드]
[📊 PPT 지문: 슬롯 스케줄링
인스타그램: 매일 오전 9시 → 슬롯 1개/일
트위터: 하루 3회 → 슬롯 3개/일
블로그: 주 1회 → 슬롯 1개/주
→ 콘텐츠를 만들면 다음 빈 슬롯에 자동 배치!]

슬롯이 뭐냐고요? 미리 정해둔 발행 시간표입니다.

"인스타그램은 매일 오전 9시에 올린다", "트위터는 하루 3번 올린다", "블로그는 일주일에 1번 올린다" 이렇게 시간표를 미리 짜놓는 거죠.

그러면 콘텐츠를 만들 때마다 다음 빈 슬롯에 자동으로 들어가거든요.

배치 작업에 딱 맞는 방식입니다. 월요일에 앉아서 콘텐츠 10개를 한꺼번에 만들면, 그 10개가 각 플랫폼의 빈 슬롯에 자동으로 배치되는 거죠. 이번 주 인스타가 이미 가득 차 있으면요? 다음 주 빈 슬롯에 자동으로 넘어갑니다.

## [DEMO] 발행 테스트

[📺 화면: 터미널에서 Claude Code 실행]

테스트해보겠습니다.

```
/social-publish

주제: AI 마케팅 자동화 팁 3가지
채널: 인스타그램, 트위터
옵션: 초안 모드
```

[📺 화면: Claude Code가 콘텐츠 생성 → 초안 미리보기 → 승인 요청]

Claude Code가 콘텐츠를 만들고, "이렇게 발행할까요?"라고 물어봅니다. 미리보기를 확인하고 "네"라고 하면 Blotato API를 통해 발행되는 거죠.

초안 모드니까 실제로 게시되지는 않고요, Blotato 대시보드에 초안으로 저장됩니다. 확인하고 마음에 들면 거기서 발행 버튼 누르면 되고, Claude Code에서 바로 발행하려면 초안 모드를 빼면 됩니다.

발행 로그도 자동으로 기록되거든요. `content/publish-log.md`에 날짜, 채널, URL이 쭉 남습니다.
