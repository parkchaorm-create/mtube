## [SECTION 1: 오케스트라 튜닝] 자주 터지는 에러 TOP 5

[📺 화면: 슬라이드 – "오케스트라 튜닝 – 삐끗할 때 대처법"]

공연 전 튜닝 시간이에요! 오케스트라도 공연 전에 반드시 악기를 조율하잖아요?

### 에러 1: "컨텍스트가 너무 길어졌어요" (AI가 아까 말한 거 까먹음)

지휘자 책상에 악보가 너무 쌓인 거예요. 해결법은 간단!

```
지금까지 작업 내용 요약해줘. 파일명이랑 핵심 결정사항 포함해서.
```

요약 복사하고 나서:

```
/clear
```

새 책상에서 다시 시작!

---

### 에러 2: "스킬 명령어가 안 먹혀요"

이거 10명 중 8명이 겪어요. 원인은 두 가지.

첫 번째, 파일 위치 문제. `.claude/commands/` 안에 있어야 해요.

```bash
ls .claude/commands/
```

여기에 스킬 파일이 보이면 정상!

두 번째, 프로젝트 폴더 밖에서 Claude Code를 실행한 경우.

```bash
pwd
# /Users/내이름/Documents/my-ai-marketing 이어야 함!
```

---

### 에러 3: "AI가 내 브랜드 톤이랑 안 맞아요"

악기 조율이 안 된 거예요. CLAUDE.md가 빈약하거나 브랜드 보이스 가이드가 연동 안 된 경우.

```
내 CLAUDE.md 읽어보고, 브랜드 보이스가 제대로 반영돼 있는지 확인해줘.
부족한 부분 있으면 추가해줘.
```

AI한테 자기 점검 시키면 됩니다!

---

### 에러 4: "토큰 한도 초과"

공연 시간을 너무 길게 잡은 거예요. 꿀팁 세 가지:

첫 번째, 오전엔 콘텐츠 스킬, 오후엔 SEO 스킬. 나눠서!
두 번째, 간단한 수정은 `/model` → Haiku(경차 택시)로!
세 번째, `/clear` 자주. 짧은 공연 여러 번이 효율적!

---

### 에러 5: "Allow 버튼이 계속 떠요"

툴 허가 요청이에요. 처음엔 하나씩 Allow하다가 익숙해지면:

Allow All 체크하면 됩니다.

단, 파일 삭제 명령은 꼭 직접 확인하세요!

## [SECTION 2: 레퍼토리 확장] 새 악기 추가하기

[📺 화면: 슬라이드 – "오케스트라에 새 악기 추가하기"]

보스님들, 지금까지 만든 악기가 5개예요. 근데 오케스트라는 악기를 계속 추가할 수 있잖아요?

"어떤 악기를 더 넣을 수 있는데요?"

세 가지만 더 소개해드릴게요!

**새 악기 1: 이메일 시퀀스 스킬 🎸**

고객 유입 후 자동 이메일 3통: 환영 → 꿀팁 공유 → 상품 소개. `.claude/commands/email-sequence.md` 파일 하나면 끝!

**새 악기 2: 경쟁사 모니터링 스킬 🎷**

매주 경쟁사 URL 3개 넣으면 "이번 주 경쟁사는 이런 주제를 다뤘고, 우리가 놓치고 있는 건 이거예요"라고 알려줘요.

**새 악기 3: 랜딩페이지 카피 최적화 스킬 🎺**

내 랜딩페이지 URL 넣으면 CTA 버튼 문구 개선, 전환율 높이는 제안을 해줘요.

만드는 방법? Part 9에서 했던 것과 똑같아요! `.claude/commands/`에 마크다운 파일 만들면 끝. 보스님들은 이미 이 방법을 완벽하게 아시잖아요!

## [SECTION 3: 공연 수익] 이 오케스트라로 어떻게 돈을 버나?

[📺 화면: 웹캠 → 슬라이드]

자, 가장 중요한 질문.

"이거 다 좋은데... 결국 돈은 어떻게 벌어요?"

이 질문 안 하시면 제가 먼저 했을 거예요!

[📺 화면: 슬라이드 – "수익화 3단계 로드맵"]

**1단계: 시간 = 돈 (월 0~100만 원 절감)**

매주 4~5시간 콘텐츠 작업 → 30분으로 단축. 한 달 16~20시간 절약. 이 시간에 고객 미팅 한 번 더 하면 이미 본전!

**2단계: 콘텐츠 폭발 → 트래픽 → 수익 (월 100~500만 원)**

혼자서 매주 인스타 3개, 블로그 1개, 뉴스레터 1개, 쇼츠 2개. 3개월이면 트래픽이 눈에 보이게! 트래픽 → 광고 수익, 제휴 마케팅, 상품 판매!

**3단계: 시스템 자체를 파세요 (월 500만 원 이상)**

"시스템을 판다고요?"

네! 보스님들이 만든 이 오케스트라 시스템이 상품이에요.

- 같은 업종 사장님께 "AI 마케팅 시스템 세팅해드릴게요" → 컨설팅
- 스킬 파일 템플릿 묶음 → 디지털 상품
- 매달 관리 서비스 → 월 구독

오늘 만든 게 단순히 편리한 도구가 아니에요. **비즈니스 자산**이에요!
