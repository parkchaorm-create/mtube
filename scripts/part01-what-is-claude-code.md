## [HOOK] 오프닝 후크

보스님들, 여기서 딱 한 가지 질문 드릴게요.

"클로드 코드가 뭔지는 아는데, ChatGPT랑 뭐가 다른 건지 모르겠어요."

이렇게 생각하시는 분 많으시죠?

저도 처음에 딱 이랬어요. 그냥 AI 챗봇 하나 더 생긴 거 아닌가? 싶었거든요.

근데 써보고 나서 한 마디로 정리됐습니다.

ChatGPT는 음악 이론만 가르쳐주는 선생님이고, Claude Code는 직접 악기를 들고 연주까지 해주는 악단이에요.

이게 전부예요. 이걸 이해하시면 왜 우리 오케스트라의 핵심 엔진이 Claude Code여야 하는지 바로 납득이 되실 거예요.

---

## [CONCEPT] 개념 설명: Claude Code vs 일반 AI

[📺 화면: 슬라이드 – "음악 이론 선생님 vs 실전 악단"]

자, 비교표 보면서 설명드릴게요.

**ChatGPT, Claude.ai 같은 일반 AI 챗봇** – 음악 이론 선생님:
- 여러분이 질문하면 → 텍스트로 답변 줌
- "곡 써줘" 하면 → 악보를 보여줌 (근데 연주는 안 해줌)
- "이 코드 실행해줘" 하면 → 못 해요, "이렇게 하시면 돼요"라고 설명만 해줌

**Claude Code** – 실전 악단:
- 여러분이 시키면 → 직접 파일을 만들어요
- "SEO 감사해줘" 하면 → 실제로 웹사이트 스캔하고 보고서 생성
- "소셜미디어 포스트 발행 준비해줘" 하면 → 실제로 파일 만들어서 저장

[📺 화면: 슬라이드 – "설명만 vs 직접 행동" 다이어그램]

쉽게 말하면 이렇습니다.

ChatGPT한테 "내 블로그 SEO 분석해줘"라고 하면요,
→ "SEO를 분석하려면 이런 걸 확인하셔야 해요"라고 방법을 알려줘요. 음악 선생님이 "이 곡은 이렇게 연주하면 좋아요"라고 설명하는 것과 같죠.

Claude Code한테 "내 블로그 SEO 분석해줘"라고 하면요,
→ 실제로 여러분 블로그를 스캔하고, 문제점 리스트를 만들고, 수정 방법까지 파일로 저장해줍니다. 악단이 직접 연주해서 녹음까지 해주는 거예요.

"아니 그러면 Claude Code는 내 컴퓨터에서 돌아가는 건가요?"

아! 좋은 질문이에요. 네, 맞습니다. Claude Code는 여러분 컴퓨터 안에서 실행돼요. 그래서 여러분 파일에 직접 접근해서 수정할 수 있거든요.

이게 왜 중요하냐면요? 오케스트라로 치면, 악단이 우리 집 거실에 상주하는 거예요. 필요할 때마다 내 악보(파일)를 가져다가 직접 연주(실행)해주니까 진짜 자동화가 되는 거예요.

---

## [CONCEPT] 마케터가 Claude Code로 할 수 있는 것 10가지

[📺 화면: 슬라이드 – "마케터를 위한 Claude Code 활용법 10가지"]

자, 그럼 실제로 마케터 입장에서 뭘 할 수 있는지 딱 10가지만 정리해드릴게요.

이 10가지를 나중에 우리 오케스트라의 악기로 하나씩 만들어볼 거예요.

**콘텐츠 생성 악기들 🎵**

첫 번째, 소셜미디어 콘텐츠 자동화. URL 하나 넣으면 인스타, 유튜브, 블로그 콘텐츠가 동시에 나옵니다.

두 번째, 뉴스레터 자동 작성. 주제 알려주면 브랜드 보이스에 맞게 뚝딱 써줘요.

세 번째, 영상 스크립트 작성. 오늘 제가 쓰는 이 스크립트도 Claude Code 도움을 받아 작성했습니다.

**분석 악기들 📊**

네 번째, SEO 감사 자동화. 내 사이트 문제점을 스캔해서 우선순위별로 정리해줘요.

다섯 번째, GA4 데이터 분석. 구글 애널리틱스 파일 넣으면 인사이트를 한국어로 설명해줍니다.

여섯 번째, 경쟁사 분석. 경쟁사 사이트를 분석해서 내가 뭘 해야 하는지 알려줘요.

**오케스트라 편성 악기들 🎻**

일곱 번째, 멀티 에이전트 팀 구성. 리서처 AI, 카피라이터 AI, SEO AI를 팀으로 만들 수 있어요. 오케스트라 단원을 배치하는 거죠.

여덟 번째, 자동화 트리거 설정. 매주 월요일 오전에 알아서 콘텐츠 계획 만드는 것도 가능합니다. 자동 공연 시스템이에요.

**실용 도구들 🔧**

아홉 번째, 이메일 자동화. 시퀀스 이메일 초안부터 A/B 테스트 문구까지 만들어줘요.

열 번째, 랜딩페이지 최적화. 전환율 높이는 카피 수정 제안부터 직접 코드 수정까지 가능합니다.

[📺 화면: 웹캠 복귀]

이 10가지 중에 지금 당장 필요한 것 몇 개 있으시죠?

오늘 영상에서 핵심적인 것들은 다 직접 해볼 거예요. 악기를 하나씩 만들고, 마지막에 합주하는 겁니다!

## [CONCEPT] 왜 지금 Claude Code인가?

[📺 화면: 슬라이드 – "Claude Code 2026 현재 상황"]

마지막으로 한 가지만 더 말씀드릴게요.

"Claude Code가 좋다는 건 알겠는데, 지금 시작해도 늦지 않은 건가요?"

아, 이건 정말 좋은 질문이에요.

데이터 하나 보여드릴게요. 최근 4개월 동안 한국의 주간 활성 Claude Code 사용자가 6배 성장했습니다. 전 세계에서 한국 사용자 성장률이 가장 높아요.

그 말은 뭐냐면, 지금 한국 시장에서 Claude Code로 마케팅 오케스트라를 돌리는 사람은 아직 극소수라는 거예요. 지금이 딱 진입 타이밍이에요.

"아니 지금 시작하면 남들보다 늦은 거 아니에요?"

천만에요. 오케스트라 비유로 말하면, 지금은 아직 관객석이 다 안 찬 상태예요. 지금 무대에 올라가면 첫 번째 줄에서 공연하시는 거예요. 2년 후에는 뒷줄에서 서서 볼 수도 있어요.

자, 충분히 이해됐죠? 이제 요금제 얘기로 넘어가겠습니다.
