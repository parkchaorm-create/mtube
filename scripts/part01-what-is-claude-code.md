이걸 이해하시면 왜 우리 오케스트라의 핵심 엔진이 Claude Code여야 하는지 바로 납득이 되실 겁니다. 그리고요, 요금제까지 한 번에 정리해드리겠습니다.

## [CONCEPT] 개념 설명: Claude Code vs 일반 AI

[📊 PPT 지문: ChatGPT vs Claude Code 비교 — 왼쪽: 음악 이론 선생님 (질문 → 텍스트 답변, 설명만) / 오른쪽: 실전 악단 (시키면 → 직접 파일 생성, 실행까지)]

자, 비교해서 설명드릴게요.

ChatGPT, Claude.ai 같은 일반 AI 챗봇은요, 음악 이론 선생님입니다. 여러분이 질문하면 텍스트로 답변을 주죠. "곡 써줘"하면 악보를 보여주긴 하는데요, 직접 연주는 안 합니다. "이 코드 실행해줘"하면 못 하거든요. "이렇게 하시면 됩니다"라고 설명만 해주는 겁니다.

근데 Claude Code는요? 실전 악단입니다. 여러분이 시키면 직접 파일을 만들거든요. SEO라는 게 있는데요, 쉽게 말하면 내 콘텐츠가 구글 검색에서 잘 보이게 만드는 겁니다. "SEO 감사해줘"하면 실제로 웹사이트 스캔하고 보고서를 뚝딱 생성합니다. "소셜미디어 포스트 발행 준비해줘"하면 실제로 파일 만들어서 저장해주거든요.

[📊 PPT 지문: 설명만 vs 직접 행동 비교 다이어그램 — ChatGPT: "이렇게 하세요" (말풍선) / Claude Code: 파일 생성 → 스캔 → 보고서 저장 (실행 화살표)]

쉽게 말하면 이렇습니다.

ChatGPT한테 "내 블로그 SEO 분석해줘"라고 하면요? "SEO를 분석하려면 이런 걸 확인하셔야 합니다"라고 방법을 알려줍니다. 음악 선생님이 "이 곡은 이렇게 연주하면 좋습니다"라고 설명하는 것과 같죠.

근데 Claude Code한테 똑같이 "내 블로그 SEO 분석해줘"라고 하면요? 실제로 여러분 블로그를 스캔하고요. 문제점 리스트를 만들고요. 수정 방법까지 파일로 저장합니다. 악단이 직접 연주해서 녹음까지 해주는 거거든요.

"아니 그러면 Claude Code는 내 컴퓨터에서 돌아가는 건가요?"

네, 맞습니다. Claude Code는 여러분 컴퓨터 안에서 실행되거든요. 그래서 여러분 파일에 직접 접근해서 수정할 수 있는 겁니다.

이게 왜 중요하냐고요? 오케스트라로 치면요, 악단이 우리 집 거실에 상주하는 겁니다. 필요할 때마다 내 악보, 즉 파일을 가져다가 직접 연주해주니까 진짜 자동화가 되는 거죠.

## [CONCEPT] 사업하시는 분이 Claude Code로 할 수 있는 것 10가지

[📊 PPT 지문: Claude Code 활용법 10가지 — 3열 그리드: 콘텐츠(소셜미디어·뉴스레터·스크립트) / 분석(SEO·GEO·GA4·경쟁사) / 자동화(에이전트 팀·트리거·이메일·랜딩페이지)]

자, 그럼 실제로 사업하시는 분 입장에서 뭘 할 수 있는지 딱 10가지만 정리해드릴게요.

먼저 콘텐츠 생성 악기들입니다.

첫 번째, 소셜미디어 콘텐츠 자동화. URL 하나 넣으면 인스타, 유튜브, 블로그 콘텐츠가 동시에 나오거든요.

두 번째, 뉴스레터 자동 작성. 주제 알려주면 브랜드 보이스에 맞게 뚝딱 써줍니다.

세 번째, 영상 스크립트 작성. 오늘 제가 쓰는 이 스크립트도 Claude Code 도움을 받아 작성했거든요.

다음은 분석 악기들입니다.

네 번째, SEO 감사 자동화. 내 사이트 문제점을 스캔해서 우선순위별로 정리해주거든요. 그리고요, 요즘은 SEO만으로는 부족합니다. 구글뿐 아니라 ChatGPT, Perplexity 같은 AI 검색에서도 내 콘텐츠가 노출되어야 하거든요. 이걸 GEO라고 하는데요, Generative Engine Optimization의 약자입니다. 쉽게 말하면 AI한테도 잘 보이게 만드는 거죠.

다섯 번째, GA4 데이터 분석. GA4가 뭐냐면요, 구글 애널리틱스라고 하는 건데요, 내 웹사이트 방문자 데이터를 분석해주는 도구입니다. 파일 넣으면 인사이트를 한국어로 설명해주거든요.

여섯 번째, 경쟁사 분석. 경쟁사 사이트를 분석해서 내가 뭘 해야 하는지 딱 알려줍니다.

그다음은 오케스트라 편성 악기들이죠.

일곱 번째, 멀티 에이전트 팀 구성. 리서처 AI, 카피라이터 AI, SEO AI를 팀으로 만들 수 있거든요.

여덟 번째, 자동화 트리거 설정. 매주 월요일 오전에 알아서 콘텐츠 계획 만드는 것도 가능합니다.

마지막으로 실용 도구들입니다.

아홉 번째, 이메일 자동화. 시퀀스 이메일 초안부터 A/B 테스트 문구까지 만들어주거든요.

열 번째, 랜딩페이지 최적화. 전환율 높이는 카피 수정 제안부터 직접 코드 수정까지 가능합니다.

[📺 화면: 웹캠 복귀]

이 10가지 중에 지금 당장 필요한 것 몇 개 있으시죠?

오늘 영상 시리즈에서 핵심적인 것들은 다 직접 해볼 겁니다. 악기를 하나씩 만들고요, 마지막에 합주하는 거죠!

## [CONCEPT] 왜 지금 Claude Code인가?

[📊 PPT 지문: Claude Code 2026 한국 시장 현황 — 4개월간 주간 활성 사용자 6배 성장, 전 세계 한국 성장률 1위 그래프]

마지막으로 한 가지만 더 말씀드릴게요.

"Claude Code가 좋다는 건 알겠는데, 지금 시작해도 늦지 않은 건가요?"

데이터 하나 보여드리겠습니다. 최근 4개월 동안 한국의 주간 활성 Claude Code 사용자가 6배 성장했거든요. 전 세계에서 한국 사용자 성장률이 가장 높습니다.

그 말은 뭐냐면요, 지금 한국 시장에서 Claude Code로 마케팅 오케스트라를 돌리는 사람은 아직 극소수라는 겁니다. 지금이 딱 진입 타이밍이거든요.

오케스트라 비유로 말하면요, 지금은 아직 관객석이 다 안 찬 상태입니다. 지금 무대에 올라가면 첫 번째 줄에서 공연하시는 거죠. 나중에는요? 뒷줄에서 서서 볼 수도 있거든요.

## [CONCEPT] 요금제 정리 – 오케스트라 대관료

[📺 화면: 웹캠 단독]

자, 이제 돈 얘기 해봅시다. 이 오케스트라 운영비가 얼마냐고요?

솔직하게 말씀드리겠습니다. Claude Code 시작하려면 돈이 필요합니다. 한 달에 약 3만 원 정도거든요.

"에이, 그냥 무료로 쓸 수 없나요?"라고 생각하실 수 있는데요. 이 3만 원이 오케스트라 공연장 대관료입니다. 공연장 없이는 오케스트라가 연주할 무대가 없잖아요? 3만 원으로 무대를 빌리면요, 그 위에서 AI 악단이 매주 수십 시간어치 일을 대신해줍니다.

[📺 화면: 브라우저 → claude.ai/pricing 접속]

[📊 PPT 지문: 요금제 비교 — Free(견학만) / Pro($20, 소극장 대관, 왕초보 추천) / Max($100~, 대극장 대관, 헤비유저)]

화면에 세 가지 플랜이 보이실 겁니다.

첫 번째, Free 플랜입니다. 무료라서 매력적으로 보이죠? 근데 치명적인 단점이 있거든요. Claude Code를 사용할 수가 없습니다. 공연장 견학만 할 수 있고요, 무대에 올라갈 수는 없는 거죠.

두 번째, Pro 플랜. 월 20달러, 한화로 약 3만 원입니다. 이게 바로 오늘 보스님들이 시작하실 플랜이거든요. Claude Code 전체 기능을 쓸 수 있고요, 하루에 상당한 양을 작업할 수 있습니다.

세 번째, Max 플랜. 월 25달러부터 100달러까지 여러 단계가 있거든요. 매일 몇 시간씩 돌리는 헤비유저용이죠. 지금 당장은 필요 없습니다.

결론만 드리면요, Pro 플랜으로 시작하세요. 왕초보 단계에서는 Pro로 충분하거든요.

## [CONCEPT] 토큰이란? – 택시 미터기로 이해하기

[📊 PPT 지문: 토큰 = 택시 미터기 비유 그래픽 — 짧은 대화 = 단거리 요금, 긴 대화 = 장거리 요금]

"그럼 Pro 쓰다가 사용량 다 쓰면 어떻게 돼요?"

좋은 질문이죠. 여기서 '토큰'이라는 개념이 등장하거든요.

토큰이 뭐냐면요, 택시 미터기입니다. 택시 타면 이동 거리에 따라 미터기가 올라가잖아요? AI도 똑같거든요. 제가 말을 많이 하고, AI가 답을 많이 할수록 미터기, 즉 토큰이 올라갑니다.

Pro 플랜은 하루에 탈 수 있는 택시 거리가 정해져 있거든요. 한도를 초과하면 그날은 속도가 느려지거나 일시 중지됩니다. 처음에는요, 작업을 몰아서 하지 말고 하루 1~2시간씩 꾸준히 쓰는 게 좋습니다.

## [CONCEPT] Claude Code vs API 직접 사용 차이

[📊 PPT 지문: Claude Code(Pro 구독) vs API 직접 사용 비교 — 초보자: Pro 구독 추천 / 개발자: API도 가능]

"API로 쓰면 더 저렴하지 않나요?" 이런 질문 꼭 나오더라고요.

API가 뭐냐면요, 쉽게 말해서 음식점 주문 전화번호입니다. 배달 앱에서 버튼 누르면 음식 주문이 들어가잖아요? 그게 API 호출이죠. AI한테 직접 전화 걸어서 주문하는 방식인 거거든요.

근데요, 왕초보한테 API 직접 사용은 비추입니다. 설정이 복잡하고요. 비용이 얼마나 나올지 예측도 어렵고요. 실수로 비용 폭탄 맞는 경우도 있거든요.

그러니까요, Pro 구독으로 Claude Code 쓰는 게 훨씬 편하고 안전합니다. 월 정액제 택시라고 생각하시면 되거든요.

## [CONCEPT] 토큰 절약 꿀팁

[📊 PPT 지문: 토큰 절약 꿀팁 3가지 — /clear 리셋 / 작업별 세션 분리 / Haiku 모델 활용]

택시 미터기 절약 꿀팁을 미리 알려드릴게요.

첫 번째는 `/clear` 명령어로 대화를 주기적으로 초기화하는 겁니다. 미터기를 리셋하는 거라고 생각하시면 되고요. 두 번째는 작업별로 새 세션을 시작하는 거죠. 새 택시 타는 겁니다. 세 번째는 가벼운 작업은 Haiku 모델을 쓰는 건데요, 경차 택시라서 기본료가 싸거든요.

이 얘기는 뒤에서 더 자세히 다루니까요, 일단 이 정도만 알고 계세요.
