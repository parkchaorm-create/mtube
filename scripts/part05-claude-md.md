저도 처음에 CLAUDE.md 없이 시작했다가 얼마나 수정했는지 몰라요. 오늘 이 실수는 여러분이 안 하시게 해드리겠습니다.

## [CONCEPT] CLAUDE.md란?

[📺 화면: 슬라이드 → 웹캠으로 설명]

[📊 PPT 지문: CLAUDE.md = 오케스트라 총보 / 총보 없음 = 매번 "무슨 곡이요?" 질문 / 총보 있음 = 알아서 딱 맞는 연주]

CLAUDE.md는 마크다운 형식의 텍스트 파일이에요. 마크다운이 뭔지 몰라도 상관없어요. 그냥 메모장에 글 쓰는 거랑 비슷해요.

Claude Code는 프로젝트 시작할 때 이 파일을 자동으로 읽습니다. 그래서 매번 "나는 마케터고, 이런 브랜드야, 이런 톤으로 써줘"라고 설명 안 해도 돼요.

비유하자면요. 총보 없이 연주하면 연주자들이 매번 "무슨 곡이요? 무슨 키요? 몇 박자예요?"라고 물어봐야 해요. 총보가 있으면 연주자들이 알아서 딱 맞는 곡을 연주합니다.

그럼 이 파일을 어디에 넣어야 하냐고요?

[📊 PPT 지문: 저장 위치 — 글로벌(~/.claude/CLAUDE.md, 모든 프로젝트 적용) / 프로젝트(내프로젝트폴더/CLAUDE.md, 이 프로젝트에만 적용)]

두 곳에 넣을 수 있어요. 하나는 글로벌 설정이에요. 홈 폴더 안에 .claude 폴더에 넣으면 모든 프로젝트에 적용돼요. 다른 하나는 프로젝트 설정이에요. 지금 작업하는 폴더 안에 넣으면 이 프로젝트에만 적용됩니다.

오늘은 프로젝트 폴더에 만들 거예요.

## [DEMO] CLAUDE.md 라이브 작성 (총보 직접 만들기)

[📺 화면: VS Code — 왼쪽 탐색기 패널]

자, 이제 우리 오케스트라의 총보를 직접 써볼게요.

아까 Part 03에서 VS Code를 설치했죠? VS Code를 실행해주세요. 그런데 먼저 우리 프로젝트 폴더를 VS Code에서 열어야 해요.

VS Code 화면 맨 위 메뉴바에서 "File"을 클릭하세요. 드롭다운 메뉴가 나오면 "Open Folder..."를 클릭해주세요.

[📺 화면: 파일 탐색기 — Documents\my-ai-marketing 폴더 선택]

파일 탐색기 창이 뜨면, 아까 만든 Documents 안의 my-ai-marketing 폴더를 찾아서 선택하고, 오른쪽 아래 "폴더 선택" 버튼을 클릭해주세요.

[📺 화면: VS Code — 왼쪽 탐색기에 my-ai-marketing 폴더 열림]

이제 VS Code 왼쪽 탐색기 패널에 my-ai-marketing 폴더가 보일 거예요. 폴더 이름 위에 마우스를 올리면 아이콘 4개가 나타나는데, 그 중 왼쪽에서 첫 번째 아이콘인 "새 파일" 아이콘을 클릭해주세요. 종이에 + 표시가 있는 아이콘이에요.

[📺 화면: 파일 이름 입력창]

파일 이름 입력창이 뜨면 "CLAUDE.md"라고 입력하고 엔터를 눌러주세요. 대소문자 중요해요! 전부 대문자로 "CLAUDE.md"입니다.

[📺 화면: VS Code 에디터 — CLAUDE.md 빈 파일]

자, 빈 파일이 열렸습니다. 이제 총보를 채워볼게요. 아래 내용을 그대로 복사해서 붙여넣으세요. 저장할 때 Ctrl+V로 붙여넣으시면 됩니다.

```markdown
# 파자마보스 AI 마케팅 오케스트라 총보

## 브랜드 소개 (우리 오케스트라는 누구인가)
- **브랜드명**: 파자마보스
- **업종**: AI 마케팅 교육 & 1인 기업 컨설팅
- **타겟 고객**: 40대 비전공자, 소상공인, 1인 기업가, AI 초보자
- **핵심 가치**: 극강의 실용성, 공감과 진정성, 즉시 실행 가능한 해결책

## 브랜드 보이스 (우리 오케스트라의 연주 스타일)
- **톤**: 친근하고 에너지 넘치는, 확신에 찬
- **호칭**: 독자를 "보스님들"이라고 부름
- **시점**: 화자는 "저는", "제가"를 사용하는 1인칭
- **시그니처 표현**: "복붙하세요", "삽질", "꿀팁", "딱 N단계만", "이게 전부예요"
- **금지 표현**: 과도한 전문 용어 남발, 수동적 표현, 과장 광고

## 콘텐츠 가이드라인 (악보 형식)
- 블로그: 1,500자 이상, SEO 키워드 자연스럽게 포함
- 인스타: 150~300자, 해시태그 10~15개, 이모지 3~5개
- 뉴스레터: 600~800자, CTA 한 개
- 자문자답 구조 선호: "이게 왜 필요하냐고요? 바로..."

## 주요 소셜 채널 (공연 무대들)
- 유튜브: 파자마보스 (주 1회 롱폼)
- 인스타그램: @pajamaboss (주 3회)
- 뉴스레터: Beehiiv (주 1회)

## 핵심 키워드 (주요 레퍼토리)
- Primary: 클로드 코드, AI 마케팅, 마케팅 자동화
- Secondary: 1인 기업, 소상공인 AI, 비개발자 AI

## 사용 가능한 스킬 (우리 악기 목록)
- /social-media: 소셜미디어 콘텐츠 생성
- /repurpose: 콘텐츠 리퍼포징
- /seo-audit: SEO 감사
- /brand-voice: 브랜드 보이스 분석
- /newsletter: 뉴스레터 작성
- /marketing-team: AI 마케팅 팀 전체 실행 (오케스트라 합주)
```

[📺 화면: 붙여넣기 후 파일 내용 확인]

붙여넣었으면 이제 저장해야 해요. 키보드에서 Ctrl+S를 눌러주세요. 파일 탭에서 수정 표시(점)가 사라지면 저장 완료예요.

물론 여기 있는 내용은 저 파자마보스 브랜드 기준이에요. 보스님들은 브랜드명, 업종, 타겟 고객, 소셜 채널 부분을 여러분 브랜드에 맞게 수정해주시면 됩니다.

보스님들, 방금 우리 오케스트라의 총보를 완성했습니다!

## [CONCEPT] CLAUDE.md 위치에 따른 우선순위

[📺 화면: 슬라이드 → 웹캠]

[📊 PPT 지문: CLAUDE.md 우선순위 — 글로벌(기본 브랜드 정보) + 프로젝트(특수 지시사항) = 합쳐서 읽음]

한 가지 더 알아두실 게 있어요.

글로벌 설정은 홈 폴더 안에 ".claude" 폴더, 그 안에 CLAUDE.md를 넣으면 돼요. 이건 모든 프로젝트에 기본으로 적용되는 단골 레퍼토리예요.

프로젝트 설정은 지금처럼 프로젝트 폴더 안에 CLAUDE.md를 넣는 거예요. 이 프로젝트에만 적용되는 특별 공연 악보예요. 글로벌 설정보다 우선순위가 높아요.

두 개 다 있으면 Claude Code가 합쳐서 읽어요. 그래서 글로벌에는 기본 브랜드 정보, 프로젝트별로는 특수한 지시사항을 넣으면 좋아요.
