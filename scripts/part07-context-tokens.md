## [CONCEPT] 토큰 = 택시 미터기

[📺 화면: 슬라이드 → 웹캠]

[📊 PPT 지문: 토큰 = 택시 미터기 / 많이 올라가는 상황: 긴 대화, 큰 파일, 고급 모델 / 아끼는 방법: /clear 리셋, 필요한 파일만 참조, Haiku 모델 활용]

토큰은 Part 02에서 잠깐 말씀드렸죠? 택시 미터기예요.

대략적으로 이렇게 생각하시면 돼요. 한국어 한 글자가 약 2~3칸 올라가고, 영어 단어 하나는 약 1칸 올라가요. 코드 한 줄은 5~15칸 정도 올라가고요.

택시비가 많이 나오는 상황이 있어요. 대화가 길어질수록 미터기가 빠르게 올라가요. 큰 파일을 통째로 올릴 때도 많이 나오고요. 고급 모델을 사용할 때는 기본요금이 비싸요.

택시비를 아끼는 방법은요. /clear로 미터기를 리셋하면 돼요. 새 택시 타는 거예요. 필요한 파일만 참조해서 짐을 최소화하는 것도 중요하고요. 가벼운 작업에는 Haiku 모델을 쓰면 경차 택시라서 기본료가 싸요.

## [DEMO] 모델 선택 – 택시 등급 고르기

[📺 화면: PowerShell — Claude Code 실행 중]

[📊 PPT 지문: 택시 등급 비교표 — Opus(리무진, 복잡한 설계) / Sonnet(일반택시, 일상 콘텐츠) / Haiku(경차, 반복 작업)]

Claude Code에서 택시 등급, 즉 모델을 바꾸는 방법을 따라해보세요.

Claude Code 입력창에 이렇게 입력하세요.

```
/model
```

[📺 화면: 모델 목록 출력]

이렇게 입력하면 사용 가능한 모델 목록이 나와요. 방향키로 선택하거나 이름을 직접 입력하면 됩니다.

모델 세 가지만 기억하시면 돼요.

첫 번째는 Opus예요. 리무진이에요. 가장 똑똑하지만 비용이 높아요. 새 악기 설계하거나 복잡한 전략 짤 때 써요.

두 번째는 Sonnet이에요. 일반 택시예요. 균형 잡힌 성능에 가성비가 좋아요. 일상적인 콘텐츠 생성할 때 기본값으로 쓰면 돼요.

세 번째는 Haiku예요. 경차예요. 빠르고 저렴해요. 간단한 수정이나 반복 작업에 쓰면 택시비를 확 줄일 수 있어요.

제 전략은요. 새 악기, 즉 스킬 처음 설계할 때는 Opus로 최고 퀄리티로 만들고요. 일상적인 콘텐츠 생성은 Sonnet으로 가성비 최고로 해요. 대량 반복 작업은 Haiku로 택시비를 줄이죠. 이렇게 섞어 쓰면 품질은 유지하면서 월말에 한도 걱정이 없어요.

## [DEMO] /clear 실습 – 책상 깨끗이 치우기

[📺 화면: PowerShell — Claude Code 실행 중]

자, 책상 청소법을 직접 해볼게요. 따라하세요!

대화가 길어졌다 싶으면, 먼저 Claude Code 입력창에 이렇게 입력하세요.

```
지금까지 작업한 내용을 요약해줘.
파일명, 주요 결정사항, 다음에 이어서 해야 할 것 포함해서.
```

[📺 화면: Claude Code — 작업 요약 출력]

Claude가 요약해주면, 그 요약 텍스트를 마우스로 드래그해서 선택하고 Ctrl+C로 복사해두세요. 중요해요!

그 다음 입력창에 이렇게 입력하세요.

```
/clear
```

[📺 화면: 대화 초기화 완료]

책상이 깨끗해졌어요! 택시 미터기도 리셋됐습니다.

이제 새 대화를 시작할 때 아까 복사해둔 요약을 붙여넣으면 돼요. 입력창에 Ctrl+V로 붙여넣고 이렇게 추가해주세요.

```
이전 대화 요약:
[붙여넣기]

이어서 작업할게. 다음은 SEO 스킬 만들 차례야.
```

[📺 화면: 새 대화 시작 — 이전 요약으로 연속 작업]

깨끗한 책상에서 새 악보로 시작! 택시비도 절약!

## [CONCEPT] 실전 책상 관리 전략

[📺 화면: 슬라이드 → 웹캠]

[📊 PPT 지문: 실전 전략 3가지 — 작업 단위로 새 세션 / CLAUDE.md 활용 / 필요한 파일만 참조]

보스님들을 위한 실전 책상 관리 전략 세 가지예요.

첫 번째, 작업 단위로 택시 나눠 타세요. 한 번의 긴 대화에서 여러 가지 다 하려고 하지 마세요. 새 악기 만들기는 새 택시, 기존 악기 수정은 또 새 택시, 테스트는 또 새 택시. 이렇게 나눠서 타세요.

두 번째, 총보인 CLAUDE.md를 활용하세요. CLAUDE.md가 있으면 매번 브랜드 설명 안 해도 돼요. 택시 탈 때마다 목적지 설명 안 해도 되는 것처럼요.

세 번째, 악보는 필요한 것만 올리세요. 예를 들어 "내 마케팅 폴더 전체 분석해줘"라고 하면 책상이 한 번에 꽉 차버려요. 대신에 "@social-media.md 스킬 파일에 링크드인 포스트 생성 기능 추가해줘"라고 특정 파일만 지정하면 공간도 아끼고 택시비도 아낄 수 있어요.
