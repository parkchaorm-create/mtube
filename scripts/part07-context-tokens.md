## [CONCEPT] 컨텍스트 윈도우 = 지휘자의 책상

[📺 화면: 슬라이드 – "지휘자 책상 크기 = 컨텍스트 윈도우"]

컨텍스트 윈도우, 이름이 어려워 보이죠? 쉽게 말하면 지휘자 책상 크기예요.

이 책상 위에 올라가는 것들:
- 내가 한 말 (악보 요청)
- AI가 한 답 (연주 결과)
- 참고한 파일 (악보 원본)
- CLAUDE.md (총보)

이 전부가 책상 위에 올라갑니다.

책상이 꽉 차면?
- 오래된 내용을 "잊어버리기" 시작해요 (맨 아래 악보가 떨어짐)
- 응답이 이상해질 수 있어요
- 택시 미터기(비용)도 더 빨리 올라가요

"아니 그러면 책상을 깨끗이 치우면 되는 거 아니에요?"

## [CONCEPT] 토큰 = 택시 미터기

[📺 화면: 슬라이드 – "토큰 = 택시 미터기"]

토큰은 Part 2에서 잠깐 말씀드렸죠? 택시 미터기예요.

대략적으로:
- 한국어 1글자 = 약 2~3칸 올라감
- 영어 단어 1개 = 약 1칸 올라감
- 코드 한 줄 = 5~15칸 올라감

택시비가 많이 나오는 상황:
- 대화가 길어질수록 (장거리 운행 = 미터기 미친 듯이 올라감)
- 큰 파일을 통째로 올릴 때 (짐이 많으면 대형 택시 필요)
- 고급 모델(Opus) 사용할 때 (리무진 택시 = 기본요금이 비쌈)

택시비를 아끼는 방법:
- `/clear`로 미터기 리셋 (새 택시 타기)
- 필요한 파일만 참조 (짐 최소화)
- 가벼운 작업엔 Haiku 사용 (경차 택시)

## [DEMO] 모델 선택 – 택시 등급 고르기

[📺 화면: 터미널 - Claude Code]

Claude Code에서 택시 등급(모델)을 바꾸는 방법, 따라해보세요.

```
/model
```

이렇게 입력하면 사용 가능한 모델 목록이 나와요.

| 택시 등급 | 모델명 | 특징 | 언제 타나 |
|------|------|------|-----------| 
| 🚐 리무진 | claude-opus-4-5 | 가장 똑똑, 비용 높음 | 새 악기 설계, 복잡한 전략 |
| 🚗 일반 택시 | claude-sonnet-4-5 | 균형잡힌 성능 | 일반 콘텐츠 생성 (기본값) |
| 🛵 경차 | claude-haiku-4-5 | 빠르고 저렴 | 간단한 수정, 반복 작업 |

제 전략:
- 새 악기(스킬) 처음 설계할 때: 리무진 (Opus) → 최고 퀄리티로
- 일상적인 콘텐츠 생성: 일반 택시 (Sonnet) → 가성비 최고
- 대량 반복 작업: 경차 (Haiku) → 택시비 확 줄이기

이렇게 섞어 쓰면 품질은 유지하면서 월말에 한도 걱정 없어요.

## [DEMO] /clear 실습 – 책상 깨끗이 치우기

[📺 화면: 터미널 - Claude Code]

자, 책상 청소법을 직접 해볼게요. 따라하세요!

대화가 길어졌다 싶으면, 먼저 이렇게 입력하세요:

```
지금까지 작업한 내용을 요약해줘.
파일명, 주요 결정사항, 다음에 이어서 해야 할 것 포함해서.
```

Claude가 요약해주면, 그 요약을 복사해두세요!

그 다음:

```
/clear
```

책상이 깨끗해졌어요! 택시 미터기도 리셋!

이제 새 대화 시작할 때 아까 복사한 요약을 붙여넣으세요:

```
이전 대화 요약:
[붙여넣기]

이어서 작업할게. 다음은 SEO 스킬 만들 차례야.
```

🎉 **깨끗한 책상에서 새 악보로 시작! 택시비도 절약!**

## [CONCEPT] 실전 책상 관리 전략

[📺 화면: 슬라이드]

보스님들을 위한 실전 책상 관리 전략 세 가지.

**전략 1: 작업 단위로 택시 나눠 타기**

한 번의 긴 대화에서 여러 가지 다 하려고 하지 마세요.

- 새 악기 만들기 → 새 택시
- 기존 악기 수정 → 새 택시
- 테스트 → 새 택시

**전략 2: 총보(CLAUDE.md) 활용하기**

CLAUDE.md가 있으면 매번 브랜드 설명 안 해도 돼요. 택시 탈 때마다 목적지 설명 안 해도 되는 것처럼!

**전략 3: 악보는 필요한 것만 올리기**

나쁜 예:
```
내 마케팅 폴더 전체 분석해줘
```

좋은 예:
```
@social-media.md 스킬 파일에 링크드인 포스트 생성 기능 추가해줘.
```

특정 악보만 책상에 올리면 공간도 아끼고 택시비도 아낍니다!
