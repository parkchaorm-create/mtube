## [CONCEPT] 설치할 세 가지 장비

[📺 화면: 슬라이드 → 웹캠으로 설명]

[📊 PPT 지문: 공연장 장비 3종 세트 — VS Code(무대/공연 공간) / Node.js(전기 배선/동력) / Claude Code(음향 장비/핵심)]

우리가 설치할 장비는 딱 세 가지예요.

첫 번째는 VS Code입니다. 오케스트라가 연주할 무대예요. 코드 에디터라고 부르는 프로그램인데, 완전 무료예요. Claude Code가 가장 편하게 일할 수 있는 환경이에요.

두 번째는 Node.js입니다. 공연장에 전기를 들어오게 하는 배선 작업이에요. 음향 장비가 작동하려면 전기가 있어야 하잖아요? 한 번 깔아두면 다시는 신경 쓸 일 없어요.

세 번째가 진짜 핵심입니다. Claude Code예요. AI 오케스트라가 실제로 연주하게 해주는 음향 장비예요.

이 순서대로, 하나씩 같이 설치해볼게요!

## [DEMO] 장비 1: VS Code 설치 (무대 세팅)

[📺 화면: 브라우저 열기 → code.visualstudio.com 접속]

첫 번째 장비, 무대(VS Code)를 세팅하겠습니다.

브라우저를 열어주세요. 엣지나 크롬 아무거나 상관없어요. 주소창에 "code.visualstudio.com"을 입력하고 엔터를 눌러주세요.

[📺 화면: VS Code 공식 홈페이지]

페이지 가운데에 파란색 큰 버튼이 보이실 거예요. 거기에 "Download for Windows"라고 써있을 거예요. 그 버튼을 클릭해주세요.

맥 사용자분들은 버튼 아래 작게 있는 "Other platforms" 링크를 클릭해서 Mac 버전을 받으시면 됩니다.

[📺 화면: 다운로드 진행 → 파일 탐색기에서 설치 파일 확인]

다운로드 완료되면 파일 탐색기에서 설치 파일을 찾아볼게요. 바탕화면이나 다운로드 폴더에 "VSCodeSetup..."으로 시작하는 .exe 파일이 있을 거예요. 그걸 더블클릭해서 실행해주세요.

설치 창이 뜨면 "동의합니다" 클릭하고, 그다음은 계속 "다음"만 눌러주시면 돼요. 마지막에 "설치" 버튼 클릭하면 끝납니다.

맥 사용자분들은 .zip 파일 압축 풀면 VS Code 앱이 나오는데, 그걸 응용 프로그램 폴더로 끌어다 놓으시면 됩니다.

[📺 화면: VS Code 첫 실행 화면 — 어두운 배경의 코드 에디터]

설치 완료되면 VS Code를 실행해보세요. 이렇게 어두운 배경에 코드 에디터 화면이 나오면 성공이에요. 무대 세팅 완료!

## [DEMO] 장비 2: Node.js 설치 (전기 배선)

[📺 화면: 브라우저 → nodejs.org 접속]

두 번째 장비, 전기 배선(Node.js)을 설치하겠습니다.

브라우저 주소창에 "nodejs.org"를 입력하고 엔터를 눌러주세요.

[📺 화면: Node.js 공식 홈페이지]

페이지 가운데에 두 개의 버튼이 보일 거예요. 왼쪽에 "LTS" 라고 써있는 초록 버튼이 있고, 오른쪽에 "Current"가 있어요. 반드시 왼쪽의 LTS 버튼을 클릭해주세요. LTS가 안정적인 버전이에요. 오른쪽 Current는 최신이지만 불안정할 수 있으니까 건드리지 마세요.

[📺 화면: 다운로드 → 설치 파일 실행]

다운로드된 파일을 더블클릭해서 실행해주세요. 설치 창이 뜨면 "Next" 버튼을 계속 눌러주시면 됩니다. 중간에 체크박스 같은 거 건드리실 필요 없어요. 그냥 "Next", "Next", "Install" 순서로 클릭하시면 돼요.

맥 사용자분들은 .pkg 파일 더블클릭하고 "계속"을 누르다 보면 설치 완료됩니다.

[📺 화면: 설치 완료 → 파워셸 실행]

설치 완료됐으면 제대로 됐는지 확인해볼게요. 윈도우 사용자분들은 키보드에서 윈도우 키를 누르고 "PowerShell"을 검색해서 실행해주세요. 파란 배경에 커맨드 라인이 뜨는 창이에요.

[📺 화면: PowerShell 창]

파워셸 창이 뜨면 여기에 "node --version"을 입력하고 엔터를 눌러주세요. 이렇게 치면 됩니다.

```bash
node --version
```

[📺 화면: v20.xx.x 버전 숫자 출력]

v20이나 v22 같은 버전 숫자가 나오면 전기 배선 성공이에요! 숫자가 뜨지 않고 "명령을 찾을 수 없습니다" 같은 에러가 나오면 PowerShell을 완전히 닫고 다시 열어보세요. 설치 후에는 창을 새로 열어야 반영이 돼요.

## [DEMO] 장비 3: Claude Code 설치 (음향 장비 – 핵심!)

[📺 화면: PowerShell 창]

이제 진짜 핵심입니다! 음향 장비(Claude Code)를 설치할게요.

아까 열었던 PowerShell 창으로 돌아오세요. 여기에 아래 명령어를 그대로 복사해서 붙여넣고 엔터를 눌러주세요. 직접 치려고 하지 마시고 꼭 복사 붙여넣기 하세요.

```bash
npm install -g @anthropic-ai/claude-code
```

[📺 화면: 설치 진행 화면 — 텍스트가 쭉 내려가는 모습]

이렇게 글자들이 쭉쭉 내려가면서 설치가 진행되고 있는 거예요. 30초에서 1분 정도 걸리니까 기다려주세요. 멈춘 것처럼 보여도 괜찮아요. 그냥 기다리시면 됩니다.

[📺 화면: 설치 완료 — 커맨드 라인으로 돌아온 화면]

설치 완료됐으면 이렇게 확인해볼게요. 파워셸에 이걸 입력하고 엔터 눌러주세요.

```bash
claude --version
```

[📺 화면: 버전 번호 출력]

버전 번호가 나오면 음향 장비 설치 완료입니다! 보스님들, 여기까지 오셨으면 공연장 세팅의 90%가 끝난 거예요.

## [TROUBLESHOOT] 자주 나오는 에러 해결법 (장비 불량 대처)

[📺 화면: 웹캠 단독]

[📊 PPT 지문: 자주 나오는 에러 3가지 — 권한 오류 / npm 명령어 없음 / claude 명령어 없음]

설치하다 에러 나는 분들 계실 거예요. 장비가 가끔 불량일 수 있잖아요? 가장 자주 나오는 에러 세 가지만 알려드릴게요.

첫 번째 에러예요. 파워셸에 빨간 글씨로 "'npm'은(는) 내부 또는 외부 명령, 실행할 수 있는 프로그램, 또는 배치 파일이 아닙니다"라고 나오는 경우가 있어요. 이건 Node.js 설치가 제대로 안 됐거나, PowerShell을 관리자 권한으로 열지 않은 경우예요. 해결법은 PowerShell을 닫고, 다시 검색해서 PowerShell 아이콘 위에서 마우스 오른쪽 버튼을 클릭하세요. 메뉴가 뜨면 "관리자 권한으로 실행"을 선택하고 다시 명령어를 입력해보세요.

두 번째 에러예요. 설치는 다 됐는데 "claude --version"을 입력했을 때 "claude를 찾을 수 없습니다"라고 나오는 경우예요. 이건 PowerShell을 새로 열지 않아서 그래요. PowerShell을 완전히 닫고, 다시 열어서 입력해보세요. 전원 코드를 뽑았다 다시 꽂는 것처럼, 창을 재시작하면 연결됩니다.

맥 사용자분들은 "permission denied" 에러가 나오면 명령어 앞에 "sudo"를 붙이고 맥 비밀번호를 입력하시면 됩니다.

그래도 해결이 안 되면 댓글에 에러 메시지 올려주세요. 제가 직접 확인해드리겠습니다.

## [DEMO] VS Code에서 Claude Code 실행 (무대와 음향 연결)

[📺 화면: VS Code 실행 화면]

이제 무대(VS Code)와 음향 장비(Claude Code)를 연결하겠습니다.

VS Code를 열어주세요. 아까 설치한 거요. 열면 이렇게 어두운 화면이 뜨잖아요?

[📺 화면: VS Code — 상단 메뉴바 "Terminal" 클릭]

화면 맨 위 메뉴바를 보시면 "Terminal"이라고 있어요. 그걸 클릭하면 드롭다운 메뉴가 나오는데, 거기서 "New Terminal"을 클릭해주세요.

[📺 화면: VS Code 하단에 터미널 창 열림]

그러면 VS Code 화면 아래쪽에 검은 창이 하나 열려요. 이게 VS Code 안에 내장된 터미널이에요. 여기에 claude를 입력하고 엔터를 눌러주세요.

```bash
claude
```

[📺 화면: 브라우저 자동 열림 — claude.ai 로그인 화면]

그러면 브라우저가 자동으로 열리면서 로그인하라는 화면이 떠요. 아까 Part 02에서 Pro 구독하면서 만든 계정으로 로그인하시면 됩니다. 구글 계정 연동도 되니까 편하신 방법으로 로그인하세요.

[📺 화면: 브라우저 로그인 완료 → VS Code 터미널로 돌아옴]

로그인하면 브라우저에 "You're now connected"라고 뜨고, VS Code 터미널로 자동으로 돌아와요. 터미널에 이런 문구가 보이면 성공이에요.

```
✓ Claude Code ready
Type your request below...
```

보스님들, 공연장 세팅이 완성됐습니다!
