## [DEMO] 장비 1: VS Code 설치 (무대 세팅)

[📺 화면: 브라우저 → code.visualstudio.com]

첫 번째 장비, 무대(VS Code)를 세팅합니다.

VS Code는 무료 코드 에디터인데요, Claude Code가 가장 편하게 연주할 수 있는 무대입니다.

code.visualstudio.com에 접속하시면 큰 파란 버튼으로 "Download" 보이시죠?

**윈도우 사용자**: Windows 버튼 클릭 → .exe 파일 실행 → 설치 마법사 따라가기 (모두 Next)

[📺 화면: VS Code 설치 완료 화면]

설치 완료되면 VS Code 실행해보세요. 어두운 배경에 코드 에디터 화면 나오면 성공입니다.

🎉 **축하합니다! 무대 세팅 완료!**

## [DEMO] 장비 2: Node.js 설치 (전기 배선)

[📺 화면: 브라우저 → nodejs.org]

두 번째 장비, 전기 배선(Node.js)을 깔겠습니다.

"Node.js가 뭔데요?"

공연장에 전기를 들어오게 하는 배선 작업이에요. 음향 장비(Claude Code)가 작동하려면 전기(Node.js)가 있어야 하거든요. 한 번 깔아두면 다시는 신경 쓸 일 없습니다.

nodejs.org 접속하시면 LTS 버전 버튼이 보일 거예요. LTS로 설치하세요. 이게 안정적인 버전입니다.

**맥 사용자**: .pkg 파일 다운로드 → 더블클릭 → 계속 → 설치

**윈도우 사용자**: .msi 파일 다운로드 → 실행 → Next 계속 클릭

설치 확인하는 방법! 터미널(맥) 또는 PowerShell(윈도우)을 열고 이걸 입력해보세요.

```bash
node --version
```

v20 이상 숫자가 나오면 전기 배선 성공이에요! ⚡

## [DEMO] 장비 3: Claude Code 설치 (음향 장비 – 핵심!)

[📺 화면: 터미널(맥) / PowerShell(윈도우) 화면]

이제 진짜 핵심! 음향 장비(Claude Code) 설치입니다.

터미널이나 PowerShell을 열고, 아래 명령어를 복사해서 붙여넣으세요.

**모든 운영체제 공통 (가장 쉬운 방법)**:

```bash
npm install -g @anthropic-ai/claude-code
```

[📺 화면: 명령어 입력 → 설치 진행 화면]

설치 진행되는 거 보이시죠? 이게 한 30초~1분 정도 걸려요. 기다려주세요.

완료되면 이렇게 확인할 수 있어요.

```bash
claude --version
```

버전 번호 나오면 음향 장비 설치 완료입니다! 🎤🎉

## [TROUBLESHOOT] 자주 나오는 에러 해결법 (장비 불량 대처)

[📺 화면: 웹캠 + 화면 공유 (에러 화면)]

설치하다 에러 나는 분들 계실 거예요. 장비가 가끔 불량일 수 있잖아요. 자주 나오는 에러 3가지만 미리 알려드릴게요.

**에러 1: "permission denied" (맥) – 전기 차단기가 내려간 경우**

```bash
sudo npm install -g @anthropic-ai/claude-code
```

앞에 `sudo`를 붙이고, 맥 비밀번호 입력하면 해결됩니다. 차단기 올려주는 거예요.

**에러 2: "'npm'은(는) 내부 또는 외부 명령이 아닙니다" (윈도우) – 전기 배선이 안 된 경우**

Node.js가 제대로 설치 안 됐거나, PowerShell을 관리자 권한으로 실행 안 한 경우예요.
→ PowerShell 우클릭 → "관리자 권한으로 실행" → 다시 시도

**에러 3: 설치는 됐는데 `claude` 명령어가 안 먹혀요 – 장비가 아직 연결 안 된 경우**

터미널이나 PowerShell을 완전히 닫고 다시 여세요. 전원 코드를 뽑았다 다시 꽂는 것처럼, 재시작하면 연결됩니다.

그래도 안 된다면? 댓글에 에러 메시지 올려주시면 제가 확인해드리겠습니다.

## [DEMO] VS Code에서 Claude Code 연동 (무대와 음향 연결)

[📺 화면: VS Code 화면]

이제 무대(VS Code)와 음향 장비(Claude Code)를 연결하겠습니다.

**방법 1: VS Code 확장 프로그램 설치 (추천)**

VS Code 왼쪽 사이드바에서 Extensions 아이콘 클릭 (네모 4개 모양)

검색창에 "Claude Code" 입력

Anthropic이 만든 공식 확장 프로그램 설치 클릭

설치 완료!

**방법 2: 터미널에서 직접 실행**

VS Code에서 터미널 열기 (`Ctrl+~` 또는 `Cmd+~`)

```bash
claude
```

입력하면 Claude Code 시작됩니다.

[📺 화면: Claude Code 첫 실행 화면]

처음 실행하면 브라우저가 열리면서 로그인하라는 창이 뜹니다.

아까 만든 claude.ai Pro 계정으로 로그인하시면 돼요.

로그인 완료되면 VS Code 터미널에 이렇게 뜹니다.

```
✓ Claude Code ready
Type your request below...
```

이게 나오면 완성입니다! 

🎉 **보스님들, 공연장 세팅 완료! 무대, 전기, 음향 다 올라갔어요!**
