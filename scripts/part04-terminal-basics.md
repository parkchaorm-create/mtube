## [CONCEPT] 터미널이란? (2분 이내 설명)

[📺 화면: 슬라이드 → 웹캠으로 설명]

[📊 PPT 지문: 터미널 = 무대 뒤 통로(백스테이지) / 마우스 클릭 = 정문 통과 / 터미널 = 백스테이지 직행 통로]

터미널을 쉽게 설명하면 이렇습니다.

여러분이 평소에 마우스로 폴더 더블클릭하고, 파일 드래그하고, 버튼 클릭하잖아요. 터미널은 그걸 글로 하는 거예요.

예를 들어 "바탕화면에 marketing 폴더 만들어줘"를 마우스로 하면 바탕화면 우클릭, 새 폴더, 이름 입력, 이렇게 세 단계잖아요. 터미널로는 "mkdir marketing" 이렇게 한 줄이면 끝이에요.

"아니 그러면 왜 굳이 터미널을 써야 해요? 마우스가 더 편하잖아요."

좋은 질문이에요. Claude Code가 이 터미널에서 실행되기 때문이에요. 오케스트라 음향 장비가 무대 뒤 통로에 있으니까, 이 통로로 가야 장비를 켤 수 있는 거예요.

## [DEMO] 딱 5개 동작 마스터

[📺 화면: PowerShell 창 전체화면]

[📊 PPT 지문: 터미널 5대 명령어 — pwd / ls(dir) / cd / mkdir / claude]

자, 이제 백스테이지 이동법 딱 5개만 익히면 됩니다. 직접 따라하면서 배워요.

윈도우 사용자분들은 파워셸을 열어주세요. 아까 Part 03에서 썼던 그 파란 창이에요. 윈도우 키를 누르고 "PowerShell"을 검색해서 실행하시면 됩니다.

맥 사용자분들은 Spotlight에서 "터미널"을 검색해서 실행하시면 돼요.

---

### 동작 1: `pwd` – "지금 나 어디에 있지?" (현재 위치 확인)

[📺 화면: PowerShell — pwd 입력 → 결과 출력]

파워셸에 "pwd"를 입력하고 엔터를 눌러보세요.

```bash
pwd
```

이렇게 경로가 출력되죠? 예를 들어 "C:\Users\내이름" 이런 식으로요. 지금 내가 백스테이지 어느 복도에 서 있는지 알려주는 거예요. 일을 시작하기 전에 늘 이걸로 위치 확인하는 습관을 만드세요.

---

### 동작 2: `dir` – "이 복도에 뭐가 있지?" (파일 목록 보기)

[📺 화면: PowerShell — dir 입력 → 파일/폴더 목록 출력]

이번엔 "dir"을 입력하고 엔터를 눌러보세요.

```bash
dir
```

지금 서 있는 복도에 어떤 방(폴더)과 물건(파일)이 있는지 목록으로 보여줍니다. 맥이나 리눅스 환경에서는 "ls"를 쓰면 같은 결과가 나와요.

---

### 동작 3: `cd` – "저 방으로 이동!" (폴더 이동)

[📺 화면: PowerShell — cd Documents 입력 → 경로 변경 확인]

이번엔 폴더 안으로 들어가볼게요. "cd Documents"를 입력하고 엔터를 눌러주세요.

```bash
cd Documents
```

커서 앞에 표시되는 경로가 바뀐 거 보이시죠? 이제 Documents 폴더 안으로 들어온 거예요.

이전 복도로 돌아가고 싶을 때는 "cd .."을 입력하면 됩니다. 점 두 개예요.

```bash
cd ..
```

한 발짝 뒤로 가는 거예요.

---

### 동작 4: `mkdir` – "새 방 만들기" (폴더 생성)

[📺 화면: PowerShell — mkdir my-ai-marketing 입력 → dir로 확인]

이번엔 새 폴더를 만들어볼게요. "mkdir my-ai-marketing"을 입력하고 엔터를 눌러주세요.

```bash
mkdir my-ai-marketing
```

그다음에 "dir"을 입력해서 확인해보시면, 방금 만든 my-ai-marketing 폴더가 목록에 생긴 게 보일 거예요. 이게 앞으로 우리 오케스트라 연습실이 될 폴더예요.

---

### 동작 5: `claude` – "음향 장비 켜기!" (Claude Code 시작)

[📺 화면: PowerShell — claude 입력 → Claude Code 실행 화면]

마지막이자 가장 중요한 동작이에요. 그냥 "claude"라고 입력하고 엔터를 누르면 됩니다.

```bash
claude
```

이걸 치면 음향 장비가 켜지고, 오케스트라가 연주할 준비가 됩니다. 아까 Part 03에서 이미 한 번 해봤죠?

## [DEMO] 5개 동작으로 오케스트라 연습실 만들기

[📺 화면: PowerShell 전체화면]

자, 방금 배운 5개를 합쳐서 실제 연습실을 세팅해볼게요. 저 따라오세요!

먼저 지금 어디 있는지 확인할게요.

```bash
pwd
```

경로 확인됐죠? 이제 Documents 폴더로 이동합니다.

```bash
cd Documents
```

이제 오케스트라 연습실 폴더를 만들게요.

```bash
mkdir my-ai-marketing
```

만든 폴더 안으로 들어갑니다.

```bash
cd my-ai-marketing
```

현재 위치 다시 확인해볼게요. "Documents\my-ai-marketing"이 나오면 정상이에요.

```bash
pwd
```

이제 음향 장비를 켜겠습니다.

```bash
claude
```

[📺 화면: Claude Code 실행 완료 화면]

어때요? 생각보다 쉽죠? 보스님들, 방금 오케스트라 연습실을 만들고 음향까지 켰습니다!

## [BONUS] 알면 편한 추가 팁 2가지

[📺 화면: PowerShell — 팁 시연]

보너스로 두 가지만 더 알려드릴게요.

첫 번째, Tab 키 자동완성이에요. 폴더 이름 치다가 Tab 키를 누르면 자동으로 완성됩니다. 예를 들어 "cd Doc"까지만 치고 Tab 키를 누르면 "cd Documents\"로 자동완성이 돼요. 오타 줄이는 꿀팁이에요.

두 번째, 위 화살표 키입니다. 파워셸에서 키보드 위쪽 화살표 키를 누르면 이전에 입력했던 명령어가 다시 불러와져요. 같은 명령어 반복할 때 타이핑 안 해도 되니까 굉장히 편해요.

## [CONCEPT] Claude Code 내부 명령어

[📺 화면: PowerShell — Claude Code 실행 중]

[📊 PPT 지문: Claude Code 내부 명령어 목록 — /help / /clear / /model / /plan / /quit]

Claude Code 실행하고 나서 쓸 수 있는 명령어도 몇 가지 있어요. 이건 파워셸 명령어가 아니고, Claude Code 안에서 쓰는 명령어입니다. 음향 장비의 버튼들이라고 생각하세요.

슬래시로 시작하는 게 포인트예요. "/help"라고 치면 모든 버튼 설명서가 나와요. "/clear"는 대화를 초기화하는 버튼이고요, 아까 말한 택시 미터기 리셋이에요. "/model"은 AI 모델을 바꿀 때 쓰고, "/plan"은 리허설 모드를 시작하는 건데 뒤에서 자세히 다룰 거예요. "/quit"은 장비를 끄는 거예요.

이것들은 나중에 하나씩 실습할 거니까, 지금은 "이런 버튼들이 있구나" 정도만 알고 계세요.
