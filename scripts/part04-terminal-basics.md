## [CONCEPT] 터미널이란? (2분 이내 설명)

[📺 화면: 슬라이드 – "터미널 = 무대 뒤 통로"]

터미널을 쉽게 설명하면 이렇습니다.

여러분이 평소에 마우스로 폴더 더블클릭하고, 파일 드래그하고, 버튼 클릭하잖아요.

터미널은 그걸 글로 하는 거예요.

예를 들어, "바탕화면에 'marketing' 폴더 만들어줘"를 마우스로 하면:
바탕화면 우클릭 → 새 폴더 → 이름 입력

터미널로 하면:
```bash
mkdir marketing
```

끝이에요. 이게 전부입니다.

"아니 그러면 왜 굳이 터미널을 써야 해요? 마우스가 더 편하잖아요."

좋은 질문이에요! Claude Code가 이 터미널에서 실행되기 때문이에요. 오케스트라 음향 장비가 무대 뒤 통로에 있으니까, 이 통로로 가야 장비를 켤 수 있는 거예요.

## [DEMO] 딱 5개 동작 마스터

[📺 화면: 터미널 전체화면]

자, 이제 백스테이지(터미널) 이동법 딱 5개만 외우면 됩니다. 따라하면서 배워요.

---

### 동작 1: `pwd` – "지금 나 어디에 있지?" (현재 위치 확인)

```bash
pwd
```

[📺 화면: 터미널 - pwd 실행 결과]

이걸 입력하고 엔터 치면, 지금 내가 백스테이지 어디에 서 있는지 알려줍니다.

예: `/Users/mac/Documents`

공연장 안에서 "지금 내가 어느 복도에 있지?" 확인하는 거예요.

---

### 동작 2: `ls` – "이 복도에 뭐가 있지?" (파일 목록 보기)

```bash
ls
```

[📺 화면: 터미널 - ls 실행 결과]

지금 서 있는 복도에 어떤 방(폴더)과 물건(파일)이 있는지 보여줍니다.

윈도우에서는 `dir`이에요.

```bash
dir
```

---

### 동작 3: `cd` – "저 방으로 이동!" (폴더 이동)

```bash
cd Documents
```

[📺 화면: 터미널 - cd 실행 → pwd로 확인]

`cd` 뒤에 가고 싶은 방(폴더) 이름 쓰면 그 방으로 이동합니다.

이전 복도로 돌아가고 싶을 때는:

```bash
cd ..
```

마침표 두 개는 "한 발짝 뒤로"라는 뜻이에요.

---

### 동작 4: `mkdir` – "새 방 만들기" (폴더 생성)

```bash
mkdir my-marketing
```

[📺 화면: 터미널 - mkdir 실행 → ls로 확인]

`my-marketing`이라는 새 방이 생겼어요.

오늘 우리 오케스트라 연습실을 이렇게 만들 거예요.

---

### 동작 5: `claude` – "음향 장비 켜기!" (Claude Code 시작)

```bash
claude
```

[📺 화면: 터미널 - claude 실행 화면]

이게 마지막이자 가장 중요한 동작이에요.

이걸 치면 음향 장비가 켜지고, 오케스트라가 연주할 준비가 됩니다!

## [DEMO] 5개 동작으로 오케스트라 연습실 만들기

[📺 화면: 터미널 전체화면]

자, 방금 배운 5개를 합쳐서 실제 연습실을 세팅해볼게요.

따라하세요!

```bash
# 1. 지금 어디 있는지 확인 (현재 위치 확인)
pwd

# 2. Documents 복도로 이동
cd Documents

# 3. 오케스트라 연습실 만들기
mkdir my-ai-marketing

# 4. 연습실로 입장
cd my-ai-marketing

# 5. 현재 위치 확인 (Documents/my-ai-marketing에 있어야 함)
pwd

# 6. 음향 장비 켜기!
claude
```

[📺 화면: 각 단계 실행 결과 보여주기]

어때요? 생각보다 쉽죠?

🎉 **보스님들, 방금 오케스트라 연습실을 만들고 음향까지 켰습니다!**

## [BONUS] 알면 편한 추가 팁 2가지

[📺 화면: 터미널 - 팁 시연]

보너스로 두 가지만 더 알려드릴게요.

**팁 1: Tab 키 자동완성**

폴더 이름 치다가 Tab 키 누르면 자동으로 완성됩니다.

예: `cd Doc` 치고 Tab → `cd Documents/` 자동완성

오타 줄이는 꿀팁이에요! 백스테이지에서 빠르게 이동하는 비법.

**팁 2: 위 화살표 키 – 이전 명령어 불러오기**

위쪽 화살표 키 누르면 이전에 입력한 명령어가 다시 나와요.

같은 동작 반복할 때 타이핑 안 해도 돼서 편해요.

## [CONCEPT] Claude Code 내부 명령어

[📺 화면: 터미널 - Claude Code 실행 중]

Claude Code 실행하고 나서 쓸 수 있는 명령어도 몇 가지 있어요.

이건 터미널 명령어가 아니고, Claude Code 안에서 쓰는 명령어입니다. 음향 장비의 버튼들이라고 생각하세요.

```
/help     → 모든 버튼 설명서
/clear    → 대화 초기화 (택시 미터기 리셋)
/model    → AI 모델 변경 (악단 편성 변경)
/plan     → Plan Mode 시작 (리허설 모드, 뒤에 자세히!)
/quit     → 장비 끄기
```

슬래시(/)로 시작하는 게 포인트예요.

이것들은 나중에 하나씩 실습할 거니까, 지금은 "이런 버튼이 있구나" 정도만 알고 계세요.
