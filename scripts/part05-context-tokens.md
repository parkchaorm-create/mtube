## [CONCEPT] 토큰 = 택시 미터기

[📺 화면: 슬라이드 → 웹캠]

[📊 PPT 지문: 토큰 = 택시 미터기 / 한국어 1글자 ≈ 2~3칸 / 영어 1단어 ≈ 1칸 / 아끼는 법: /clear 리셋 + 필요한 파일만 참조 + Haiku 경차 활용]

토큰은요, 택시 미터기입니다.
대화할수록 칸칸이 올라가죠.

대략 이렇게 생각하시면 됩니다.
한국어 한 글자가 약 2~3칸.
영어 단어 하나는 약 1칸 올라갑니다.

근데요, 택시비가 많이 나오는 상황이 있거든요.

대화가 길어질수록 미터기가 빠르게 올라가고요.
큰 파일을 통째로 올리면 확 튑니다.

택시비 아끼는 방법은요?
/clear로 미터기를 리셋하면 됩니다.
새 택시 타는 거죠.

그리고요, 필요한 파일만 참조해서 짐을 최소화하는 것도 중요합니다.

## [DEMO] 모델 선택 + /clear 실습

[📺 화면: 터미널 — Claude Code 실행 중]

[📊 PPT 지문: 택시 등급 비교 — Opus(리무진, 복잡한 설계) / Sonnet(일반택시, 일상 콘텐츠) / Haiku(경차, 반복 작업)]

택시 등급, 그러니까 모델을 바꾸는 법부터 따라해보겠습니다.
Claude Code 입력창에 이렇게 입력하세요.

```
/model
```

[📺 화면: 모델 목록 출력, 방향키로 선택하는 장면]

모델 세 가지만 기억하시면 됩니다.

Opus는 리무진입니다.
가장 똑똑한데요, 비용이 높죠.
새 스킬 설계할 때 쓰면 됩니다.

Sonnet은 일반 택시입니다.
균형 잡힌 가성비로 일상 콘텐츠 생성에 딱이고요.
기본값으로 쓰시면 됩니다.

Haiku는 경차입니다.
빠르고 저렴해서 간단한 수정이나 반복 작업에 딱이죠.

자, 이제 책상 청소법입니다.
대화가 길어졌다 싶으면요, 먼저 이렇게 입력하세요.

```
"지금까지 작업한 내용을 요약해줘. 파일명, 주요 결정사항, 다음에 이어서 해야 할 것 포함해서."
```

[📺 화면: Claude Code — 작업 요약 출력]

Claude가 요약해주면 그 텍스트를 복사해두세요.

그 다음에 입력창에 이렇게 입력합니다.

```
/clear
```

[📺 화면: 대화 초기화 완료]

책상이 깨끗해졌습니다!
택시 미터기도 리셋됐죠.

새 대화를 시작할 때 아까 복사해둔 요약을 붙여넣고 이어서 작업하면 됩니다.

```
"이전 대화 요약:
[붙여넣기]

이어서 작업할게. 다음은 SEO 스킬 만들 차례야."
```

깨끗한 책상에서 새 악보로 시작하는 겁니다.
택시비도 절약되고요.
꿀팁이죠?
