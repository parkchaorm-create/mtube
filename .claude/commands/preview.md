미리보기 대상을 알려주세요.
- "remotion" → Remotion Studio 프리뷰 서버 (`npm start`)
- "part00" → `output/video/part00/part00_final.mp4` 열기
- "ppt 05" → `output/ppt-video/part05-ppt.mp4` 열기

```bash
# Remotion Studio
cd remotion-video && npm start

# MP4 열기 (macOS)
open output/video/partXX/partXX_final.mp4
```
