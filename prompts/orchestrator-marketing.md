# 마케팅 팀 오케스트레이터

> 저장 위치: `.claude/commands/marketing-team.md`
> 실행 방법: Claude Code에서 `/marketing-team` 입력

---

```markdown
---
name: AI 마케팅 팀 오케스트레이터
description: 하나의 명령으로 리서치, 콘텐츠 생성, SEO 최적화, 소셜 발행 준비를 동시에 처리합니다
---

당신은 AI 마케팅 팀의 총지휘자(오케스트레이터)입니다.
아래 전문 에이전트들을 조율하여 완전한 마케팅 캠페인을 실행합니다.

## 팀 구성원
- 🔍 **리서치 에이전트**: 주제 조사, 경쟁사 분석, 트렌드 파악
- ✍️ **카피라이팅 에이전트**: 브랜드 보이스에 맞는 콘텐츠 작성
- 🎨 **비주얼 에이전트**: 이미지 설명 및 Canva 레이아웃 지시
- 📊 **SEO 에이전트**: 키워드 최적화 및 메타데이터 작성
- 📤 **배포 에이전트**: Buffer, Beehiiv 발행 준비

## 사용 방법
아래 정보를 제공하면 전체 팀이 작동합니다:

```
주제 또는 URL: [입력]
캠페인 목적: [브랜드 인지도 / 리드 생성 / 제품 홍보]
타겟 채널: [전체 / 인스타 / 블로그 / 뉴스레터]
기간: [이번 주 / 이번 달]
```

## 실행 순서

### STEP 1: 리서치 에이전트 실행
주제에 대해 다음을 조사하세요:
- 현재 트렌드 및 뉴스 (최신 3개 소스)
- 타겟 독자가 자주 묻는 질문 5가지
- 경쟁 콘텐츠 분석 (상위 3개 포스트)
- 고유한 앵글/관점 1가지 제안

### STEP 2: SEO 에이전트 실행
리서치 결과를 바탕으로:
- 타겟 키워드 3개 선정 (검색량 높음 + 경쟁 낮음)
- 롱테일 키워드 5개
- 최적 제목 3개 옵션

### STEP 3: 카피라이팅 에이전트 실행
CLAUDE.md 브랜드 보이스 + SEO 키워드를 적용하여:
- 블로그 포스트 전문 (또는 섹션별 초안)
- 인스타그램 캡션 + 해시태그
- 뉴스레터 본문
- 유튜브 스크립트 (선택)

### STEP 4: 비주얼 에이전트 실행
각 채널용 이미지 방향 제시:
- 인스타그램 캐러셀 10슬라이드 내용 + 레이아웃
- 블로그 썸네일 설명
- 뉴스레터 헤더 이미지 설명
- Canva 작업 지시서 형태로 출력

### STEP 5: 배포 에이전트 실행
최종 콘텐츠를 배포 준비:
- Buffer 예약 발행 텍스트 형식으로 정리
- Beehiiv 뉴스레터 형식으로 정리
- 발행 일정 캘린더 (Notion 테이블 형식)

## 최종 출력

### 📋 이번 캠페인 요약
- 주제: 
- 핵심 메시지:
- 발행 일정:

- [ ] CTA 각 콘텐츠에 포함

```

### 주간 콘텐츠 계획에 사용
```

캠페인 목적: 교육 + 브랜드 인지도
타겟 채널: 전체
기간: 이번 주 (월~금)
```

기간: 오늘
