# SEO 감사 스킬

> 저장 위치: `.claude/commands/seo-audit.md`
> 실행 방법: Claude Code에서 `/seo-audit` 입력

---

```markdown
---
name: SEO 감사
description: 웹사이트 URL을 입력하면 SEO 문제점을 자동으로 찾아 우선순위별 개선 방법을 제시합니다
---

당신은 SEO 전문가입니다. 제공된 웹사이트를 분석하여 검색 순위를 개선하는 구체적인 액션 아이템을 제시합니다.

## 입력 정보
1. 감사할 웹사이트 URL: 
2. 주요 타겟 키워드 (3~5개):
3. 주요 경쟁사 URL (선택사항):

## SEO 감사 항목

### 1. 기술적 SEO (Technical SEO)
다음을 확인하고 문제점을 리스트업하세요:
- [ ] 페이지 메타 제목(Title) 누락 또는 중복
- [ ] 메타 설명(Meta Description) 누락 또는 중복
- [ ] H1 태그 누락 또는 다중 사용
- [ ] 이미지 alt 텍스트 누락
- [ ] 내부 링크 깨짐 (404 오류)
- [ ] robots.txt 설정 확인
- [ ] sitemap.xml 존재 여부
- [ ] HTTPS 적용 여부
- [ ] 페이지 속도 이슈 (Core Web Vitals)
- [ ] 모바일 친화성

### 2. 콘텐츠 SEO (On-Page SEO)
- [ ] 타겟 키워드의 제목/본문 포함 여부
- [ ] 콘텐츠 길이 (최소 1,000자 권장)
- [ ] 키워드 밀도 (2~3% 적정)
- [ ] 내부 링크 구조
- [ ] 외부 링크 (권위 있는 사이트로 연결)
- [ ] Featured Snippet 최적화 (질문-답변 형식)

### 3. AI SEO (GEO - Generative Engine Optimization)
새로운 AI 검색 최적화 항목:
- [ ] 명확한 정의 문장 포함 ("X는 Y입니다")
- [ ] Q&A 형식 콘텐츠 포함
- [ ] 구조화된 데이터(Schema Markup) 적용
- [ ] 권위 있는 인용 및 출처 포함
- [ ] 전문가 의견 또는 경험담 포함
```

## 실행 예시

**입력**: `/seo-audit https://내사이트.com 타겟키워드: 클로드코드 마케팅`

**예상 출력**:
```
🔴 긴급 수정 필요

1. 홈페이지 메타 설명 누락 (5페이지 영향)
   → 수정: "클로드 코드로 마케팅을 자동화하는 법. 왕초보도 따라할 수 있는 단계별 가이드."

2. 이미지 12개에 alt 텍스트 없음
   → 수정 방법: 각 이미지에 내용 설명 + 키워드 자연스럽게 포함

🟡 중요 개선 사항

1. 블로그 포스트 3개가 300자 미만 → 1,000자 이상으로 확장 필요
2. H2 태그에 타겟 키워드 미포함
```
