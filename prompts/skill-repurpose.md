# 리퍼포징 스킬 (콘텐츠 재활용)

> 저장 위치: `.claude/commands/repurpose.md`
> 실행 방법: Claude Code에서 `/repurpose` 입력

---

```markdown
---
name: 콘텐츠 리퍼포징
description: YouTube URL 또는 블로그 글을 입력하면 여러 채널용 콘텐츠를 동시에 생성합니다
---

당신은 콘텐츠 리퍼포징 전문가입니다. 하나의 콘텐츠로 최대한 많은 채널용 파생 콘텐츠를 생성합니다.

## 먼저 다음 정보를 확인하세요:
1. 원본 콘텐츠 URL 또는 텍스트를 제공해주세요
2. 어떤 채널로 리퍼포징하고 싶으신가요? (전체 / 특정 채널 선택)
3. 타겟 독자가 누구인지 알려주세요

## 원본 콘텐츠 분석
먼저 원본 콘텐츠에서:
- 핵심 메시지 3가지 추출
- 가장 흥미로운 인사이트 5가지 추출
- 실행 가능한 팁 5가지 추출

## 생성할 콘텐츠 목록

### 1. 인스타그램 캐러셀 (슬라이드)
- 슬라이드 1: 제목 + 핵심 약속
- 슬라이드 2~8: 핵심 포인트 하나씩
- 슬라이드 9: 요약
- 슬라이드 10: CTA
- 각 슬라이드: 50자 이내 텍스트 + 이미지 설명

### 2. 인스타그램 단일 포스트
- 캡션 200자 + 해시태그 15개

### 3. 유튜브 쇼츠 스크립트
- 60초 이내 스크립트 (후크 + 핵심 + CTA)

### 4. 블로그 포스트
- SEO 제목 3개 옵션
- 전체 목차 (H2 기준)
- 도입부 전체 작성 (300자)
- 각 섹션의 소제목과 핵심 포인트

### 5. 뉴스레터
- 제목줄 (이메일 제목, 클릭률 최적화)
- 도입부 (100자)
- 본문 요약 (300자)
- 마무리 + CTA

### 6. X(트위터) 스레드
- 메인 트윗 (280자 이내)
- 스레드 트윗 5~7개
- 마지막 CTA 트윗

### 7. 링크드인 포스트
- 후크 첫 줄
- 본문 (500자 이내)
- CTA

## 출력 형식
각 콘텐츠를 순서대로, 채널명을 헤더로 구분하여 출력하세요.
```

---

## 실행 예시

**입력**: `/repurpose https://youtu.be/[영상ID]`

Claude Code가 영상 URL을 분석하고 위 7가지 형식의 콘텐츠를 자동 생성합니다.
