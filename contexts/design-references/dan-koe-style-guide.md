# Dan Koe 스타일 디자인 레퍼런스

## 핵심 스타일 특징 (이미지 분석 기반)

### 색상 팔레트 (4색만)
- **배경**: #0A0A0A (딥 블랙)
- **텍스트**: #FFFFFF (순백)
- **강조색**: #E53E3E (레드) — 핵심 키워드, 하이라이트, 화살표 끝
- **보조**: #6B7280 (그레이) — 부연 텍스트, 비활성 요소, 라인

### 타이포그래피
- **제목**: 굵은 산세리프 (올캡스 영문: Inter Black / 한글: Pretendard ExtraBold)
- **본문 키워드**: Pretendard Bold, 24~36px
- **코드/경로**: JetBrains Mono, 18~24px, 그레이 배경 박스 안
- **캘리그래피 사용 안 함** — Dan Koe는 캘리그래피가 아닌 산세리프 중심

### 레이아웃
- **여백이 핵심**: 화면의 60~70%가 빈 공간
- **가운데 정렬**: 핵심 요소가 화면 중앙에 위치
- **개념 하나 = 한 페이지**: 하나의 슬라이드에 하나의 아이디어만
- **텍스트 자체가 비주얼**: 단어 2~3개를 크게 띄워서 이미지처럼 사용

### 인포그래픽 스타일
- **다이어그램 중심**: 원, 화살표, 계층구조를 최소한의 흰색 라인으로
- **라인 두께**: 1~2px 얇은 선
- **도형**: 원, 직선, 화살표 정도만. 복잡한 장식 없음
- **플로우차트**: 위→아래, 왼→오른쪽 흐름을 단순한 선과 화살표로
- **벤 다이어그램**: 겹치는 원으로 관계 표현
- **계층 구조**: 나무 형태 다이어그램

### 일러스트레이션 (블로그/뉴스레터용)
- **흑백 판화/에칭 스타일**: 빈티지 판화 느낌의 상세 라인 드로잉
- **대비가 강함**: 검정 배경에 흰색 일러스트 또는 그 반대
- **메타포 시각화**: 추상적 개념을 구체적 이미지로 (뇌, 손, 운동 선수 등)

### 애니메이션 스타일
- **발광(glow) 효과**: 흰색/보라색 빛이 은은하게 퍼짐
- **기하학적 도형**: 구체, 원, 선이 천천히 움직임
- **라인 그려지기**: SVG stroke 애니메이션
- **텍스트 페이드인**: 단어가 하나씩 나타남
- **절제된 모션**: 화려하지 않지만 세련됨

### 프로그레스 맵 / 목차
- 전체를 하나의 다이어그램으로 시각화
- 현재 위치를 밝게(화이트/레드), 나머지를 어둡게(그레이)
- "THE FUTURE CAREER PATH" 스타일: 중앙 노드에서 가지가 뻗어나가는 형태

## 우리 프로젝트 적용 방법

### 정적 인포그래픽 (PPT 대체)
- Remotion 컴포넌트로 제작
- JSON 데이터 → 7종 템플릿 자동 렌더링
- Dan Koe 색상 팔레트 적용

### 애니메이션 영상 (B-roll)
- Remotion interpolate + spring으로 모션
- SVG stroke 애니메이션 (라인이 그려지는 효과)
- 오프닝/아웃로에만 glow 효과 사용

### 프로그레스 맵
- 4ACT를 한 줄 타임라인으로 표현
- 현재 ACT = 화이트/레드, 완료 = 그레이 체크, 미래 = 어두운 그레이
- ACT 시작/끝에서 5초간 표시

## 참고 이미지 위치
사용자가 제공한 Dan Koe 스타일 참고 이미지:
- 블로그/뉴스레터 썸네일 (흑백 판화 스타일)
- 인포그래픽 다이어그램 (다크 배경 + 흰색 라인)
- 애니메이션 레퍼런스 (glow + 기하학적 도형)
- 강조색 레퍼런스 (흑백 + 레드 하이라이트)
