# 썸네일 제작 지시서 – 파자마보스

## 최종 선택 제목
왕초보도 4시간이면 AI 마케팅 팀 만든다 | 클로드 코드 완전 정복 풀코스

---

## 썸네일 버전 A (메인 추천)

### 배경
- 색상: 다크 네이비 (#0f0f1a)
- 질감: 미세한 그라디언트 (왼쪽 #0f0f1a → 오른쪽 #1a1a2e)
- 우측 하단: 희미한 오케스트라 무대 실루엣 (투명도 15%)

### 텍스트 레이아웃 (3줄 구성)

| 줄 | 텍스트 | 폰트 크기 | 색상 | 폰트 두께 |
|----|--------|-----------|------|-----------|
| 1줄 | 왕초보 OK | 소 (72px) | 노란색 (#FFD700) | SemiBold |
| 2줄 | AI 마케팅팀 | 대 (128px) | 흰색 (#FFFFFF) | ExtraBold |
| 3줄 | 4시간 완성! | 대 (112px) | 오렌지 (#FF6B00) | ExtraBold |

- 텍스트 위치: 좌측 정렬, 수직 중앙
- 텍스트 그림자: 블랙 (#000000) 불투명도 60%, 오프셋 2px

### 뱃지 / 레이블

**"풀코스" 뱃지 (우측 상단)**
- 형태: 원형 (지름 140px)
- 배경: 빨간색 (#FF0000)
- 텍스트: "풀코스" (흰색, Bold, 36px)
- 보조 텍스트: "4시간+" (흰색, Regular, 24px)

**클로드 로고 (우측 중단)**
- Claude 공식 로고 (주황색 앤트로픽 로고)
- 크기: 80×80px
- 위치: 우측 상단 뱃지 아래

### 얼굴 사진 구성
- 위치: 우측 하단 (전신 또는 반신)
- 표정: 놀란 표정 + 손가락으로 텍스트 가리키기
- 의상: 파자마 (채널 아이덴티티 강조)
- 배경 제거 후 누끼 처리

## 썸네일 버전 B (A/B 테스트용)

### 배경
- 색상: 딥 퍼플 (#1a0a2e)
- 우측 하단: 코드 에디터 스크린샷 (흐림 처리, 투명도 20%)

### 텍스트 레이아웃

| 줄 | 텍스트 | 폰트 크기 | 색상 |
|----|--------|-----------|------|
| 1줄 | 클로드 코드 | 소 (68px) | 민트 (#00FFB2) |
| 2줄 | AI 마케팅 자동화 | 대 (120px) | 흰색 (#FFFFFF) |
| 3줄 | 왕초보 풀코스 | 중 (96px) | 노란색 (#FFD700) |

---

## 제작 체크리스트

- [ ] 해상도: 1280×720px (유튜브 표준)
- [ ] 파일 형식: JPG (최대 2MB)
- [ ] 텍스트 가독성: 스마트폰 화면 크기에서도 읽히는지 확인
- [ ] 얼굴 명확히 보이는지 확인 (CTR 향상에 중요)
- [ ] 유튜브 제목과 썸네일 텍스트 중복 최소화 (서로 보완 관계여야 함)
- [ ] 경쟁 영상과 구별되는 색상 확인 (검색 결과에서 눈에 띄는지)

## 참고 벤치마크 채널
- 빌더 조쉬 (한국): 강렬한 색 대비, 큰 숫자 강조
- MrBeast 스타일: 극단적 표정 + 포인팅 제스처
- 국내 IT 교육 채널: 다크 배경 + 네온 계열 텍스트

## 제작 도구 추천
- **Canva Pro**: 유튜브 썸네일 템플릿 제공, 팀 협업 가능
- **Adobe Express**: 무료, 빠른 제작
- **Photoshop**: 고급 보정이 필요한 경우
- **Remove.bg**: 누끼 처리 (무료 월 1개)
