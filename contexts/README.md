# Contexts — 브랜드별 컨텍스트 폴더

이 폴더에는 브랜드/채널별 컨텍스트를 저장합니다.
새로운 풀코스 영상을 만들 때 해당 브랜드 폴더를 만들고 아래 파일들을 작성하세요.

## 사용법

1. `contexts/` 아래에 브랜드 폴더를 만듭니다 (예: `contexts/my-brand/`)
2. 아래 3개 파일을 작성합니다
3. SCRIPT_GUIDE.md와 스킬/에이전트가 자동으로 컨텍스트를 참조합니다

## 필수 파일

### brand-voice.md
브랜드 보이스, 어조, 페르소나, 호칭, 금지 표현 등.
이 파일이 스크립트의 말투와 톤을 결정합니다.

### course-outline.md
풀코스의 전체 구성, 파트 목록, 타임스탬프, 메타포 체계.
이 파일이 스크립트의 구조와 흐름을 결정합니다.

### meta.md
영상 제목, 키워드, 촬영 장비, 검증 기준 등 메타데이터.

## 현재 등록된 컨텍스트

| 폴더 | 브랜드 | 상태 |
|------|--------|------|
| `pajama-boss/` | 파자마보스 (김보스) | 스크립트 작성 중 |

## 새 브랜드 추가 예시

```bash
mkdir contexts/my-new-brand
# brand-voice.md, course-outline.md, meta.md 작성
```

그 후 SCRIPT_GUIDE.md 상단의 `ACTIVE_CONTEXT`를 해당 폴더명으로 변경하면 됩니다.
