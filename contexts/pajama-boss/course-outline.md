# 파자마보스 풀코스 아웃라인

## 영상 정보
- **제목**: 왕초보도 4시간이면 AI 마케팅 팀 만든다 | 클로드 코드 완전 정복 풀코스
- **목표 길이**: 4시간+ (편집 후 약 4시간~4시간 30분)
- **핵심 도구**: Claude Code
- **대상**: 코딩 비전공 왕초보

## 전체 메타포: 나만의 콘텐츠 마케팅 오케스트라 만들기

| 기술 요소 | 오케스트라 비유 |
|-----------|----------------|
| VS Code | 무대 |
| Node.js | 전기 배선 |
| Claude Code | 음향 장비 / 핵심 악단 |
| CLAUDE.md | 총보 (전체 악보) |
| 터미널 | 무대 뒤 통로 (백스테이지) |
| 스킬 (.claude/commands/) | 악기 |
| 에이전트 (.claude/agents/) | 오케스트라 단원 |
| 오케스트레이터 | 지휘자 |
| Hooks | 자동 공연 시스템 / 알람 시계 |
| Plan Mode | 리허설 |
| Edit Mode | 본공연 |
| 컨텍스트 윈도우 | 지휘자의 책상 크기 |
| 토큰 | 택시 미터기 |
| 모델 (Opus/Sonnet/Haiku) | 택시 등급 (리무진/일반/경차) |
| API | 주문 전화번호 |
| Webhook | 초인종 |
