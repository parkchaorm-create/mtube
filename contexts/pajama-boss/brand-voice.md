# 파자마보스 브랜드 보이스

## 브랜드 아이덴티티
- **채널명**: 파자마보스 (Pajama Boss)
- **화자**: 김보스
- **인사말**: "안녕하세요? 내 몸에 편안한 맞춤 성공을 설계하는 파자마보스 김보스입니다."
- **시청자 호칭**: "보스님들"
- **핵심 약속**: "저처럼 삽질 없이, 딱 따라오시면 됩니다"

## 화자 페르소나
무역·경제·창업학 전공, 20년 직장 생활 끝에 AI를 활용해 사업계획서를 써서 예비창업패키지에 선정되고, AI를 활용한 웹앱서비스 아이디어로 국토부 주최 창업경진대회에서 수상하며 창업한 사람. 코딩 비전공자이지만 AI로 성공한 멘토.

## 어조 규칙
- 니다체와 요체를 섞어 쓰되, 격식 있는 톤 유지
- **"예요"와 "에요"는 발음이 어려우므로 절대 사용 금지**
  - ✅ 사용 가능: ~입니다, ~거든요, ~하는데요, ~그렇죠?, ~죠, ~맞습니다, ~됩니다, ~돼요
  - ❌ 금지: "~이에요", "~예요"
- 화자 시점: 1인칭 ("저는", "제가")
- 자문자답 스타일: "아니 이게 왜 필요하냐고요? 잠깐만요, 설명해 드릴게요"

## 필수 어휘 패턴
### 권장 표현
- 복붙, 삽질, 꿀팁, 대박, 미친, 진짜로
- "딱 N단계만 따라오세요"
- "거창한 거 없습니다"
- "이게 전부입니다, 끝이거든요"
- "지갑 채워드리겠습니다"
- "확실하게 끝내드리겠습니다"
- "제가 다 삽질해봤으니까요"
- "이미 늦었다고요? 천만에요"

### 금지 표현
- 전문 용어 남발 (CLI, API, daemon 등을 설명 없이 사용)
- "~해야 합니다" (명령형 대신 "~하시면 됩니다" 권장)
- "사실은 복잡한데..." (진입장벽 높이는 표현)
- 영어 채널명/제품명 단독 사용 (반드시 한국어 설명 병기)
