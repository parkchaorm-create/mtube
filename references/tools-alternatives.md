# Blotato 대체 도구 목록

## blotato가 제공하는 기능 분석

blotato는 소셜 미디어 콘텐츠 생성 + 멀티채널 예약 발행 + AI 비주얼 생성을 통합한 SaaS 도구입니다.
이 영상에서는 blotato 없이 무료/저비용 도구들로 동일한 기능을 구현합니다.

---

## 추천 무료 스택 (왕초보용)

```
Claude Code
   ↓ (콘텐츠 생성)
Notion (콘텐츠 저장 + 계획)
   ↓ (예약 발행)
Buffer (소셜미디어 예약)
   ↓ (뉴스레터)
Beehiiv (뉴스레터 발행)
```

**총 비용**: Claude Pro $20/월 + 나머지 무료

---

## 추천 스택 (중급, 자동화 강화)

```
Claude Code
   ↓ (콘텐츠 생성)
Make.com (워크플로우 자동화)
   ↓
Buffer + Beehiiv (발행)
   ↓
Airtable (성과 추적)
```

**총 비용**: Claude Pro $20/월 + Make.com 무료/9$ + 나머지 무료

## API 키 발급 가이드

영상에서 사용하는 API 키 발급 방법:

### Buffer API
1. buffer.com 로그인
2. 계정 설정 → 개발자 → Access Token 생성
3. `.env` 파일에 저장

### Make.com Webhook
1. make.com 시나리오 생성
2. Webhook 모듈 추가 → URL 복사
3. Claude Code 스킬에 webhook URL 삽입

### Notion API
1. notion.so/my-integrations 접속
2. 새 통합 생성
3. 시크릿 토큰 복사 → `.env` 저장
